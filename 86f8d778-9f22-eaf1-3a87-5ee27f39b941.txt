　　“我们可以灵魂出窍，到外面之后，通知别人……或者干脆就附在人身上，然后带着一些工具来救我们。我想，如果有个凿子或者撬棍之类的，我们可以把缺口附近的石头撬开，只是要花点时间。”
<br/>　　大伙睁大了眼睛，互相看去，都觉得这办法不错，他们都是法师，虽然人不能出去，但是鬼魂可以出去……
<br/>　　说做就做，大伙讨论了一下，妙心和梧桐决定一起尝试，当下作法，灵魂出窍，结果刚走到石门附近，就无法再往外走，尝试了好几次都不行，无奈只好回到身体里。
<br/>　　“这石室有结界，我们的魂魄没什么修为，闯不出去！”梧桐沮丧地说道。
<br/>　　“果然啊。”叶少阳无奈地哼了一声，“建墓的这个坑比，可能已经想到前来盗墓的会是法师，早就做好准备了。”
<br/>　　有些结界是无形的，只针对一切灵体，活人是感觉不到。因此叶少阳对这件事也不感到意外。
<br/>　　人出不去，魂魄也出不去，难道真要活活饿死在这里了？
<br/>　　大伙又尝试了走阴，想要去鬼域搬救兵，间接通知人间的熟人知道，结果因为结界的存在，打不开虚空裂缝……这个法子也只好作罢了。
<br/>　　叶少阳双手托着自己的下巴，仔细想着出路，反正总不能真的饿死在这里面。
<br/>　　大伙也都安静地呆着，各自思考着出去的办法，只有陈晓宇表现得有些歇斯底里，不过也很快冷静下来，一筹莫展地蹲在边上，他的猴王也明白眼前的处境，急的抓耳挠腮的。
<br/>　　看到这只鬼面猴，叶少阳猛然想到什么，道：“我有办法试一试。”
<br/>　　大伙立刻都把目光投射过来，叶少阳站起来，把阴阳镜拿出来，伸手在上面点了一下，道：“出来吧。”
<br/>　　没有动静……
<br/>　　叶少阳当场就震住了，几秒钟后，他的神识离开了肉身，试图投射到阴阳镜中，结果……失败了。也就是说，原本存在着另一个空间的镜面，现在成为实心的了，虽然外形没变，但是本质上却完全不是之前的阴阳镜了。
<br/>　　“牛逼啊，一模一样。”叶少阳打量着阴阳镜，忍不住发表感慨。
<br/>　　“什么意思？”毛小方好奇问道。
<br/>　　“我是说，这幻觉真的很可怕，居然这么真实，简直比真的还真。”
<br/>　　“幻觉？”大伙一下子就怔住了，“你在说什么，什么幻觉？”妙心追问道。
<br/>　　叶少阳放下阴阳镜，望着她，缓缓说道：“我是说，我们现在经历的一切，根本就是幻觉。”
<br/>　　大伙都怔住了，互相看去，一个个都觉得不可思议。
<br/>　　“不可能，通灵佛子花不是都死了吗，怎么可能还是幻觉？”梧桐问道。
<br/>　　叶少阳想了一下道：“也许，它根本就没死。”
<br/>　　没死……
<br/>　　梧桐想了想，摇了摇头，“这更不会，我亲手用金莲业火烧了它……”
<br/>　　“如果从那时候开始，我们就进入幻境了呢？烧掉这株花，也是在幻觉中进行的呢？”
<br/>　　大伙呆住，一个个表情都是不信。连最信任叶少阳的毛小方也不信，说道：“少阳子，这不会吧，那我们看到的这些金银珠宝，难道也是幻觉？”
<br/>　　“当然是了。”
<br/>　　“可……有这么真实的幻觉吗？”
<br/>　　叶少阳笑了笑，说道：“想要验证是不是幻觉，很简单，只要咬一下舌头……”说完，自己先咬了一下舌头，舌尖一疼，血流了出来，叶少阳猛地一个激灵，眼前的一切也逐渐模糊起来，然后又变得清晰。
<br/>　　叶少阳睁开眼睛，转头看去，石室什么的都没有了，自己仍然站在自己呆过的石桥上，面前是那个花坛，花坛的中间，那株被烧成灰烬的通灵佛子花依旧好好地在中间伫立着，花叶随风轻轻摆动。
<br/>　　“真的是幻觉，怎么会这样！”身后有人叫起来。
<br/>　　大伙陆续也都清醒过来，站在一起，傻傻地看着面前的通灵佛子花，没有语言能够形容他们内心的震惊。
<br/>　　过了好久，大伙才缓过来，一起往后移动，离开石桥，开始讨论之前所经历的一切。
<br/>　　“这一次，通灵佛子花玩的比之前还狠，我们所有人进入了同一个幻境之中……并且在幻境中杀死了它。”叶少阳是最早恢复的，在恢复之初，他把所有事情从头想了一遍，得出了这个结论。
<br/>　　“还记得之前我们要烧死通灵佛子花的时候吗，它喷吐出了一阵香气，我们当时也警觉了，生怕进入幻境，但是事实上，我们当时已经进入幻境了，只是幻境和真实的世界一样，让我们误以为眼前的一切还是真实的，我们烧掉通灵佛子花，也是在幻境之中……”
<br/>　　大伙沉吟不语。少顷，梧桐深吸了一口气，喃喃叹道：“太可怕了。”
<br/>　　“是很可怕。”叶少阳抬头望着通灵佛子花，心中十分感慨，所谓的杀人于无形，大概就是这个样子了，什么都模拟了，一切都如同真实，让人根本不会想到自己身在幻境之中……如果不是自己巧合地让美华躲在阴阳镜里，根本不可能发现那是一场幻觉，此时估计早就死在幻觉中了。
<br/>　　神奇而可怕的攻击手段……
<br/>　　这样的邪物，叶少阳还是第一次遇到。
<br/>　　“可是，我们在幻境中去过的那间石室，也是虚构的了？”卢晓清提出质疑，“那石室里的一切，看上去都无比真实啊，包括石碑上的那些字……难道也是佛子花自己想象出来的？”
<br/>　　叶少阳想了想道：“我不觉得佛子花有这个本事，我想，那些应该都是真实存在的，只是被投影到了我们的幻境之中，也就是说，它所创造的幻境，多半是按照现实世界投影产生的，说白了就是在幻境中模拟了一下。”
<br/>　　妙心沉吟道：“极有可能。那间石室，距离这边最多也就几百米，佛子花的灵力能够覆盖到那么远，也是可可以理解。”
<br/>　　“香气。”叶少阳道，“多半是因为香气，香气能覆盖到的地方，应该都是它的灵力势力范围，然后投影到幻境中。”
