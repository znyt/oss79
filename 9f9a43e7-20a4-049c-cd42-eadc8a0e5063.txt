　　蝉衣小姐缓步走入宫殿之中，似乎对众人的热烈反应，早已经习以为常，脸色始终平淡如水，一步一步的走着。
<br/>　　而她所到之处，众人纷纷自觉的让出一条道来。
<br/>　　很快，蝉衣小姐来宫殿的最前端，在考核大台前站定。
<br/>　　虽然人群之中，倒是有不少人，垂涎这位丹道天才的美貌，或者倾慕她的才华，可是她往那一战，自然而然的便生出一股生人勿进的气场。
<br/>　　而且，众人对这位丹道天才，也多少有些了解，她醉心于炼丹之道，对男人一向不假辞色，所以自然也就没人，再敢上前找不自在，全都远远观望着。
<br/>　　不知道是不是因为，这位蝉衣小姐的到来，她身上的冰冷气场，影响到了整个大殿的氛围。
<br/>　　原本还十分喧闹的大殿，一下子变得安静许多，就算有人交谈，也是低声细语，似乎生怕打扰到这位绝世佳人。
<br/>　　不过这种情况，只持续了片刻，却被一道狂放的笑声打断，
<br/>　　“哈哈哈！看来今天还真是热闹啊！”
<br/>　　一道充满着活力的声音，自门口的方向传来，众人纷纷循声望去，
<br/>　　只见来人，是一个满头火红色头发的青年男子，和冰冷、淡雅的蝉衣小姐相反，此人的身上，却透着一股炙热、奔放的气息。
<br/>　　而众人看到此人，顿时纷纷惊呼起来，
<br/>　　“没想到连屠苏公子也来了！”
<br/>　　“屠苏公子的丹道天赋，虽然要比蝉衣小姐逊色一些，可同样也是丹云宗的天才人物！”
<br/>　　“不错！当初蝉衣小姐没有出道之时，屠苏公子才是云丹宗的第一天才，可惜后来被蝉衣小姐后来居上，抢走了第一天才的宝座！”
<br/>　　“据传言，屠苏公子二十八岁那年，晋升为四品炼丹师，之后一连参加过三次五品丹师的考核，可惜都以失败告终，莫非他今日，也是来参加五品炼丹师考核的？”
<br/>　　“看他这架势，是明显要和蝉衣小姐打擂台啊！今天这下可有好戏看了！”
<br/>　　……
<br/>　　伴随着众人的议论声，屠苏公子已经大步迈入宫殿之中，众人顿时再次纷纷让出一条道来。
<br/>　　很快，
<br/>　　屠苏公子也来到考核大台之前，和之前的蝉衣小姐并肩而立。
<br/>　　“蝉衣师妹，听说你今天，要来参加五品丹师考核，作为过来人，师兄我必须要提醒你一句，欲速则不达！这五品炼丹师的考核，可不是那么容易通过的！！”
<br/>　　屠苏公子皮笑肉不笑的开口道，不但火药味十足，眼眸中透着一抹浓浓的嫉妒之色。
<br/>　　曾经，他才是整个云丹宗最为耀眼的天才，就因为眼前这个女人的出现，不但让他沦为了配角，还成为他心头的一根刺，甚至已经成为了他的心魔！
<br/>　　尤其是听说，这个女人今天要来参加五品炼丹师的考核，屠苏公子顿时就坐不住了。
<br/>　　之前他的风头，虽然全部被这个女人抢了过去，可他毕竟比对方年长，所以在炼丹实力上，还是可以稳压对方一头，这也是唯一令他感到欣慰的地方。
<br/>　　可是，如果一旦让这个女人，成功晋级为五品炼丹师，到时候对方就将会彻底超越他。
<br/>　　到时候，他屠苏公子，也将彻底成为过去，永远被这个女人踩在脚下了。
<br/>　　一想到这里，屠苏公子说什么也忍受不了！
<br/>　　不料，蝉衣小姐听到这话，脸色依旧平淡无比，仿佛没有听出，屠苏公子言语中的挑衅之意，语气无悲无喜，淡淡道：
<br/>　　“多谢屠苏师兄提醒！屠苏师兄之前一连三次失利，尚能屡败屡战，师妹虽然不才，若是这次不能通过，明年再考便是！”
<br/>　　“你！……”
<br/>　　屠苏公子听到这话，顿时气得差点没有喷出一口血来。
<br/>　　他之前一连三次，都未能通过五品炼丹师的考核，一向引以为耻，没想到这女人，哪壶不开提哪壶，这让他如何不怒？
<br/>　　“哼！本公子懒得和你这女人做口舌之争，我虽然一连失利三次，却也因此积累了不少宝贵的经验，这一次我必定不会输给你！”
<br/>　　不料，蝉衣小姐微微点了点头，淡淡道：
<br/>　　“是呀，若论失败的经验，屠苏师兄的确无人能及！！”
<br/>　　“噗！”
<br/>　　周围众人听到这话，已经有人忍不住嗤笑起来。
<br/>　　就连叶尘也不由得为之莞尔，
<br/>　　别人或许不知道，他前世和这位蝉衣小姐有过交集，对她的性格倒也了解一些。
<br/>　　这女人，表面看上去冷冰冰的，那是因为修炼了一种特殊功法的缘故，其实骨子里是一个很有趣的女人，说话犀利直接，分分钟能把人怼的怀疑人生。
<br/>　　屠苏公子显然也没有想到，他原本是想用话语故意刺激这女人，好令她方寸大乱，从而可能影响到她的发挥。
<br/>　　万万没有想到，这个平日里安安静静的小师妹，嘲讽起人来，居然如此犀利！
<br/>　　现在可倒好，对方云淡风轻，随口两句话，反倒把他怼的差点暴走，心态已经有些崩了。
<br/>　　经过这一番较量之后，屠苏公子顿时不敢再轻易招惹自己的这位师妹，乖乖躲在一旁，暗暗调息了一阵之后，总算将躁动的心情，渐渐平复了下来。
<br/>　　而这时，
<br/>　　众人期盼已久的丹师考核，也终于要开始了。
<br/>　　几名老者，自大殿的后方，缓步走到考核高台之上。
<br/>　　台下众人顿时又是一阵躁动，
<br/>　　“快看！那位不是云丹宗的火龙长老吗？没想到连他老人家也来了！”
<br/>　　“少见多怪！这丹师考核，除了丹师盟会的长老之外，每次都会邀请一名云丹宗的长老作为评审，以示公正！”
<br/>　　“这次，屠苏公子和蝉衣小姐，云丹宗的两大天才，一同前来考核，场面不同以往，云丹宗自然会派一位，更有分量的长老啦！”
<br/>　　……
<br/>　　在众人的议论声中，丹师盟会的一名长老，双手在空中微微一按，
<br/>　　“诸位！”
<br/>　　声音不大，却清晰无比的传送到了所有人的耳中，原本躁乱的大殿，瞬间便安静了下来。
<br/>　　“老夫现在正式宣布，丹师考核，正式开始！”
