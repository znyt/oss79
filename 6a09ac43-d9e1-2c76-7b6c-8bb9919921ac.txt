        &lt;h3&gt;第三卷第三百一十四章  攻心为上&lt;/h3&gt;
<br/>
<br/>        江俞轩要去北京参加公司耗材上市推广会，他在准备回江氏集团的时候接到陈子霖的电话，便决定去见一见顾恩赐。
<br/>
<br/>        解铃还需系铃人，顾恩赐是这件事情的重点，如果从官方上来解决这件事情已经来不及了，明天耗材就会要推向市场，一部分商品是要给客户的。
<br/>
<br/>        还有买回来的那些酒都是问题。
<br/>
<br/>        这些酒的最大源头都指向红砖路，可以肯定的说，红砖路的那家商铺是有问题的。只不过可惜的是凌辰他们还没有找到真正的源头。
<br/>
<br/>        顾恩赐坐在办公室里在电脑里下着五子棋，他酷爱玩这个游戏，一坐就是一个上午。
<br/>
<br/>        玩着玩着，他老婆的电话来了，他有些不耐烦的接起来了：
<br/>
<br/>        “怎么了？我在开会呢？”
<br/>
<br/>        “刚刚那个陈子霖什么话都没有说，回去了，让把他的耗材给他看好。”
<br/>
<br/>        “你和他们见面了？”
<br/>
<br/>        “我傻呀？我和他们见什么面？”
<br/>
<br/>        “那你打电话给我干什么？”
<br/>
<br/>        “你说他们会不会找别的人来纠缠这个事情？”
<br/>
<br/>        “陈家在西城也没有多大的靠山，最主要的就是他们老四，据说老四现在也不吃香了，你担心啥。咱这后面的大树难道还怕了他们不成？”
<br/>
<br/>        “不是怕他们，只是怕这件事情捅出来也不是很好解决，你想个办法，将耗材退回去，然后让他们不再追究酒的事情了。”
<br/>
<br/>        “你这个女人，你以为我是万能的？你弟弟只贪图眼前利益，也不看看买货的人是谁，真是蠢不可及，要不是我发现了蹊跷，怕连你都得一锅端。”顾恩赐说着话，手中的棋子差一点就走错了。
<br/>
<br/>        “我们怎么知道厂家会来暗访，平时来都会给通知的，这一次不但不通知，还套路起来了。”
<br/>
<br/>        “你们就按照你们的正常手术走不就行了，反正他们现在也没有办法证明这批货的来源，对了，店里的东西都处理了吧？”顾恩赐依然盯着电脑。
<br/>
<br/>        “处理了。”
<br/>
<br/>        “处理了就好，那个什么耗材能压就压，他们明天搞什么联动，让他们的联动泡汤去吧！”顾恩赐终于是下完了一盘棋，看了一眼手机，这娘们唧唧歪歪的，迟不打电话早不打电话，偏偏这个时候打，输了！
<br/>
<br/>        “好像他们耗材的手续齐全着呢。”
<br/>
<br/>        “齐全着你们拉人家的货干什么？吃饱了撑着的?一码是一码，要么你们现在干脆直接给他贴黄条得了，拖上几天再说。就说你们最近太忙，这件事情处理也需要时间，让他们等着！”顾恩赐盯着电脑，在回忆刚才哪里走错了。
<br/>
<br/>        “也不能拖太久了，太久了我怕生变。”
<br/>
<br/>        “怕什么，天塌下来不是还有人顶着嘛？”顾恩赐不以为意。
<br/>
<br/>        “是有人顶着，但是没有让我们明目张胆的做啊。”
<br/>
<br/>        “拖两天在看，看看陈子霖他们有什么动作，我们也摸摸他们的底。”顾恩赐这几年确实有些眼红陈子霖他们，一个项目一个项目的拿。
<br/>
<br/>        “行吧，那就先这样，你盯着他们的动作，我把这件事情也给上面汇报一下。”
<br/>
<br/>        “先别汇报，你自己能解决了就自己解决了，别什么事情都说，这点小事你都搞不定，还能做什么大事情？”顾恩赐自从和夏颜他们拉上关系之后就觉得自己比老婆高了一等。
<br/>
<br/>        “你什么意思啊？”周笑笑语气里已经很不满了。
<br/>
<br/>        “好了，我这里来人了。不说了。”顾恩赐听见敲门声挂了他老婆周笑笑的电话。
<br/>
<br/>        ........
<br/>
<br/>        “进！”顾恩赐挂了电话，对着房门皱着眉喊。
<br/>
<br/>        今天流天不利，连打个游戏都打不安分。
<br/>
<br/>        “顾总，有人找您。”公司的商务带着江俞轩和凌辰进来了。
<br/>
<br/>        顾恩赐认识江俞轩，不认识凌辰，因为代理商的名义是他的妻弟，实际经营者也是他妻弟。
<br/>
<br/>        “哎呀，江总，什么风把你吹来了？”江俞轩离开陈子昂的公司人尽皆知，大家一致在心里猜测俩个人可能是产生了分歧。
<br/>
<br/>        “寒冷的北风加枕边风。”江俞轩笑意不达眼底。
<br/>
<br/>        “哈哈哈，现在可是三月份，应该是春风，来，请坐，这位是？”顾恩赐看着凌辰。
<br/>
<br/>        “凌总，我的一个朋友。”江俞轩和凌辰坐下，顾恩赐摆起了茶具。
<br/>
<br/>        江俞轩能到他公司来，肯定是有事情，顾恩赐很想知道江俞轩是因为什么事情来找他。
<br/>
<br/>        “来，江总、凌总喝茶！”
<br/>
<br/>        江俞轩并没有拿起顾恩赐递过来的茶，他看了一眼顾恩赐：“顾总，凌总是我的朋友，想做一笔小生意，不巧，被有心人给看到了，现在血本全无。”
<br/>
<br/>        “不知道凌总做的什么生意？”顾恩赐不明所以。
<br/>
<br/>        “现在除了吃喝拉撒赚钱之外还有什么能赚钱呢？他在你们家批的商品又被你们家给拿回去了，做人啊，不能这么过分。”江俞轩含笑看着顾恩赐。
<br/>
<br/>        “我们家？”顾恩赐惊讶的长大嘴巴。
<br/>
<br/>        “红砖路的商铺是你们家的，你是实际控股人，你在陈子霖的库房看到了你们家的商品，知道你们家东西的来龙去脉，为了栽桩嫁祸给陈子霖他们，你这就不怕陈子霖破罐子破摔和你杠到底？”江俞轩翻来覆去的看着自己的手，似乎他的手上掌握了很多的东西。
<br/>
<br/>        “江总，你可是冤枉我了，我和陈子霖无冤无仇的，我嫁祸他们干嘛？”顾恩赐有些慌乱。
<br/>
<br/>        “地下库房只有你去过，不是你还有谁，再说了，东西在你老婆单位放着呢。你这也做得太明显了点，陈子霖如果和你杠起来，你们怕是八两对半斤吧？我不管你和陈子霖他们有什么过节，但是我朋友却是借用了陈子霖的库房，你不能让他遭受损失。”
<br/>
<br/>        “江总，你真的是误会我了。”顾恩赐准备打死不承认。
<br/>
<br/>        “其实，顾总，要不是我朋友的事我就不管，可是偏偏就碰到我这里了，我在西城混了这么多年，也不是白混的，你可以打听一下江家。我能来找你，肯定是情况都了解清楚了，只是想给彼此一个退路，抬头不见低头见的，你说是不是？”江俞轩已经不再笑了。
<br/>
<br/>        顾恩赐沉默了一会，权衡利弊之下方艰难开口：“我老婆确实是管着他们这些个破事了，我可以帮你问一下情况，别的我不能担保。”
<br/>
<br/>        “我觉得顾总可以和你老婆商量一下，将这批货回收一下，既然怕大批量的卖出去有问题，不如你们自己找关系处理掉，我朋友就少损失点，他原本想在周边搞个批发，被你这样一弄，吓坏了。”
<br/>
<br/>        “这个......”
<br/>
<br/>        “顾总，你觉得你的主子希望你们出问题吗？你们这可是自己查自己啊！你以为你老婆可以一手遮天？陈子昂在北京那可是混得风生水起，你和她哥哥作对，唉，我都替你着急。”江俞轩冷冷一笑。
<br/>
<br/>        “特么的，这是妥妥的威胁！”顾恩赐在心里暗暗的骂了一句，想想，江俞轩的话还是很有道理的。
<br/>
<br/>        “顾总，我也就是做个小生意的，那些商品什么的，我不要了，你把陈总的货还给他们就好，我不能连累人家，顾总大人大量求放过。”凌辰期期艾艾的开口。
<br/>
<br/>        “江总、凌总、你们真是想多了，我老婆他们的工作性质就那样，我也只能回去吹吹枕边风，请她高抬贵手，我们也算是朋友一场，能帮的忙我会尽量帮的。”顾恩赐咽了口唾沫。
<br/>
<br/>        “哈哈哈，那就谢谢顾总了，我们回去等你的好消息。”江俞轩和凌辰相视一眼，站起来告辞。
<br/>
<br/>        “呵，还真是搬起石头砸自己的脚。”顾恩赐看着离去的江俞轩和凌辰生气的坐在沙发上。
