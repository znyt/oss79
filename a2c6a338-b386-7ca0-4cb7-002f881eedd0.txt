　　　　涂小安在上方继续看戏，心中思量着，青枭是二十级，黄金瞳也是二十级，这两只超等生物血脉是高，战斗力也远比一般的精怪要强。
<br/>　　但是让它们面对一位人类三阶生灵，就有点捉襟见肘。
<br/>　　看的出来，杜天易有心里负担，一直没用全力，连自己真正的手段都还没使出来。
<br/>　　能成为妖精管理局的一组组长，那么必定有过人之处。
<br/>　　当初二组组长冷霜，都是身负元素冰霜之力，而这位杜天易更是三阶生灵了，一定更为的不凡。
<br/>　　涂小安一直在观察，很是好奇他拥有什么样的能力。
<br/>　　“大牛，你知道这杜天易有什么能力吗，看他的腿技到是用的出神入化”涂小安看着下方开口。
<br/>　　牛妖王苦笑道：“主人，我人名叫牛大力，不是大牛”
<br/>　　“我喜欢叫大牛，亲切不行吗”
<br/>　　牛妖王跟受气的媳妇一样，不说话了，你是主人，你高兴就好。
<br/>　　“你跟杜天易打过吗”
<br/>　　牛妖王喷了喷粗气，：“没有，不过杜天易能成为妖精管理局的第一人，手段自然不凡”
<br/>　　“你觉得自己打的过他吗”
<br/>　　“哼哼...老牛要不是看在这小子的特殊身份上，早就将他打成肉泥了”
<br/>　　哦...涂小安笑了笑：“那你下去吧，也不用打成肉泥，打个半死不活就好”
<br/>　　半死不活后，涂小安就可以将他收为战仆了。
<br/>　　三阶妖灵涂小安收服了两只，人类三阶生灵还没有，这个杜天易不但实力高，身份特殊，是个非常好的战仆，能有大用。
<br/>　　隐隐间，涂小安脑海中想起了那只有趣的乌鸦，当初本来说三天将它从妖精管理局带出来，显然他是食言了。
<br/>　　也不知道那只乌鸦会不会画个圈圈诅咒自己。
<br/>　　要是能收服杜天易，或许能将那只乌鸦给弄出来，或许更直接点，涂小安之后就可以进出妖精管理局了。
<br/>　　“铁棍来”
<br/>　　牛妖王身上恐怖残暴的气息涌动，一伸手，铁棍就地显化，被他握在手中，整个人骤然跃起，仰天咆哮：“姓杜的，吃老牛一捧”
<br/>　　涂小安一张脸都黑了，口头禅还要抄袭的吗，有没有一点自己的风格。
<br/>　　下方的杜天易看到牛妖王凶煞无比的跳下来，当头棒喝，一张脸怒火中烧的扭曲起来，眼里闪烁着一股无法遏止的怒火，大喝：“你这个背信弃义的牛妖，下来正好，受死吧”
<br/>　　杜天易觉得自己是被这头牛给坑了，才使得自己陷入困境，周身白色气焰暴涨，狂暴无匹的一拳，夹杂着撼天地之威狠狠向砸来的铁棍轰去。
<br/>　　这完全是正面刚，说明她底气十足。
<br/>　　一般人谁敢用拳头迎接牛妖王的铁棍，吓都吓死了。
<br/>　　澎...
<br/>　　一拳憾动一根铁棍，空间劈噼里啪啦的爆响，狂风骤起，电闪雷鸣。
<br/>　　“哈哈...拳头不错，能挡住老牛的一棍，再来”
<br/>　　牛妖王眼中迸发无比强大的战意，空中一个大呼气，猛然大转身，双手混动如雨，啪啪啪，敲击空气，隆隆作响如同暴雷，只见牛妖王一瞬间砸出去十七八棍，两条手臂都在半空里幻化出来一天残牛像。
<br/>　　这架势是真的准备将杜天易砸成肉泥啊。
<br/>　　世间没有无缘无故的爱，也没有无缘无故的恨，这牛妖王以前没少被杜天易仗着自己妖精管理局的身份敲诈。
<br/>　　还经常让他办事。
<br/>　　比如杜天易想某个人从这个世上消失，这种肮脏的事情，都是牛妖王来做，暗地中，早就憋着气。
<br/>　　杜天易没想到这头牛那么狠，还真往死里打，漫天棍影呼啸而下，空间大范围撕裂，他不敢怠慢，大喝一声，白色气焰愈演愈烈，回头就是一拳，一拳套着一拳，每一击碰撞下，空间都为之破碎。
<br/>　　一双铁拳跟铁棍缠绕，还能看到他的双腿不时扫射，踢在牛妖王皮糙肉厚的身上。
<br/>　　一时间，别墅大院中都是嗤嗤声响，就宛如鬼哭神嚎一般的尖啸。
<br/>　　如今的战场，由于牛妖王的出手，青枭跟黄金瞳直接退出战场，一个飞，一个跳，直接上了阳台，当成了看戏的观众。
<br/>　　“天师，有没有瓜子呀，突然好像嗑”涂小安看的兴起，开口。
<br/>　　张天师翻了翻白眼，直接无视。
<br/>　　涂小安没办法，直接从系统拿出了蛇莓当零食吃，顺便给青枭跟黄金瞳分了几颗。
<br/>　　看戏没点东西吃，总感觉不对劲，融合不到氛围去。
<br/>　　张天师刚才无视他，哼...不给他蛇莓吃，涂小安现在就是那么的傲娇。
<br/>　　此时，下方战斗形成了一个风暴漩涡，近身搏斗，好不精彩。
<br/>　　正常来说，妖灵的肉身很强，战斗力也普遍比人类修武者要强，尤其是这头牛，更是皮糙肉厚，刀枪不入。
<br/>　　但是杜天易既然凭借着一拳一脚，跟他打个难舍难分，看的涂小安都诧异不已。
<br/>　　而且，到现在为止，杜天易也没用什么特殊的能力。
<br/>　　十几个回合打下来，涂小安表情俨然起来，只见下方的杜天易周身涌现一股特殊能量波动，忽然，空气像是煮沸了一般，连青枭跟黄金瞳都躁动不安起来。
<br/>　　张天师到是淡定，没有受影响，毕竟是三阶虺蛟，别人以后是要做龙一样的男人。
<br/>　　也不知道杜天易做了什么，牛妖王被一脚踹飞了出去，然后一脸的懵逼，因为他的铁棍已经在杜天易的手中。
<br/>　　杜天易脸色噙着一丝自信的冷笑，手指轻点了下铁棍，这根不知道多少公斤重的棍子既然软绵绵的下来。
<br/>　　对，是软绵绵，形成了波浪状，扭动了起来。
<br/>　　不知道的，还以为这是一条绳子。
<br/>　　这...？
<br/>　　涂小安看的惊奇，这杜天易用的什么诡异手段，将铁棍变成了麻花绳一样。
<br/>　　牛妖王更是日了狗的表情，怀疑这是不是自己的铁棍。
<br/>　　他的铁棍可是无坚不摧，神威无比，怎么能跟跳舞一样的扭动起来，还有点婀娜多姿的味道。
<br/>　　“杜天易，你对老牛的棍子做了什么”
<br/>　　只见杜天易轻蔑的看了慌张的牛妖王一眼，手指弹了一下铁棍，叮...，一阵金属声音激荡起来，非常刺耳，铁棍又恢复如常。
<br/>　　这尼玛什么能力，那么神奇。
