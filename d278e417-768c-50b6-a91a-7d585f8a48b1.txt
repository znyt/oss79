　　第705章
<br/>　　长安周围并不是没有兵，皇宫的三千鱼龙卫，负责守卫长安的两万城防军，以及分布京州各地的京州兵，京州一地的总兵力达到可怖的三十万。
<br/>　　当然，这些兵大部分都由相府掌握，军内遍布相府的亲信。便是李乐天，发出一道军令，也许相府点头，否则他一员兵也调动不了。
<br/>　　做皇帝做到这个份上，也难怪李乐天整日郁郁不乐。
<br/>　　李乐天心中有一份账，盘算着自己能动用的兵力有多少。其实，他也并不缺乏亲信，当初守卫长安，李乐天带着兵将舍身死战，在底层士兵中还是很有威望的。
<br/>　　但这次做事，李乐天没敢与任何人联络。人多嘴杂，一旦走漏消息，这件事便是自己的催命符。
<br/>　　他只是心中悄悄谋算着，百里胜的八百兵，以及鱼龙卫中自己的亲信，算来算去，也不过是两千兵力。靠这些兵力与根深蒂固的相府掰手腕，的确有几分不自量力。
<br/>　　如果有的选，李乐天定不会跨出这一步，而问题是，他现在已没得选了。
<br/>　　李乐天是打算做些事的，他不想成为某个人的提线傀儡，于是铤而走险，做一次亡命徒仿佛成了理所应当的事情。
<br/>　　夜里的长安静悄悄。
<br/>　　宵禁之后，大街长道之上便没了人，只是偶尔有几家青楼，几处大院响起歌舞乐器之声。
<br/>　　这些背后，大概有王孙贵族撑腰，宵禁，禁的是普通人，自然禁不到他们。
<br/>　　长安大街上，一乘小轿出了府，四名轿夫都是精壮后生，健步如飞，小轿却并不摇晃。
<br/>　　一直到宫门前，崔云间下了轿，理了理官服，望了望眼前高耸的宫墙。
<br/>　　夜里收到宫中的消息，说是有要事相商。至于是什么事，送信人并没有说。请的本来是崔相，但崔相因故来不了，只有崔云间过来跑一趟。
<br/>　　他迈步跨过宫门，提着官服向里走。宫墙深深深几许，到了夜里，大内还是颇安静的，平白一种阴森味道。宫院内，除了前面一个引路的小黄门，几乎看不到人影。
<br/>　　从这里到天子起居之所，还有很远的距离，崔云间埋着头走路，心底不由得寻思：天子居住在这里，每日会是怎样心情呢。
<br/>　　万里江山，一手便可掀起风雨，一手又可驱散风云。有这世上最稀有的宝物，最动人的女子……普天之下莫非王土，率土之滨莫非王臣。这天下的东西，我可以说我不要，但我不给你，你不许碰。
<br/>　　崔云间长吸一口气，他并非一个有野心的人，生来也未对那张椅子有过什么想法。但现在不同，现在自己距离那张椅子已经十分近了。
<br/>　　只要自己挥挥手，就可以将椅子上的人赶下去。
<br/>　　天予不取，反受其咎。父亲曾说过，崔家不反。可是，那张椅子就摆在自己面前，自己真的要拒绝么？
<br/>　　“崔大人，到了。”小黄门立在大殿前停住脚步。
<br/>　　“喔喔。”崔云间恍然回过神来，最后一次理了理官服，迈步踏入大殿。
<br/>　　大殿之内，灯火通明。一着黑袍的年轻人坐在书案前，纹龙袍，紫金冠，一股逼人的气势。
<br/>　　崔云间双膝跪倒，道：“微臣叩见陛下。”
<br/>　　说罢，崔云间就恨不得给自己一个耳光，当真是管不住自己的膝盖。刚才跪倒仿佛是下意识肢体的反应，根本没来得及过脑子。
<br/>　　“崔大人……”天子放下手中书，慢吞吞道：“崔相大人为何不来？”
<br/>　　“启禀陛下，家父体染微疾，还请陛下恕罪。”
<br/>　　“得病了么。”天子喃喃一声，摆摆手道：“不来便不来吧，谈不上恕罪，毕竟身体要紧，崔相大人的病不严重吧？”
<br/>　　“劳烦陛下挂念，家父只是偶感风寒，将养两日便可痊愈，并不严重。”
<br/>　　“不严重就好，不严重就好呐。”
<br/>　　天子长叹一声，也说不得他是高兴，还是觉得遗憾。
<br/>　　崔云间抬起头，心里在打鼓，问道：“不知陛下深夜召臣前来，所为何事？”
<br/>　　“也无什么大事。”天子道：“崔大人过来看，可认得这柄剑？”
<br/>　　崔云间这才起身，慢慢靠近龙书案，只见上面摆着一柄长剑。
<br/>　　他点点头道：“这是先帝留给陛下的山河天子剑，持此剑执掌帝国山河。”
<br/>　　“是呐，当初父王采天山铁铸剑十三，天子剑一，诸侯剑十二。天子剑由先帝执掌，十二柄赏赐给十二位皇子。可崔大人是否知道，在这十三柄剑外还有一柄剑。”
<br/>　　崔云间点点头：“还有一柄匹夫剑，原先在六皇子手中，后来落到程大雷那恶贼手中。”
<br/>　　“正是如此，程大雷手中还有一柄剑。前些日子他给朕送来一封信，说匹夫剑是先帝赏赐给他的，持匹夫剑斩尽天下不臣，这是先帝的遗命。”李乐天笑着道。
<br/>　　崔云间也笑了：“哈哈，这逆贼也实在太过异想天开，这种信口开河的言语天下有几人会相信。”
<br/>　　“不，他说的是真的，朕已差人去过凉州，封他为凉州之王。”
<br/>　　李乐天脸上的笑容渐渐凝固，崔云间下意识抬起头，惊愕的发现，眼前那还是一位高高在上的帝国天子。两眼恶毒，表情狰狞，分明就是大街上走投无路，输红眼的亡命徒。
<br/>　　“陛下……”
<br/>　　“匹夫剑能斩天下不臣，天子剑能斩谁？”李乐天咬着牙问。
<br/>　　崔云间下意识往后退一步，他猛然发觉，今天的事情有些不对。
<br/>　　天子拔出剑，一步步靠近，口中一字一顿：“朕是当今天子，天下是朕的，朕想杀谁便杀谁。”
<br/>　　崔云间下意识后退，恍然想起，眼前这位不仅是位养尊处优的帝皇，也是位能够在马上和林少羽过招的高手。
<br/>　　而崔云间自己，委实是位十指不沾阳春水的文弱书生。
<br/>　　“来人，来人，快来人……”
<br/>　　李乐天哈哈大笑，口中道：“这是皇宫，是朕的皇宫，你喊来人，来什么人，来杀朕的人么？”
<br/>　　山河剑披头斩下，将崔云间的脖子砍开一道血线，血咕咚咚涌出来，张大嘴巴，有句话想说却说不出来。
<br/>　　“天子……疯了。”
<br/>(本章完)
