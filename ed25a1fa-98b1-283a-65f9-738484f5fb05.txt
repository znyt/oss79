　　第268章
<br/>　　手下人散开，暂时潜入山林中隐蔽。程大雷则是带着徐神机、秦蛮、李行哉、福德勒四人去探查地形。
<br/>　　往前行出现浩浩一片水泊，方圆百里，芦苇连绵，在水中央有一座小岛。
<br/>　　“倒是一个休养生息的好所在呐。”程大雷暗暗叹道。
<br/>　　“这里是大河之水的支流，大河流经这里，冲开一片水泊，中间那座岛称作蛇岛，当初是莫鸣米的水寨，只是现在已经荒废不用。据说，抛弃这座岛时，莫鸣米在上面放养了许多毒蛇，如今十几年过去，上面遍布毒蛇，我曾经上去看过，情景很可怕。”
<br/>　　“你还有胆子上去。”程大雷不得不对李行哉刮目相看：“怎么上去？”
<br/>　　“有处地方离岸约有五里地，泅渡可以过去。”李行哉。
<br/>　　“走，我们上去看看。”
<br/>　　由李行哉引路，来到靠近小岛的岸边，程大雷、秦蛮、李行哉都有不错的水性，只不过徐神机和福德勒皆不会水，二人在岸上等待，三人泅水登上了蛇岛。
<br/>　　一登上蛇岛，李行哉顿时紧张起来，谨慎叮嘱程大雷二人：“小心些，这里到处都是毒蛇，如果被咬上一口，你的命就交待在这里了。”
<br/>　　秦蛮捡了一根长棍打草惊蛇，三人都不敢往岛中心走，只见随着木棍的敲打，一条条花皮蛇从草丛中窜出来。
<br/>　　这些蛇也不怕人，窜出来后就呲牙探出蛇信。
<br/>　　程大雷的剑很快，短短时间就将十几条柳条粗细的蛇切成两段，如此才将蛇群驱散。
<br/>　　李行哉露出吃惊的眼神，随之笑笑道：“这把剑好用吧？”
<br/>　　“也就凑活吧。”程大雷将剑收起来，问秦蛮道：“认得出来是什么蛇么？”
<br/>　　“北地的蛇和这里的蛇不一样，我认不得。”秦蛮严肃道：“不过肯定有毒，剧毒。”
<br/>　　程大雷站在岸边打量这座小岛，岛上大树丛生，绿草成片，远处隐隐有几栋建筑，不过他不敢再往前走，所以也看不真切。
<br/>　　“走了，撤吧。”程大雷挥挥手。
<br/>　　三人又泅水离开蛇岛，一来一去都累得气喘吁吁，瘫倒在岸边没有站起来的力气。
<br/>　　“大当家，岛上什么情况？”
<br/>　　“蛇岛四面有水，易守难攻，岛上地形平坦，也适合种植，是一个安营扎寨的好所在，只不过岛上蛇太多了。”程大雷叹了口气，忽然骂道：“姓莫的真不是东西，不要的地方还放一群毒蛇，祸害后来人。”
<br/>　　“关键是怎么把蛇赶走？”程大雷念叨了一句，忽然翻了个身：“蛇怕什么？”
<br/>　　秦蛮与李行哉几乎是异口同声的回答：“雄黄？”
<br/>　　程大雷点点头，坐直身子：“哪里有雄黄？”
<br/>　　几个人都闭上了嘴巴，谁也无法回答这个问题。
<br/>　　程大雷回到了营地，接下来就是把人散出去，看方圆的城镇，那里有雄黄。而程大雷要做的，就是等待了。
<br/>　　“大当家，大当家……”在弟兄们都散去时，徐神机找到程大雷：“我忽然想起一件事，咱们不能在岛上住啊？”
<br/>　　“为什么？”看徐神机如此焦急，程大雷也不得不认真起来。
<br/>　　“犯地名啦。”
<br/>　　“呃……”
<br/>　　“咱们是蛤蟆寨，大当家你的外号又是蛤蟆大王，这蛇可是蛤蟆的天敌，咱们到蛇岛，岂不是自投罗网么。”
<br/>　　程大雷无语，若说以前，他是一个坚定的唯物主义者。但最近刀山血海里闯，程大雷很难说一点也不信。
<br/>　　可总不能因为一个名字，就把这座岛放弃了吧，再想找到合适的地方可不容易。
<br/>　　程大雷心中忽然一动，道：“好啦，本来要是别人提出来的，可能我还真得考虑考虑，但既然是你说的，那这座岛不是大凶之地，反而有可能是大吉之地。”
<br/>　　“大当家，你可要三思呢，一定不能冲动。”
<br/>　　“嗯嗯，我知道啦。”程大雷道：“回头我给它改个名字，实在不行再请个道士做做法事，这样你可以放心了吧。”
<br/>　　徐神机还准备再说什么时，程大雷已经起身离开。现在，大部分弟兄都被散出去，打探何处有雄黄。只留下几人警惕琴川关方向，现在还没有与莫鸣米一战的本事，暂且不能让对方知道己方已经来了。
<br/>　　程大雷找到李行哉时，他正在烤一只山鸡。在山鸡外裹上泥巴，放在火炭下烘烤，等水分被烤干，里面的山鸡也熟透了。
<br/>　　程大雷看着他砸开泥巴，撕了一半山鸡给福德勒，二人配合很默契。
<br/>　　“你做这些很熟练？”程大雷在他身边坐下：“都是这一年以来学的。”
<br/>　　“不是一年，是两年。”李行哉摆摆手：“我和婉儿从蛤蟆寨离开后，婉儿回了长安，我却突然想去外面看看。就从青牛山去了草原，在草原上放过羊，也挤过马奶，跟着行商做过生意。”
<br/>　　“感觉怎么样？”
<br/>　　“糟糕透顶。”李行哉吐出一根鸡骨头：“哪有在长安待着舒服，有侍女丫鬟伺候着，不过我只是想去看看。结果戎族打过来了，帝国兵败如山倒，我就想我是帝国王子呐，我怎么能不管，我一定要逆流而上，力挽狂澜。”
<br/>　　“结果……”
<br/>　　“结果就根本没人搭理我，还把我当骗子抓起来，搞得我像个蠢货似的。”
<br/>　　“我在长安听说，你逃跑了，如果被送回长安，应该不会有事的？”
<br/>　　“被送回来很丢人的好不好，我好歹是六王子，难道不要脸面啊。”李行哉道，他摇摇头，道：“我就和大家一起跑啊，逃啊，看着成片成片的败兵，成片成片的死亡，我就想，或许问题不在戎族，而是帝国出了什么问题。”
<br/>　　“这一年我就走啊，看啊，走了好多地方，见过终日劳累的农夫，苦工，也见过仗势欺人的恶霸，贵族。我发现，不管有钱的还是没钱的，有权的还是没权的，都过得不开心。至于问题的答案，我还是没找到……”
<br/>　　李行哉晃了晃脑袋，这一刻他脸上的颓唐，才让他显得像个帝国王子。
<br/>　　“程大雷，你说帝国的问题究竟在哪里？”
<br/>　　
<br/>(本章完)
