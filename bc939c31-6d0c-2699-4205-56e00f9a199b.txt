<br/><br/>　　    傅华知道这五张牌游戏是一种简化了的梭哈玩法，各门花色的牌只取8、9、10、j、q、k、a，有黑桃、红桃、草花、方片四种，因此游戏共有28张牌。玩法规则基本跟梭哈是一样的。他笑了笑说：“我多少知道一点玩法。”<br/>　　<br/>　　    倪氏杰笑了笑，说：“既然这样子，那坐下来玩吧，多一个人还热闹些。”<br/>　　<br/>　　    五个人重新分座次坐了下来，另外两人据介绍一个叫张毅辉，一个叫李运廷，都是什么公司的董事长，傅华虽然没听说过他们公司的名字，但老板能够有资格跟倪氏杰坐在一起玩的，公司想来也应该是实力不俗的。<br/>　　<br/>　　    头几把牌傅华跟到第二张牌的时候弃牌了，一来他的牌面都不大，跟下去没意思；二来他跟今天玩的这些人，除了苏南之外，都不熟悉，并不知道对方是一个什么样的性格，而玩牌想要赢，除了运气特别好之外，是要了解对手玩牌的脾性的。不过是这样，傅华也输了几十万出去了，因为这帮人玩的很大，底注是十万，然后每次跟注也是十万。<br/>　　<br/>　　    倪氏杰看傅华几把都弃牌了，笑了笑，说：“傅董玩牌到很谨慎啊。”<br/>　　<br/>　　    傅华笑了笑说：“没有好牌，我不做无谓的牺牲了。”<br/>　　<br/>　　    又弃了几把，傅华终于拿到了一对a，这在发到第二张牌的时候算是大牌了，他笑了一下说：“总算拿到了可以玩一玩的牌了。”于是他这一次没有弃牌，而是扔了一个十万的筹码下去。<br/>　　<br/>　　    苏南笑了一下，说：“你这把肯玩了，说明你的牌面很大，我的牌面不行，这把不陪你玩了。”<br/>　　<br/>　　    苏南弃了牌，李运廷也跟着弃了牌，张毅辉则是选择了跟注。在前几把当，傅华观察过这个张毅辉玩牌的手法，知道这家伙有没有大牌都是会跟的，。<br/>　　<br/>　　    倪氏杰看了看傅华，笑了笑说：“傅董好不容易才玩了一把，那我跟你凑凑趣好了，我跟你十万，再大你十万。”<br/>　　<br/>　　    傅华看倪氏杰桌面的牌仅仅是一张十，微微愣了一下，倪氏杰现在的这个牌面并不大，顶多是一对十而已，他跟十万并不令人意外，他再加注十万没必要了。这个倪氏杰也不是玩五张牌的新手，他这么做可是有意而为之了。<br/>　　<br/>　　    傅华抬头看了倪氏杰一眼，倪氏杰也正在看他，两人的眼神对到了一起，傅华从倪氏杰的眼神看到了一丝蔑视，明白倪氏杰加这十万是表达他对自己的轻蔑了。<br/>　　<br/>　　    傅华心有些恼火了，但是他也明白这个时候千万不能真的生气，玩牌最需要的冷静，如果被倪氏杰激怒，导致心浮气躁，那接下来他会失去对牌局准确的判断了。<br/>　　<br/>　　    傅华笑了一下，说：“倪董兴致这么高，我不陪陪您也不好意思，跟了。”<br/>　　<br/>　　    傅华说着扔了十万筹码下去，张毅辉也跟了十万。第三张牌发了下来，傅华拿到的是一张q，而倪氏杰拿到的是一张十，这样子桌面是倪氏杰一对十最大了。<br/>　　<br/>　　    倪氏杰笑了笑说：“看来是被我等到了，我也懒得十万十万的下了，下注一百万好了。”<br/>　　<br/>　　    倪氏杰说着扔了一个百万的筹码下去，然后微笑着看着张毅辉和傅华，等着两人做出下注还是不下注的抉择。<br/>　　<br/>　　    张毅辉笑了笑，说：“倪董下这么大的注，看来很可能是三条十了，这我可不过，我不跟了。”<br/>　　<br/>　　    张毅辉说着把牌弃了。倪氏杰的眼睛转到了傅华身，笑了笑说：“傅董什么意思啊，根还是不跟啊？”<br/>　　<br/>　　    傅华这个时候有些为难了，如果倪氏杰真是三条十，他一对a显然是不过倪氏杰的，他应该弃牌了。但是如果倪氏杰是虚张声势，手的牌仅仅是一对十，那他这对a弃了太可惜了。<br/>　　<br/>　　    傅华抬头看了倪氏杰一眼，想从倪氏杰脸的表情看出他究竟是真有三条十还是虚张声势。但他马失望了，倪氏杰显得很平静，脸丝毫看不出兴奋或者紧张的样子。<br/>　　<br/>　　    傅华想想也是，对于倪氏杰这种掌控着诺大资产的人来说，一百万实在算不什么大钱，他根本不会因此兴奋或者紧张的。于是傅华要自行判断是跟还是不跟。<br/>　　<br/>　　    傅华意识到他这个判断不仅仅要从牌面考虑，还要考虑到倪氏杰对他所做出行为的感受。如果他的行为正迎合了倪氏杰的想法，倪氏杰很可能会因此而改变对他的看法，说不定会给他机会谈谈项目的合作事宜呢。<br/>　　<br/>　　    而要判断倪氏杰对他行为的感受，要看是倪氏杰是那种不能输的人呢，还是不愿意接受别人让他的人呢？目前傅华还无法对此作出判断。他决定试试倪氏杰再说了。<br/>　　<br/>　　    “我不信倪董已经拿到了三条十，你这是想吓退我，呵呵，没那么容易，我跟了。”傅华说完也扔了一百万的筹码下去。<br/>　　<br/>　　    倪氏杰的眼神亮了一下，笑了笑说：“这牌局总算是有意思了。”<br/>　　<br/>　　    从这一点看，傅华觉得这个倪氏杰应该是一个赌性很大的人，这种人如果能让他的话，他反而会感觉到无趣，最好是赢得他心服口服，他才会真心的佩服你，也才会有兴趣跟你交往。<br/>　　<br/>　　    这时第四张牌发了下来，傅华拿到了一张十，而倪氏杰则是拿到了一张a。倪氏杰的牌面大，由他说话。倪氏杰笑了起来，说：“傅董啊，我拿到了你的a，看来你想凑成三条a的可能性不大了，我还是下注一百万，你要不要跟啊？”<br/>　　<br/>　　    倪氏杰并没有继续加大筹码，而只是跟回下注一样，再次下了一百万。傅华对此的判断是要么倪氏杰手真的有三条十，怕下注太大惊走了他，所以才下注一百万，诱骗他继续跟下去。<br/>　　<br/>　　    要么倪氏杰手并没有拿到三条十，他手仅仅有一对十，底气不足，所以才会仅仅下了一百万。<br/>　　<br/>　　    傅华略微一思忖，决定继续跟下去了，他这倒不是有赢倪氏杰的把握，而是因为既然倪氏杰赌性很大，他继续跟下去，正好投合了倪氏杰的想法，也许这能够引起倪氏杰对他的好感。反正这笔钱他也是输得起的，他赢雎才焘的钱还放在胡东强那里没动用过呢，今天倒是正好派了用场了。<br/>　　<br/>　　    傅华笑了笑说：“我已经跟到了这一步了，没理由会放弃的。那我舍命陪君子了。跟倪董一百万好了。”<br/>　　<br/>　　    傅华说着先扔下去了一百万的筹码，然后又拿起了一百万的筹码，拿在手掂量着，笑着说：“老这么玩下去似乎有点不够劲，我再……”<br/>　　<br/>　　    傅华做出这个架势似乎是想再加注一百万的样子，但实际他这是在试探倪氏杰的反应，他在说话的时候，一直都在偷眼打量着倪氏杰，看看倪氏杰神情会不会有什么变化。倪氏杰当了，在傅华拿着一百万筹码似下不下的时候，他眼闪过了一丝对手掉入他的陷阱的得意。这个得意虽然是一闪即逝，却已经看在了傅华的眼。<br/>　　<br/>　　    傅华心暗自好笑，心说倪氏杰这只老狐狸还是没修炼够道行，在这最关键的时候露出了破绽。这个时候傅华基本已经是认定倪氏杰手已经拿到了三条十了。既然是这样子，他没必要做无谓的牺牲了。<br/>　　<br/>　　    傅华笑了一下，说：“本来我是想加注的，但倪董是前辈，初次交手，我还是礼让一点好，这一次算了。”<br/>　　<br/>　　    第五张牌发了下来，看到自己拿到的牌，傅华心里不由得笑了，心说倪氏杰今天遇到自己可真够倒霉的，这一把他算是输定了。原来傅华最后一张牌居然拿到了一张a，这样子他手已经拿到了三条a，对倪氏杰可能拿到的三条十，这已经是稳赢的牌了。<br/>　　<br/>　　    而这个时候傅华算是想让牌也是没法让的，他桌的牌面是一对a，而倪氏杰的牌面是一对十，他如果在这个时候弃牌的话，其他人会看出来他是在让牌的。那样子的话，恐怕会让倪氏杰更下不来台的。<br/>　　<br/>　　    傅华笑了笑说：“倪董啊，我的运气看来不错啊，我还是又拿到了一张a，好吧，为了这张a，我下注一百万好了。”<br/>　　<br/>　　    这个时候，傅华觉得倪氏杰应该猜到了他拿到了三条a，所以无论他下多少筹码，倪氏杰应该都不会跟的。<br/>　　<br/>　　    果然倪氏杰把牌给弃了，笑了笑说：“不得不说，傅董这把的运气还真是不错的。”<br/>　　<br/>　　    傅华听得出来倪氏杰语气之是有些不服气的，似乎他输在了运气，而非输在技术。这一点傅华倒也是承认的，这一局在傅华拿到最后一张a之前，倪氏杰招数都是很成功的，他成功的诱使傅华多跟了两次一百万。要不是运气让傅华在最后翻了盘，这一局的输家应该是他傅华了。<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
