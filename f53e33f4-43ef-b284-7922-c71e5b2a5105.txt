　　下一刻，混沌道君的身影出现在一处混沌洞中，洞中闪烁着道道北极仙光，宛如一条仙道一般。
<br/>　　“这是一个通往上界的节点！可惜，此道路已损，不能通人。”道君感慨道。
<br/>　　蓦然，道君看向洞口，在那里，有两道身影，静静盘坐在混沌尽头，背对众生，有一种难以言喻的气韵，好似没有什么能够阻拦他的气息。一人身资伟岸，满头发丝浓密，于沉寂中有一种落寞。另一人满脸高傲，气贯长虹，威压苍生。
<br/>　　两道身影好似在对峙，在相互对抗。
<br/>　　他知道，这只是两道身影，多年之前在此对峙。之所以他能够在此见到他们，是因为他们的强大，使得自身印记被烙印在此。此刻，他们早已经不见了。
<br/>　　同时，在那里，有一口巨大仙钟，一声悠悠钟响长鸣，一片符文，交织在一起，闪烁着各种道符神光。
<br/>　　一处仙洞，悬挂着一口完全由秩序神纹化的神钟！
<br/>　　钟声悠扬，带有无尽苍茫，充斥着一种特别的岁月气息。尽管它只是神纹所化，但它所展现出来的威力，仍然很强大。
<br/>　　正是这口大钟，使得此处的混沌通道并没有就此消失，依旧显化。如若不然，此混沌通道早已不见了。
<br/>　　“不同的道，不同的法，来自异世界的神器道则。”混沌道君眼神闪烁，眼中闪现出混沌光芒，看向道则所化的大钟。
<br/>　　不过片刻时间，他便敛去了眼中的神光，恢复了平静。他轻声道：
<br/>　　“这个世界的道则，有点意思。”
<br/>　　混沌道君将瞬息便将这些道则领悟学会。尽管它只是秩序神纹所化，不是真正本体，但却也蕴含了夺天地造化的奥秘。
<br/>　　若是一般人见此，还没等他看见钟影，便会被这些神纹反噬而死，而混沌道君显然不会。
<br/>　　他不再犹豫，一步踏出，越过两道虚影以及神钟虚影，来到洞外。这大钟已经被禁锢了，显然是那两道虚影对战之时，其中一人想要引神钟爆炸，但却被另一人以无上神通禁锢了。被打回了原点，停在此处。
<br/>　　道君眼前出现了一片宫殿，宏大无比，周围血肉尸体有很多，被打碎了，血肉满地，鲜血流尽，散了一地，失去了精华。看样子至少死去了数百上千年了。
<br/>　　大多数都很普通，其中有三具大约有堪比接近金仙的实力。一头是神虎，一具是尸道修炼者，还有一具人族尸体。
<br/>　　此地为死亡禁地，阴气万重，尸体遍地，死气极多。但正是这样一个死地，却孕育出一种诡异的生机，且极为强大。
<br/>　　广袤的大地之上，尸骸无数，一眼看不到边，好似葬了一个时代所有的强者。连土地都是血泥与碎骨筑成的。
<br/>　　道君眼眸深处爆射神光，神眼睁开，破开一切虚妄，让整片殿宇都亮堂起来，阴霾尽散。
<br/>　　只见那宏大的殿宇上方，一片虚空之中，有一只血色的肉球，又像是一只大茧。
<br/>　　肉球散发着一种魔性的力量，影响着人的思绪，忍不住想要为它献祭，奉上自己的血精。但那种魔性力量一近混沌道君身边，便消散而去，不能影响分毫。
<br/>　　“这是……一只涅槃的凤凰？！”
<br/>　　道君眼中闪过一丝异色，这只凤凰还没有苏醒，他的力量也不是很强，仅仅有洪荒太乙金仙的实力。
<br/>　　让他惊异的是，眼前这一幕，让他感觉有些熟悉。他心中顿时有了一丝猜想。
<br/>　　只见混沌道君大手向着那巨大血球一抓，混沌道则交织在一起，禁锢了这片虚空，不让它逃走。
<br/>　　毫无任何阻拦，那血球被道君径直抓到手中。肉球复苏，内部迸发出一片灿烂的光，五彩照耀宇宙，十光横断星空，太过恐怖，这是一种无与伦比的可怕气息。
<br/>　　锵！
<br/>　　如同天刀出鞘，在那血肉中有五道仙光飞出，每一条都铿锵作响，铮铮而鸣，仔细看那竟然是五根翎羽，妖异而美丽，灿烂而炫目！
<br/>　　混沌道君不管，快速在它身上设下禁制封印，不管它怎么动，都挣脱不了道君所设的禁锢。道君瞬间便将它的神魂连同全身的力量、法力封印，此刻的它，就如同一个普通人。
<br/>　　实在是混沌道君的速度太快了，从抓住血球到封印住它，不到一个呼吸。它刚刚苏醒，想反抗都来不及了，便被封印了，当然，就算他完全苏醒了，在道君面前依旧不够看。
<br/>　　宇宙边荒，混沌气汹涌，一口天刀雪亮如闪电，划破了混沌，朝着混沌道君此处劈开。
<br/>　　此刀杀气逼人，气浪茫茫，初看是一个整体，为一种古朴的颜色，再细看之际，却发觉，共分五色，竟是五种仙金混合铸成的，完美融合在一起，成为一个整体。
<br/>　　天刀如同开天辟地前的第一缕曦光，刀身晶莹，光束璀璨无比，震慑人心，刀气如海般扩散，震动了万古星空。
<br/>　　这一日，摧残的倒忙横空，划破了无垠的宇宙，黑暗与冰冷被切开，天刀当世，万物不可阻挡！
<br/>　　可怕的光芒扫过，一挂挂星河都要暗淡，仿佛没有什么能与它抗衡，这刀有一种难言的神性，时间?光彩在它面前都要失色。
<br/>　　它此刻被唤醒，行动了起来，切开大宇宙，直接来到混沌道君所在星域。
<br/>　　锵！
<br/>　　天刀复苏，内部的神邸苏醒，如同一尊或者的大帝，睥睨四方，不可仰视。
<br/>　　轰！
<br/>　　突然，这片星空被切开。一把巨大的天刀飞了出来，星辰一颗接着一颗的炸开。浩大的帝威轰动星空。在这一刻，一些绝代高手都纷纷大吃一惊，他们心中若有所感。
<br/>　　“不死天刀！”有人认出这柄天刀，大惊失色，纷纷觉得不可思议。
<br/>　　“有人成帝了吗？”也有人感受到那股浩大的帝威，匍匐在地，瑟瑟发抖。
<br/>　　“太……太恐怖了！”有人艰难的咽了一口唾沫，满心苦涩。
<br/>　　“天啊！……”
<br/>　　这一刻，宇宙中没人能够沉静得住，纷纷猜测这是怎么一回事。
<br/>　　五彩纷呈，流光四射，刀气如海，卷起万重大浪，扑灭星河，向着混沌道君压来。
<br/>　　“哼！”
<br/>　　混沌道君冷哼一声，拍手就是一掌轰出，普普通通，仿佛并没有出力一般，但谁也不能小看它的威力。
<br/>　　轰！
<br/>　　天刀被崩飞，倒飞出去。尽管天刀复苏的的实力极为强大，接近太乙金仙高手一击，但道君此刻可是混元高手啊，仅仅一柄刀，他还不放在眼里。他的实力早已恢复，此刻就算是此界天道，都休想将他封印，更不用说一柄实力仅达到太乙金仙巅峰的刀了。
<br/>　　“哧！”
<br/>　　天刀很强势，再次飞来，根本没有人主导，自己复活，所有一切都被它主掌，进行屠戮，刀芒强烈。
<br/>　　“哼！”
<br/>　　道君冷哼一声，表示不满。紧接着，一道淡漠的声音在虚空响起：
<br/>　　“禁锢！”
<br/>　　……
