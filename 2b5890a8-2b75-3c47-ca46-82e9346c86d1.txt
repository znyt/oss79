        &lt;h3&gt;第三卷第二百八十三章  父子共识&lt;/h3&gt;
<br/>
<br/>        凌辰送走了宁雅，心里那种幸福感突然就消失了，似乎一下子从云层深处跌落下来，在寒冷的街头打了一个哆嗦。
<br/>
<br/>        爱情很美好，现实也很残酷，他必须得解决掉眼前的麻烦，追妻之路漫漫，工作上也需得努力，董亚子就是厂家的眼睛。
<br/>
<br/>        打开手机，工作群里有很多信息，也有董亚子的电话，还有董亚子发的信息：“凌辰，我给你打了很多电话，怎么回事？当真只要美人？”
<br/>
<br/>        “凌辰，刘诚之发的邮件，你为什么只转发给何少华？”
<br/>
<br/>        “凌辰？？？？？出什么事情了？”
<br/>
<br/>        还有他爸爸的电话和信息：“小辰，怎么回事？你得给我一个解释。”
<br/>
<br/>        “你在哪里？为什么不接电话？我在宁家，商谈你们的婚事，速速回话。”
<br/>
<br/>        宁玥：“辰哥哥，你在哪里？”
<br/>
<br/>        宁玥：“辰哥哥，我在家里等你！”
<br/>
<br/>        凌辰看完微微冷笑。
<br/>
<br/>        他给董亚子回了个电话：“亚子，我这里确实是出了点问题，回头给你细说，我这会去解决一些问题，你等我。”
<br/>
<br/>        然后又给他的父亲打电话：“凌老头，你现在在哪里？”
<br/>
<br/>        凌思永听着儿子的电话，心里一喜：“这货还知道分寸。”
<br/>
<br/>        “我在你宁叔叔家。”
<br/>
<br/>        “我不管你在哪里，现在你必须听我把话说完，我和宁玥一分钱的关系也没有，我不会对她负责的，我爱的人是宁雅，宁玥去我们家也是因为她是宁雅的妹妹，你别再和别人一起坑你儿子了。”
<br/>
<br/>        “那你为什么提前不告诉我？”
<br/>
<br/>        “我怎么告诉你？八字还没有一撇呢。我告诉你，你现在不准在提起宁玥的事情，要不然我离家出走。”
<br/>
<br/>        “你这臭小子，你说什么呢？我可是你爸！”凌思永直接就骂出声了。
<br/>
<br/>        宁少天和兰乔、宁玥都听见了。
<br/>
<br/>        “你要是合着外人坑你儿子，这样的爸不要也罢。”凌辰气呼呼的说。
<br/>
<br/>        “你在哪里，我要去好好收拾你。”凌思永气呼呼的挂了电话。
<br/>
<br/>        “哎呀，宁总啊，不好意思，我这儿子被我们家娇惯得不成体统，有什么不对之处、误会之处敬请谅解。我先去找到他再说。”凌思永站起来就准备走。
<br/>
<br/>        宁少天和兰乔一愣，尤其是宁少天，这凌思永火烧火燎的从山西来提亲，现在二话不说就走了？
<br/>
<br/>        “行，凌辰现在在哪里？”宁少天问凌思永。
<br/>
<br/>        “问了不告诉我，好像我在强迫他一样，我这也是玥儿姑娘给我打了电话，我才急匆匆的赶来，在此之前没有和凌辰沟通，很是抱歉。”凌思永叹了口气。
<br/>
<br/>        “玥儿？”宁少天看了一眼宁玥和兰乔，俩人都不说话。
<br/>
<br/>        “宁总，我先去找那个不孝子去了，回头我们好好聊聊。”凌思永站起来就走，乌龙闹大了，仔细想一想，儿子这么多年洁身自好，哪有什么不负责任的表现？
<br/>
<br/>        “好，凌总慢走，回头我们在商讨一下后期合作的问题，孩子们的事情孩子自己解决，我们做老人的不用太过操心。”宁少天站起来相送。
<br/>
<br/>        看着凌思永出门，宁少天的脸立马乌云密布。
<br/>
<br/>        “说，怎么回事？”宁少天厉声呵斥。
<br/>
<br/>        “你喊什么啊？那么大声干嘛？凌辰那臭小子占了你女儿的便宜，你不但不为孩子讨公道，你还在这里大喊大叫？”兰乔冷笑着说。
<br/>
<br/>        “你什么意思？”宁少天语气森冷。
<br/>
<br/>        “字面上的意思。玥儿，这个家里已经没有我们娘立足之地了，走，妈妈带你走。”
<br/>
<br/>        宁玥泪流满面，她好不容易才喜欢一个人，怎么会是这样的一种结果？
<br/>
<br/>        宁少天看着兰乔母女哭哭啼啼，心里烦躁，却又不忍心了。
<br/>
<br/>        “你们站住，给我说说到底是怎么回事？”宁少天柔声问。
<br/>
<br/>        “你是个男人，这种话你让女孩子怎么说得出口，亏你还是个当父亲的。”兰乔伤心欲绝。
<br/>
<br/>        “妈。”宁玥扑在兰乔的怀里放声痛哭，她的爱情啊！
<br/>
<br/>        “好了，你们两个不要哭了，我会给玥儿做主的。”宁少天冷冰冰的说。
<br/>
<br/>        兰乔拍了拍宁玥的脊背：“玥儿，别哭了，你庆幸你有个疼你爱你的好爸爸，有爸爸给你做主呢，乖，别哭了。去，向爸爸道歉，以后别那么任性的到处跑了。”
<br/>
<br/>        “爸，我以后不到处乱跑了，我就呆在您身边，好好的孝顺您，伺候您，被别人欺负了就欺负了吧，反正我以后一直都会住在家里。”宁玥走过去眼泪汪汪的看着宁少天。
<br/>
<br/>        “说什么傻话呢？乖，别哭了，再哭就不好看了。”宁少天弄纸巾替宁玥擦去脸上的泪水。
<br/>
<br/>        “谢谢爸，爸，我爱您！”宁玥破涕为笑。
<br/>
<br/>        凌思永出了宁家大门，给凌辰回拨过去电话：“臭小子，敢威胁我，你死定了，说，你在哪里？”
<br/>
<br/>        “臭老头，你出来了？我在某薇酒店2308室，你过来。”
<br/>
<br/>        凌思永按照凌辰提供的地址找到凌辰，凌辰正在工作群里回复信息，忙得一塌糊涂。
<br/>
<br/>        “爸，您过来了，快坐下休息会。”凌辰为他父亲倒了一杯茶。
<br/>
<br/>        “你怎么回事，突然就跑到宁家提亲？”
<br/>
<br/>        “还不说那个宁玥打电话给我，说你们俩人已经在一起了，你会向她爸爸提亲，还说她想要在正式一点的，希望我能到她家，向她父亲正式提亲。”凌思永皱着眉头，这个宁玥真是一个厉害的角色，连自己的清白都可以不要。
<br/>
<br/>        “我只是想告诉你，我爱的人是宁雅，我想你也能看得出来，所以拜托你动脑袋想一想，这么多年以来我有毁人清白吗？”
<br/>
<br/>        “可是宁家好像不待见宁雅。”
<br/>
<br/>        “我娶的人不但要漂亮还要有能力，这样才能和我同甘共苦，荣誉与共。宁家不待见宁雅正是我的好机会，此时我不是应该站在她的身边吗？”凌辰微微一笑。
<br/>
<br/>        “现在已经搞成这个样子了，该怎么回旋？”
<br/>
<br/>        “你又没有正式向他们提亲，这个事情先不管了，你没有和宁家有什么要合作的吧？”
<br/>
<br/>        “准备后期有合作，我看如果想要合作的话，你怕是得娶宁玥了。”凌思永很是纠结。
<br/>
<br/>        “不好意思，如果你拿我的幸福作为交换的条件的话，我会脱离出来的，你不用担心我没有能力。”凌辰冷冷的一笑。
<br/>
<br/>        “你这孩子说什么话呢，这么多年我有拿你的幸福做交接条件吗，我只是不知道该如何向宁少天开口啊，一个是人家的大女儿，一个是小女儿，你怎么就这么渣呢？”
<br/>
<br/>        “不是我渣，是哪个宁玥渣，你希望我娶那样的人吗？”
<br/>
<br/>        “宁玥都是这一个样子，哪个宁雅能好到哪里去？你呀就什么都先别想了。和我一起回家！”凌思永想抓着自己的儿子一起回家。
<br/>
<br/>        娶个媳妇管三代，这个一定要考虑清楚再说。
<br/>
<br/>        他们老凌家虽然没落了，但是娶儿媳妇还真得如凌辰所说，不能只会耍心机，没能力，这事情需得从长计议，如今凌家还得靠儿子支撑着，这个臭小子一直是说一不二的，别偷鸡不成蚀把米，儿媳妇没娶成儿子也丢了。
<br/>
<br/>        “宁雅和他们不一样的。”凌辰为宁雅辩解。
<br/>
<br/>        “那个宁雅现在在哪里？”
<br/>
<br/>        “北京，某源公司。”
<br/>
<br/>        “最近这几车兴起的搞得轰轰烈烈的信息化公司？”凌思永吃惊的问。
<br/>
<br/>        “你这老头还能与时俱进嘛。”
<br/>
<br/>        “宁雅昨天就是代表某源公司来和她父亲谈判？”凌思永再一次发问，这一次是皱着眉头。
<br/>
<br/>        “聪明了！”凌辰看着他的父亲微微一笑。
