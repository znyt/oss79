　　在两人相视中，炎良盯着高漫秋开口：“我在你身上嗅到了水族的气息。”
<br/>　　高漫秋没有因为眼前站着的人是火族少主就退缩，哪怕对方无论是身份地位还是血脉，都高过了高漫秋一头。
<br/>　　“哦，我见了水族圣女。”高漫秋漫不经心的回答，与火族少主的咄咄逼人形成鲜明对比。
<br/>　　火族少主炎良眯起了眼睛，淡淡道：“你很放肆啊……”
<br/>　　高漫秋轻笑，随意将一缕发丝撩到耳后：“圣女果然不愧是得到了上古传承之人，她请我喝了一杯茶。”
<br/>　　炎良瞳孔缩了缩，水族圣女是个特殊存在，能让整个毫无战力的水族被阴阳谷的所有家族忌惮千年，一切都是因为圣女。
<br/>　　而这种可怕的存在，竟然请高漫秋喝了一杯茶，实在太不可思议！
<br/>　　尤其是高漫秋身上所携带的强大气息，的确说明她在不久前与水族圣女相距很近。
<br/>　　炎良一时半会儿并不能想通其中的关键，他只是在看到高漫秋的第一眼，便觉得此女不简单，但到底哪里不简单，他并没有水族天生的敏锐感，察觉不出来。
<br/>　　所以炎良只能试探。
<br/>　　“告诉我那两家人来了多少，阳谷有哪些需要注意的地方？我就告诉你她为什么会请我喝茶。”这时候的高漫秋直接开口，她很聪明，了解炎良留住她的目的，也不打算在这里浪费时间。
<br/>　　炎良深吸一口气，看向高漫秋的眼神再次不同，他已经在心中将这个女人放在了重点关注位置。
<br/>　　“高家这一代的嫡系很真是不简单。”炎良说着，紧接着道：“罗家来了八十人，各个比你高家的人弱；白家来了五十人，各个比你高家的人强。”
<br/>　　高漫秋听着这番话，暗自在心中作比较，而后瞥了眼不远处的一滩黑色木炭样的粉末，又问：“离开阴谷时他们剩下多少人？”
<br/>　　炎良在心中又给高漫秋打了一个高分，那一团木炭粉末真是他一团火烧死的白家某旁支成员的残渣，高漫秋什么时候不看偏偏在这个时间明显的看了眼，显然就是在展示给炎良看，告诉他她知道火族出过手。
<br/>　　“你还没告诉我她为什么请你喝茶。”炎良气势开始逐渐强硬·起来，并不被高漫秋左右。
<br/>　　高漫秋浅笑：“到底还是上古火族，无论是传承还是地位，都底气十足。”
<br/>　　说罢，高漫秋退后一步，朝着炎良微微一鞠躬：“高家神女高漫秋，见过火族少主。”
<br/>　　哗——
<br/>　　一股飓风刮起，吹乱了高漫秋的发丝，更吹的火族不少人都震惊无比。
<br/>　　刚刚谁说了什么来着？
<br/>　　能与炎良平等对话的四偏门嫡系，除非是神女。
<br/>　　而现在，高漫秋她竟然真的是神女！
<br/>　　四偏门多少年未出神女？
<br/>　　自从千年前阴阳谷大乱，所有家族被迫离开这里，回炉守护者失踪后，血脉的诅咒也就此在四偏门中滋长，作为惩罚想要再出神女的条件尤其苛刻。
<br/>　　这是压制，也是严惩。
<br/>　　炎良的眼神瞬息万变，露出原来如此的目光，他难得目光和煦起来，上前一步将庄重行礼的高漫秋扶起，整个过程中傲气与压迫感不再，取而代之的是一种欣慰。
<br/>　　“你不必再对我行礼，你们高家当年果然有苦衷。”炎良语重心长的开口。
<br/>　　高漫秋一愣，抬眼看向炎良的目光有些惊讶。
<br/>　　“水火两族知道的法则和条约比四偏门多的多。”炎良微笑着，解释道：“四偏门千年内不出神女属于正常情况，可在千年后阴阳谷重启阶段出现神女，还是在末世纪元中忽然出现的，说明了两点。”
<br/>　　“一，守护者回来了；二，出了神女的那个家族，当年并无罪孽。”说着炎良伸出两根手指：“以上两点缺一不可。”
<br/>　　高漫秋双目闪出一抹辉光，再次低头恭顺道：“高家，从未背叛回炉一门，当年忍辱负重，不过是想要调查更多真·相。”
<br/>　　炎良忽然哈哈大笑，看向高漫秋的目光意味不明：“除了调查真·相，恐怕你们高家还怀疑其中有水火两族甚至制裁一门的参与吧？”
<br/>　　高漫秋眼神内精光一闪，不承认，但也不否认。
<br/>　　“没想到最终厉害的竟然是高家。”炎良再次感叹。
<br/>　　而后他忽然双指一燃，以火为笔在半空中画出一张张肖像，很快十一个女人的肖像便呈现而出。
<br/>　　炎良问道：“告诉我水族那个女人长得像这里的谁？”
<br/>　　高漫秋仔细看了眼，毫不犹豫的摇头：“没有一人长得像她。”
<br/>　　炎良顿时皱眉，弩定道：“不可能，这是水族全部的十一脉圣女！”
<br/>　　“水族从双尾的初代开始之后，第二代便分成了十一脉不同的人鱼，之后的每一代圣女都跳不出那十一脉的圣女传承，相当于让这十一脉圣女的记忆和能力永存，确保毫无战力的整个水族在上古各大种族中依然立于不败之地，到现在那十一脉的圣女骸骨还埋葬在阳谷的水族墓地呢！全星河的人都在找这十一个女人的棺材，谁知道咱们的回炉大人偏偏反其道而行之，将她们埋葬在谁也想不到的这个落后星球，甚至还只是这个星球上的一处极小的地域里。”
<br/>　　“简葬却很好的得以保全！而十一后则是一个轮回……”
<br/>　　说着炎良忽然停下，双目瞪大不可思议，同时半空中的那十一个肖像‘哗’的一声消散不见，他双手开始快速舞动起来，不一会儿一副全新的肖像图呈现，这回他画的更细致更清晰，每一个细节都用细细的火丝勾勒而成。
<br/>　　“你再看，是不是像她？”炎良紧紧盯着高漫秋。
<br/>　　高漫秋仔细望去，紧接着瞳孔一缩：“很像，不是外表……”
<br/>　　“是气质！是感觉！”炎良的情绪有些过分外放，有些夸张。
<br/>　　高漫秋有些莫名，但还是点点头道：“感觉很像，很奇怪，明明两个人是完全不同的相貌，却给我感觉像是同一个人，或者说，姐妹？母女？”
<br/>　　此时的炎良已经完全听不到高漫秋在说什么，震惊的恨不得抱住脑袋怀疑人生：“怎么可能？她回来了？究竟怎么回事？怎么回事！”
