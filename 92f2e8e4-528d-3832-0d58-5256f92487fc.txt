　　天照3000没有飞向阴阳谷，而是反向朝着海岸飞行，最终停在了一处孤岛上。
<br/>　　高少辉的设想很好，在没有人帮助的情况下，楚涵不可能离开这座岛，也就不可能卷入到阴阳谷的纷争中，就不会有丧命的危险。
<br/>　　为了避免一切变数，高少辉放下楚涵后，只给楚涵留下了些许生活用品和药物，就直接驾驶着天照3000离开，带着旺财停驻在了一个临近的小岛上。
<br/>　　楚涵已经是高阶新人类，高烧中参与战斗的确危险无比，但如果有足够的时间休息，也一定会恢复过来，所以在这座安静的小岛上高少辉很放心，决定每天前来巡视情况即可，直到楚涵彻底恢复。
<br/>　　这期间楚涵的情绪和要求，高少辉已经做好完全无视的决定！
<br/>　　此时在临近的另一小岛，高少辉正用岸边的礁石现场烤着鱼，日子似乎回到了当年与楚涵初识的时候，一样的随地烤肉随处而睡。
<br/>　　只是这时候在高少辉身边的只有旺财，并且高少辉每烤出一条鱼，旺财就吃掉一条，最后干脆守在用礁石做烤盘的地方，边烤边吃。
<br/>　　高少辉一言不发，就这么将一条条鱼烤熟，再下海捕捞，循环反复。
<br/>　　终于在旺财吃爽吃饱，挺着个大圆肚子躺在沙滩上小憩时，高少辉才打破沉默开口：“你是楚涵从阴阳谷内带出来的？”
<br/>　　旺财翻了个白眼，张口出声：“大·爷我看上去这么廉价？”
<br/>　　高少辉沉静了一会儿，点头道：“的确，阴阳谷内的守护者，并不会开口说话，而灰色绒兔，听说是皇族？”
<br/>　　旺财剔了剔牙：“白允儿也说过这话。”
<br/>　　高少辉瞳孔一缩，定定的看着旺财：“那么你果然……”
<br/>　　“别问我，我不知道，我失忆了。”旺财打断高少辉，而后翻了个身就开始呼呼大睡。
<br/>　　高少辉朝旺财的方向看了很久，心中百转千回，守护者有两种说法，一是白色绒兔的阴阳谷守护者，二是回炉的守护者。
<br/>　　前者肯定不是旺财，而后者是否为灰色的绒兔皇族，却没有任何资料记载，全部在历史的长河中因为这样那样的原因被毁。
<br/>　　四偏门的神秘家族之白、肖、高、罗，在这千年间也并非相安无事，各家族间的明争暗斗甚至正面冲突都不少，只是世人所知甚少。
<br/>　　他们想要争夺的东西，也正是和守护者三个字有关。
<br/>　　只不过常年的纷争让罗家最终退出了华夏地带，远离到另一大陆开启新的发展，剩下的三家持续争相掠夺，并为了家族血脉浓度开启了不同的繁衍模式。
<br/>　　白家走了偏门，虽说诡异扭曲，但血脉浓度的保存却的确为四家中翘首。
<br/>　　高家采取了竞争模式，让全部的旁支成员都参与了进来。
<br/>　　罗家则大力发展了外围成员，让他们的家族成员数量不断壮大。
<br/>　　唯有肖家最为古怪，他们似乎在血脉浓度上什么都措施都没有，甚至在肖七之前，就有了不少半血统的成员，可肖家又与罗家追求数量不一样，既没有将半血统的肖家成员发展成一批战力，也没有将剩下的纯种血统者统一聚集。
<br/>　　就这样毫不在意的，甚至半血统者与嫡系结合也不反对，让肖家的血脉浓度变得越来越低，直到末世爆发之前连最起码的自保能力都没有，就惨遭想要回到华夏的罗家暗算，以至于全家族的覆灭，只留下了最后一名混血儿。
<br/>　　高少辉不知道自己为什么自己的思维会忽然围绕起肖家转，明明这个家族已经不足以让人重视，明明他们只剩下了最后一个成员，还是一个不男不女的怪物。
<br/>　　“四偏门，四偏法……”高少辉喃喃自语着，皱起眉道：“肖家的偏法不就是性别转化么，因为鸡肋所以最弱。”
<br/>　　一旁的旺财被他吵醒，起身眨了眨眼睛问道：“什么性别转化？”
<br/>　　“对了。”高少辉看着旺财问道：“楚涵是怎么认识肖七的你可知道？”
<br/>　　旺财点头：“那时候虽然我还不在楚涵身边，但楚涵跟我提过这件事，并且还是两次。”
<br/>　　高少辉猛然坐直了身子，洗耳恭听，他忽然想到很多不合常理的地方，最明显的一点就是四偏门的四大家族，每一个家族都有各自的最隐秘偏法，可偏法除了返祖之外，还有另一个早就存在的秘术。
<br/>　　白家有秘术，高家也有，罗家更有，但是肖家……
<br/>　　肖家似乎人人都反了祖，每一名家族成员都在男女之间转换自由，所以在见到肖七的第一眼，高少辉就让楚涵少与之接触，并与肖七争锋相对的吵起来大骂人妖变·态。
<br/>　　只是所有人，却偏偏从未听说过肖家有什么秘术！
<br/>　　因为肖家太弱了？
<br/>　　没错，都弱到灭族，连个秘术都没有……
<br/>　　但高少辉却忽然不知为何，觉得这其中有诡异！
<br/>　　旺财回忆了一翻，道：“楚涵第一次说起与肖七的相识，是说他的修罗战斧就是肖七指引他得到的，他很奇怪肖七为什么选了他。”
<br/>　　“第二次提前肖七，是知道了修罗战斧的制作者叶默的真实身份后，他在奇怪肖七当年为什么选中他的同时，还奇怪肖七既然已经知道地下基地内的叶默就是远古的叶家人，却为什么没有任何行动，让楚涵拿走修罗战斧，让白允儿带走那双手套，还放任楚涵给叶默找了三个徒弟。”
<br/>　　旺财说完后，就愣愣的看着高少辉：“就这两次。”
<br/>　　高少辉听完后先是愣了一会儿，紧接着一跳而起，神情好似受到了极大刺激：“叶默！哪个叶默？叶家人？修罗战斧，那那那……那把斧头是肖七带着楚涵拿到的？肖七？！”
<br/>　　“是啊。”旺财看着高少辉的情绪暴走，有些莫名其妙。
<br/>　　“你不觉得很奇怪吗？”高少辉难以抑制自己的情绪，死死的盯着旺财，眼神也一点点深邃和犀利起来：“那个肖七，是不是知道的太多了点？”
<br/>　　“不奇怪。”哪知旺财平静异常，道：“楚涵从未觉得肖七简单。”
