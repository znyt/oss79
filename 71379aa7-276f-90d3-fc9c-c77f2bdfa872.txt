        回到云城后，童筱颖搬到了外婆家居住，但她因为跟傅昼景离婚时是净身出户，所以她卡里没有多余的钱供她下半生过日子，迫使她又不得不出来找工作。
<br/>
<br/>        她想先赚点钱回去，把外婆的老屋重新修筑一下。
<br/>
<br/>        这一次，童筱颖没有进入什么大公司大企业里去工作，而是给一个私人建筑设计工作室的老板当临时助手。
<br/>
<br/>        工作室里老板和员工，就他夏凌焰一个人。
<br/>
<br/>        而且，夏凌焰还是个喜欢留着络腮胡渣，且性格脾气十分古怪，习性又风流的设计师。
<br/>
<br/>        夏凌焰是外地人，他是以私人的名义，接了云城里一个古建筑物设计的项目。
<br/>
<br/>        所以，他来云城做实地考察，了解一下这里的人文风情，寻找一些灵感。
<br/>
<br/>        他来到这里后，发现自己对这里完全不熟悉，于是在网上发了招聘贴，临时招一个本地的建筑设计师来辅助他。
<br/>
<br/>        碰巧，童筱颖在网上看到了他的招聘贴，就来他这里面试了，而且，一试通过。
<br/>
<br/>        因为要把建筑模型先赶制出来，童筱颖这几天都陪着夏凌焰住在了这家酒店里。
<br/>
<br/>        眼瞧着模型快完工了，这个夏凌焰却自己偷起懒来，而且还不知道是从哪里弄来的两张邀请卡，拉着她一起跑到这晚宴上来蹭吃蹭喝。
<br/>
<br/>        但是，童筱颖可没这心思在这里蹭吃蹭喝，只想赶紧把事情做完，赶紧拿了钱，寻找下一家需要她打临时工的地方。
<br/>
<br/>        “啧啧啧，筱颖，你这么说，让大家都误会的！”夏凌焰啧啧嘴，暧昧不已地说道。
<br/>
<br/>        他留着络腮胡渣，眼窝深邃，明明五官立体英俊，但是他的言行举止却让他看起来像一个痞里痞气的混混。
<br/>
<br/>        “你要是不做，我也不做了。我现在想回家，明天休假。”童筱颖顿时没好气地说。
<br/>
<br/>        夏凌焰立即搂住童筱颖的肩膀，咧嘴一笑：“要不，你先回包房去休息吧！这么晚了，哪有公车回你那个山窝窝里去。”
<br/>
<br/>        “算了算了，我自个儿先去把模型做完！”童筱颖不得不挪开夏凌焰的手，没好气地说道，“我明天一定要休假！”
<br/>
<br/>        夏凌焰向童筱颖眨了眨左眼，比了个“OK”的手势，然后推了推手，笑道：“好好好！你赶紧去吧！你今晚做完了，我明天就让你休假。现在，让我再玩一会儿。今晚，来参加这个晚宴的美女实在是太多了。我还未认识够了！”
<br/>
<br/>        “……”童筱颖简直无言以对，只好独自一个人先去包房里做模型。
<br/>
<br/>        夏凌焰订的那间包房是个小总统套房，但仍旧是标准的两室一会议厅。
<br/>
<br/>        童筱颖回到包房里，便开始继续动手做模型。
<br/>
<br/>        夏凌焰则继续游刃在各色美女之间。
<br/>
<br/>        古建筑的模型并不太好做，童筱颖之所以专注，还有个原因是她想在外婆的老屋旁，自己动手盖一座只有木头和土砖，靠鲁班锁来固定，没有钢筋混凝土的房子。
<br/>
<br/>        童筱颖好不容易把模型做完，结果肚子却在此时此刻“咕咕”地叫嚣了起来。
<br/>
<br/>        云城的夜市现在比以前热闹多了，各种夜宵摊更是营业到第二天早上六点。
<br/>
<br/>        童筱颖放下手中的模型棍，非常满意自己做的迷你版的亭台楼榭，随后才想起要摸摸肚子，打算去买点夜宵回来吃，于是走到了房门口，拿起了房卡准备出门。
<br/>
<br/>        她刚把房门打开，一个黑影突然从一侧扑倒在地，将她十足地吓了一跳。
<br/>
<br/>        童筱颖拍拍胸脯，定了定神，蹑手蹑脚地移步到门外，探出头来瞧了瞧。
<br/>
<br/>        当她发现是个男人后，便小心翼翼地蹲下身去，伸出手来轻轻地摇了摇这个侧躺在地上背对着她的男人。
<br/>
<br/>        “喂，先生，你没事吧？要不要送你去医院？”
<br/>
<br/>        童筱颖弱弱地问道，随着她摇晃的动作，这个男人突然转过身来，平躺在地上面对着她。
<br/>
<br/>        大、大哥？！
<br/>
<br/>        童筱颖瞪大了眼睛，表情惊愕了。
<br/>
<br/>        她完全没料到，自己会在这种地方，再次邂逅三年未见的大哥傅夜沉。
<br/>
<br/>        童筱颖看着酩酊大醉的傅夜沉，只见他眉宇轻轻地皱起，俊脸上露出了痛苦的表情。
<br/>
<br/>        大哥好像向来千杯不醉的吧！他今晚又何以醉成这样？
<br/>
<br/>        “大哥，你醒醒啊！”童筱颖随之拍了拍傅夜沉的脸，发觉他的脸颊有些发烫，但不打紧，只是体内的酒精在发挥作用，让他的热血沸腾罢了。
<br/>
<br/>        见傅夜沉没有要醒过来的迹象，童筱颖又转而在他的裤袋里摸了摸，总算是摸出一张房卡来。
