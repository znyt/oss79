　　事实证明，倭人其实是很好骗的，而且脑子还很单纯，因为晚上回去之后唐义就看到一个倭人在等着自己，操着一口诡异的大唐官话，还非要让唐义把他带进科学院。
<br/>　　唐义对此表示无语，无奈的告诉他，大唐科学院不收身高不足五尺的人，除非你有极高的天赋或者能力，会倭语不算。
<br/>　　倭人表示了对此事的怀疑，但唐义表示，如果带他去了，出了问题由他负责，并且由此引发的一切后果同样也要由他负责，这才打消了那个倭人的想要冒充他儿子的念头。
<br/>　　之后倭人又问起唐义有没有探听到什么，唐义又随便找了一个借口敷衍了过去。
<br/>　　……
<br/>　　一晃三天时间已经过去，唐义在下班之前最后见了一次程处弼。
<br/>　　“你放心回去吧，你的邻居现在已经全都换成了我们的人，只要今晚他们敢去，老子保证他们一个都跑不了。”程处弼模仿着老程的凶狠样子，就好像那几个倭人已经在他面前一般。
<br/>　　唐义深深吸了一口气：“程将军，那我回去了。”
<br/>　　“去吧，今晚你不用担心，只要他们敢现身，老一定会让他们有来无回。”
<br/>　　“好的。”唐义看了看程处弼身后二十余个腰上挂着左轮手枪，身上背着新式步枪的手下，胆气不由一壮，重重一点头转身离开。
<br/>　　自从大唐开始推行免费教育以来，很多类似于唐义这样的人经过了数年初小的学习，然后走上了工作的岗位。
<br/>　　他们因为错过了最佳的学习年龄，无法继续深造，但这数年的学习却让他们懂得了许多的道理，忠孝信悌，礼义廉耻成为他们最为看重的东西。
<br/>　　倭人自以为已经成功将唐义控制在手中，实际上他们却不知道，这个看起来呆呆傻傻的家伙并不那么好骗，三天时间装傻充愣的硬是把那些倭人玩弄于股掌之间，就连那个自视甚高的倭人王子也不例外。
<br/>　　所以当天晚上唐义一如继往的回到了自己的住处，如往常一般点起灯火。
<br/>　　房间中那个与他联系的倭人正无聊的发着呆，看着盯着那并不大的烛火想着心事，一点都没有注意到，隔壁隐约间传来的敲击声。
<br/>　　但是他没有听到，唐义却听到了，那是他和程处弼他们约定的暗号，只要暗号响起就代表着那些倭人已经来了，让他自己注意安全，如果不能降伏房间里的家伙，那就把人引到外面。
<br/>　　不过当唐义看到那个还在发呆的倭人时，他不禁暗笑程处弼将军那帮人的小心，一个小个子的倭人，能有个屁的危险。只需要一块板砖就可以把他干挺喽。
<br/>　　……
<br/>　　而在外面的巷子里，倭人青年此时正带着他的几个护卫走了进来，一边走一边还讨论着如果唐义没有得到他们要的消息要怎么办，到底是杀了他还是带着他一起走。
<br/>　　可是，就在他们走到唐义的家门口，刚刚想要推门而入的时候，便听到里面传来一声沉闷的敲击声‘嘭’，接着就是有人倒地的声音。
<br/>　　不好，出事了！这是倭人青年的第一个反应，但还没等他决定是冲进去还是转身跑的时候，四下里已经‘呼啦’一下闪出无数人影，一支支闪着寒光的步枪就那么顶在他和他那几个手下的脑袋上，一个风骚的声音自小巷的再深入传来：“老子早就听说这段时间城里不太平，有一**细出没，没想到今天出马果然抓了个正着！”
<br/>　　倭人青年此时也终于反应过来发生了什么，立刻连声辩解试图蒙混过关：“我们是正经游客，这位将军，我们是正经的游客啊，是从倭国赶来参加马球比赛的。”
<br/>　　但是黑暗中的那个声音似乎并没有上当，听着他的狡辩冷冷一笑说道：“是么？参加比赛参加到咸阳来了？参加比赛参加到大唐科学院去了？莫不是你真当本将军那么好骗？”
<br/>　　“呃，不，不不，我们没有……。”
<br/>　　“有没有去情报科说吧，看看到了那里你是不是还有这么多的理由。”黑暗处走出一个高大的身影，不是那程处弼又是何人。
<br/>　　倭人青年听到情报科，浑身就是一个哆嗦。
<br/>　　他可没有忘记在来大唐之前听到过的那些传说，像什么红衣之类的那就不用说了，传说中的人物就算是想见都见不到，而且大唐这么多人，也不大可能见到，见到了最多也就是个死，没啥大不了的。
<br/>　　但是大唐情报科那可绝对不是什么好地方，人进去了就算是想死都难，不把小时候床过几次床数清楚绝对别想出来。
<br/>　　所以一听程处弼说要把他送进情报科，这货立刻就怂了，把头摇的飞快坦白道：“别，将军，我真的没有别的意思，我真的是好人啊，我父亲是秦吉胜，是东倭道的大都督，您把我放了吧，我就是好奇大唐的风土人情，所以才出来看看，以后我真的不敢了，真的不敢了。”
<br/>　　不过程处弼显然并没有给他爹面子，洒然一笑说道：“老子管你爹是谁呢，今儿在这儿逮住你，你就和奸细逃不开关系。”
<br/>　　“我真不是奸细啊！”
<br/>　　“你是不是奸细你说了不算，老子说的才算。”程处弼似乎也不想跟这个倭人闲扯蛋，从腰间掏出左轮手枪往倭人青年嘴里一塞：“你院子里的同伙把什么都交待清楚了，今天就是他引着我们过来的，所以就算你再狡辩也没用，再不承认信不信老子一枪打死你！”
<br/>　　“呜呜……”倭人青年嘴里塞着粗大的枪管，连连摆手，心里已经把那个手下骂了个狗血淋头，发誓只要自己能够活着回去，非得杀光他全家不可。
<br/>　　当然，唐义也在要必杀之列，这个唐人应该早就已经把自己等人出卖了吧，之所以那个大唐的将军没有出手应该也是在等一网打尽的机会。
<br/>　　只可惜，这次行动什么都没有得到，估计就算有命回去，老爹那里也会有些麻烦。
<br/>　　不过为了保命现在还是把所有的事情都承认了吧，否则三木之下该说的还是要说，何必去受那份罪呢。
