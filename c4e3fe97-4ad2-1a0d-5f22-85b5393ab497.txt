　　罗杰无法反驳，只得与白忧一同直径朝着阳谷走去，一路上一百多人的队伍紧密的排着，谁也不敢走散，就这么一路走下去没有多久便直接顺着底下的拼图磁场，到达了阴谷出口处。
<br/>　　阴谷的出口，便是阳谷的入口，这是所有神秘家族成员都知道的事。
<br/>　　而在这安静到没有一丁点声音的陌生地带，对于普通人踏入后的好奇，神秘家族的成员更多的则是恐惧。
<br/>　　因为他们知道，阳谷是一座墓地，巨大的墓地，埋葬着千年前的死人，以及所有受到制裁的成员。
<br/>　　罗杰有些不敢相信一路走来的顺风顺水，他吞咽着口水，呐呐道：“不过没想到这里荒芜成这样，一丝生命的气息都没有。”
<br/>　　白忧像是在看白·痴一样的看着他：“早就荒废了一千年，怎么可能有生命。”
<br/>　　罗杰一点没有看到白忧的眼神，自顾自的说道：“按理说，阴谷的入口和出口……”
<br/>　　他的话并没有说完就此打住，但却让白忧面色大变，很显然的一点，这两个神秘家族嫡系成员都很清楚，阴谷的入口由水族掌管，而出口则由火族掌管。
<br/>　　他们一群人平安进入阴谷，至始至终没有发生任何意外，理应能够说明水族人并不在阴谷，说不定根本就已经没有水族，毕竟在这些心里有鬼的神秘家族看来，当年的事水族人根本不可能不计较，也不会让他们就这么安然进来。
<br/>　　只是……
<br/>　　眼下的这个地理位置，正是阴谷出口之地，也便是属于火族掌管的地方。
<br/>　　虽说他们不可能发现的了火族旧址，可火族人如果存在，就一定能够知道他们的踏足。
<br/>　　“水族不在，火族应当也不在吧？”一名白家的旁支成员弱弱出声，面上有着一丝害怕。
<br/>　　白忧镇定了一下，道：“我说过，水火两族早已不复存在，不用心慌！”
<br/>　　白家人都开始强行给自己洗脑，鼓足勇气不去想当下的地理位置有多危险，更何况如果火族人在此，他们早已踏入火族掌管的地盘，有事发生也早就发生，不会等到现在还什么都没有。
<br/>　　“那，继续走？”罗杰随口问道。
<br/>　　白忧看了他一眼，忽然后退了半步，道：“你们先。”
<br/>　　一瞬间，白家的五十人都集体让出一条道，强势又逼·迫的目光望向了队伍后方的罗家人。
<br/>　　罗杰傻了眼，震惊喊出声：“什么意思？！”
<br/>　　显然这种事情属于炮灰才做的，前方就是阴谷与阳谷的交界地，也是最为让他们警惕的地方，白忧的忽然决定，让罗家人都措手不及，一个个表情呆愣站在原地。
<br/>　　白忧理直气壮：“进谷的时候是我们白家人先，出谷由你们罗家人先，很公平吧？”
<br/>　　“哪里公平？！”罗杰大怒：“前方有什么谁也不知道，万一危险呢？”
<br/>　　“难道入谷之地的危险就可以推算？”白忧冷笑：“你还不是一样怀疑有水族人守着？”
<br/>　　罗杰当即无话可说，白忧的这一招用的实在漂亮，尤其是他们得到唯一有关水火两族的消息，还是来自于已经死去的罗梵，而罗梵遇到的只是可能为火族人，至于水族还未有任何存在的线索。
<br/>　　无论从哪个角度看，进谷的危险都要比出谷小，而白忧在两家的队伍相聚之时却对这个只字不提，等到了眼下才说明这一点，显然是将罗家人当成了枪使。
<br/>　　罗杰盛怒，可面对战力更强的白家人，以及此时已经失了先机的情况，他也毫无办法，只能冲着罗家人招了招手，示意由他们八十人先走。
<br/>　　八十名罗家人哪里愿意涉险，只是在白家人虎视眈眈的眼神，以及罗杰强硬的目光中不敢反抗。
<br/>　　他们拖着步伐一点点向前移动，渐渐便于白家人交换了位置，只是在这一过程中，并非罗家人的楚云天被所有罗家人下意识放在了队伍最前方。
<br/>　　哪怕是炮灰，先死的炮灰也要是外人才行！
<br/>　　罗杰对于这一情况没有发表任何意见，他带着楚云天进来是有碰运气的想法，可若与罗家人的性命相比，楚云天的这条命在他眼中则不值一提。
<br/>　　楚云天一路听着这两大神秘家族之间的谈话，哪怕原本所知甚少也根本没有轮廓，但他毕竟智力超绝，很多秘密都在一步步的分析中了解的七七八八。
<br/>　　此时站在这阴谷的出口地带，只觉得浑身脊背发凉，这帮人的确根本不把普通人的性命当回事，就像对待一只猫一条狗……
<br/>　　“前面的人，走啊！”已经有白家人等的不耐烦，出声呵斥。
<br/>　　“还不快走！”站在楚云天身后的罗家人也出声催促，此人同样紧张，已经满脸汗水。
<br/>　　楚云天深吸了一口气，直接往前踏了一步，然后第二步，第三步……
<br/>　　穿着厚重防护服的身影就这么一点点走远，最终消失在前方那像是迷宫一样的绿色植被地带。
<br/>　　看着楚云天就这么走了出去，没有任何意外发生，罗家人和白家人都松了一口气，罗杰紧张的心情也渐渐放松，白忧面上更是浮现了最为难得的笑容。
<br/>　　火族人，怕也早就灭绝了吧！
<br/>　　如此想着，罗家人开始朝着出口快步走去，比起水火两族带来的恐惧感，似乎那个埋着无数死人的阳谷，更让他们有安全感。
<br/>　　只是就在所有人都认为不会再有意外发生之际，忽的一阵‘莎莎’声响起，紧接着走在最前方的罗家人便看到，那一簇簇围成植被墙的树木，就这么莫名开始移动，原本清晰可见的道路渐渐被挡住，出口的笔直大道在变化！
<br/>　　惊恐，一瞬间侵袭了所有人的内心，尤其是白忧好似三观颠覆一般，眼神充满了不可置信。
<br/>　　“怎么……可能？！”他发出了此生最为情绪暴走的声音。
<br/>　　这一出口地带的大道变化，唯一的可能便是火族人在操控，也就是说他最为担心的事，是个事实！
<br/>　　可是刚才那个罗家人不是好端端的走了出去吗？
<br/>　　不是足以证明这里没有火族人吗？
<br/>　　到底是哪里不对！
