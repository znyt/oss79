　　在这炼药殿内，只剩下了这九龙神苍鼎。
<br/>　　不过这一尊九龙神苍鼎，要比起千万古圣丹，价值还要高。
<br/>　　这九龙神苍鼎，不止是一尊药鼎，更是一件灵器，无比强大的天级灵器，比起楚风眠得到过的法理之门，都毫不逊色。
<br/>　　天下百鼎，根据记载，其中的前七尊鼎，都是位列极道仙兵。
<br/>　　这九龙神苍鼎，虽然并非极道仙兵，可也是天级灵器之中品级极高的几件灵器了。
<br/>　　不用是用来对敌，还是炼药，都是一件至宝。

<br/>　　“收取！”
<br/>　　可就在楚风眠抓向这九龙神苍鼎的一刻，突然间九龙神苍鼎好像是察觉出了危险一样。
<br/>　　九尊龙头，同时咆哮而起，化为了一道火焰屏障，阻拦住了楚风眠的手掌。
<br/>　　“反抗？”
<br/>　　楚风眠眼神之中闪过了几分惊讶。
<br/>　　这九龙神苍鼎可是十万年前的灵器，现在应该已经是无主之物了，就算是一件品级在高的灵器，十万年的岁月，也足以将力量磨灭了。
<br/>　　现在居然是还在反抗楚风眠的炼化？
<br/>　　而且看起来，这九龙神苍鼎还蕴含这不弱的力量。
<br/>　　楚风眠目光一闪，死死的盯着这九龙神苍鼎，突然是在这九座龙头上，感觉到了一股独特的灵力。

<br/>　　这九龙神苍鼎上，居然还有烙印，也就是说，这九龙神苍鼎还被其他人所炼化这，不是无主之物？
<br/>　　怎么可能？
<br/>　　九龙神苍鼎，十万年都没有出世过，一直一来都是在这天元商会的废墟之中才对。
<br/>　　怎么可能现在还是有主之物？
<br/>　　“难道说天元商会的末代会长，还没有死？”
<br/>　　楚风眠突然想到了一个结果，在想到这个结果的时候，楚风眠都是吓了一大跳。
<br/>　　天元商会，现在早已经是消失在了历史长河之中了。
<br/>　　甚至是如今的时代之中，知晓这个名字的存在，都已经是寥寥无几了。
<br/>　　这九龙神苍鼎，肯定是没有离开过这废墟，而根据记载，九龙神苍鼎的最后一任主人，就是天元商会的末代会长。
<br/>　　现在这上面还有烙印，只有可能是属于天元商会的末代会长。
<br/>　　可如今天元商会的末代会长还活着，怎么可能十万年默默无闻，而且十万年的时间，纵然是天人境的人物，都未必可以活那么长久的岁月。
<br/>　　“不管了，先收取再说！”
<br/>　　楚风眠眼神之中闪过几分冰冷，不管这烙印是谁的，现在这九龙神苍鼎，既然是落到了楚风眠的手中，就是楚风眠的了。
<br/>　　就算是有麻烦到来，也不怕。
<br/>　　轰隆！
<br/>　　楚风眠运转起战龙血脉，一股股的龙力在楚风眠的身边凝聚而出，化为了一条条远古战龙的模样，同时蕴含着阵阵龙威，一股龙威顿时笼罩你在这九龙神苍鼎之上。
<br/>　　一时间龙吟咆哮，不止是在楚风眠的身边群龙咆哮，那九龙神苍鼎也不甘示弱，同样发出阵阵龙吟回应。
<br/>　　这九龙神苍鼎，据说是曾经一位绝代强者，斩杀了九尊远古战龙，将他们的肉身全部融入了其中，在加上许多珍贵材料，炼制而出的一件灵器。
<br/>　　楚风眠的龙威，非但是没有压制住这九龙神苍鼎，反而是让这九龙神苍鼎开始疯狂的反抗了起来。

<br/>　　楚风眠冷笑一声，远古战龙的确尊贵无比，难以降伏。
<br/>　　但是现在楚风眠可是有着更加克制远古战龙血脉的血脉！
<br/>　　始祖天龙血脉，化为龙威笼罩而下，顿时这九龙神苍鼎，仿佛就是碰到了克星一样，根本不敢抵挡。
<br/>　　始祖天龙，万龙之祖，对天下间一切龙族的威严，都被记载了血脉之中。
<br/>　　就算是这九龙神苍鼎，是一件灵器，但是一旦是融入了远古战龙的血脉，也要受到始祖天龙血脉的影响。
<br/>　　九龙神苍鼎的反抗，越来越弱。
<br/>　　抓住这个机会，楚风眠手心一动，一股灵力一分为九，向着那九座龙头上的烙印打了过去。
<br/>　　一道道的灵力打在上面，顿时楚风眠运转起吞天秘术来，化解这上面的烙印。

<br/>　　楚风眠感觉到烙印的力量，心中也轻松了许多。
<br/>　　这烙印证明，至少是九龙神苍鼎的主人，数万年的时间没有动用过这九龙神苍鼎了。
<br/>　　上面的烙印已经是开始腐朽了。
<br/>　　楚风眠炼化起来也轻松无比，催动吞天秘术，很快的便是将上面的烙印全部吞噬。

<br/>　　楚风眠心神一动，这九龙神苍鼎便是直接化为了一道遁光，进入到了楚风眠的身躯之中。
