　　第355章
<br/>　　这自然不是一套尽善尽美的制度，可相比帝国目前所行，实在已好了太多。
<br/>　　这是程大雷参考了前生诸多事情，再由大家一起讨论总结，最后拿出的成果，就连和珅都不免为之赞叹。
<br/>　　许多切入点令人耳目一新，却又能给和珅这样的人带来许多思考。
<br/>　　“这些都是大当家自己想出来的？”和珅难以置信问。
<br/>　　“难道不能么。”程大雷压低声音：“悄悄告诉你，这都是神仙梦中与我说的。”
<br/>　　“呃……”
<br/>　　和珅难免无语，鬼神之说糊弄糊弄一般人还可以，可对于和珅这种聪明人而言，他自然是一个字都不信。
<br/>　　如此，他只能认为是大当家太过低调，甚至在心中对程大雷的智慧程度重新估计。甚至有种『不愧是我看重的男人』的感觉。
<br/>　　而在这个过程中，蛤蟆寨和凉州城的谈判依旧在进行，这件事完全由和珅负责。
<br/>　　为了应对此人，凉州城拿出一整个谈判队伍，由宋伯康亲自领衔。可即便于此，还是在和珅这满清第一奸臣手中败下阵来。
<br/>　　凉州城接连送来五次赎金，一次比一次多，一次比一次抱的希望大，觉得可以将宋游渠等人接走。
<br/>　　结果却是同样的失望。
<br/>　　和大人将拖字诀用到极限，一次又一次压榨凉州方面的底线。最后被折腾得欲仙欲死，甚至自己都不知道怎么接受了和珅提出的条件。
<br/>　　宋游渠、薛问婷等人作为人质软禁在琴川关，蛤蟆寨保证他们的正常生活，不会被人欺负。凉州方面可以每月派人过来探望一次，当然，他们住在这里也不是白住的，凉州要向蛤蟆寨提供一笔生活费。
<br/>　　但程大雷看到最后谈妥的条件后，也是愣了半晌，最后用难以置信的眼神看着和珅。
<br/>　　“他们同意了？”
<br/>　　“嗯，很愉快的同意了，他们很满意目前这个结果。”
<br/>　　“他们怎么会同意，而且还很愉悦？”程大雷有些难以相信。
<br/>　　“因为这是他们努力争取到的结果。”见程大雷还不明白，和珅解释道：“要先将他们的期待值降下来，他们一开始是打算平安把人接回去，世间哪有这么好的事。我一说把这些人丢进矿场出苦力，他们立刻就同意了。”
<br/>　　程大雷咂舌不止，这和珅果然是个人才，这种智慧是读多少书，走多少路都学不来的，人家天生就该干这个。
<br/>　　“对了，还有一件事，凉州想要向我们订一批武器，数量很大，咱们要不要卖给他们？”
<br/>　　“卖，为什么不卖，要多少给多少。”
<br/>　　“那价格方面？”
<br/>　　“卖给戎族是什么价格，卖给凉州城就是什么价格，我们要做到童叟无欺，公平厚道。”
<br/>　　程大雷很满意和珅的存在，自从他加入山寨后，山寨的一切变得秩序井然，有条不紊的向前发展。
<br/>　　在和珅的主持下，蛤蟆寨就算和凉州城也保持了友好关系。至少，是表面上的友好。凉州城向程大雷订购大量的兵器，程大雷保质保量，双方合作得还算愉快。
<br/>　　陆陆续续有流民从外地迁来，他们在琴川关外安家，伐木造屋，开垦荒地，琴川关渐渐有了人气。
<br/>　　有人便有需求，需求促进商业的发展。大大小小的商队，不远千里而来，带来帝国特产的瓷器、绸缎、茶叶等物，换走草原上的兽皮，骏马，草药等物。
<br/>　　就连宋伯康也不得不承认，琴川关在程大雷手中，要比原先在莫鸣米手中时强上百倍。
<br/>　　当然，程大雷并没有被眼前的繁荣迷乱了眼，在发展商业、农业，制造业的同时，程大雷也没有忘了军事。
<br/>　　目前单单琴川关的驻兵就超过一万，程大雷没有着急扩充兵员，虽然他不是没有这个实力。对于这支部队，程大雷反复锤炼，堪称千锤百炼，目的是将他们训练成一支攻无不克战无不胜的精兵。
<br/>　　关外的各村各镇抽调壮年男子组成民兵，由关鱼亲自派去人训练。以这个时代的士兵素质，怕这些民兵的战斗力，都能超过各地城防军。
<br/>　　琴川关说是目前只有一万战士，可真如果进入战争状态，实际可发动的人数要远远超过这个数目。
<br/>　　光阴如白马，捻指消逝，转眼便到了这一年的秋天。
<br/>　　这一年的琴川风调雨顺，没有战火，没有灾荒，在琴川开垦出来的数万亩荒地都喜获丰收。
<br/>　　稻米、高粱、大豆堆满了各家各户的粮仓。琴川关要收两成的税，按每家每户的土地数量征收，相比如今帝国各种各样的苛捐杂税，这税收简直轻得不能再轻。
<br/>　　同时，琴川关又以固定价格收购大家手中的粮食，换得的钱财采购肉类、绸缎、茶叶，如此也促进了蛤蟆寨的商业发展。
<br/>　　一旦有了钱，人缘就会越来越好。这种事，放在一个人身上一样，放在一家势力上也一样。程大雷同周围的城池，边关都保持了不错的关系。
<br/>　　互相交易物资，各取所需，在商业上，琴川关拥有不错的信誉。
<br/>　　大家主动忽略了程大雷是个山贼的事实，而程大雷也的确是朝廷封的边关守将。在这种事情上，大家都知道该如何装糊涂。
<br/>　　即便和凉州城，程大雷也没有闹翻脸。宋伯康向琴川关采购武器，马匹，粮食丰收后又采购了大量粮食。如果不考虑，宋伯康的亲儿子还被关在琴川关做人质，双方的合作还是蛮愉快的。
<br/>　　而每月，宋伯康也差人去看望宋游渠，据去的人所说，宋游渠在琴川关并没有收到任何歧视或者虐待，享受一个基本居民的所有权力，除了不可以离开琴川关这一点。
<br/>　　过去一年时间，据去的人所说，宋公子发生了很大改变。原本他是十指不沾阳春水的性格，看上去谦逊有礼，其实一肚子高傲。可在程大雷的锤炼下，人却变得越来越扎实，学问见识也成长了很多。
<br/>　　有时候宋伯康甚至会想，也许自己儿子在程大雷手中调教一番，对他而言是福非祸。
<br/>　　
<br/>(本章完)
