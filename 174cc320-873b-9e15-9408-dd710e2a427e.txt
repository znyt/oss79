　　大清早。
<br/>　　涂家后院，一位清丽的女子做了一套简单的肢体运动，好看的眉头就紧蹙起来：“小弟真是越来越不像话了，又是一个晚上不回家”
<br/>　　涂小月早上起来，发现自己的弟弟不在房间，出来看看狗窝，见就小八孤零零的趴着，哪里有一条蛇的影子。
<br/>　　“这臭小子，变成蛇，夜生活还丰富了？”
<br/>　　三天两头的不着家，让涂小月操碎了心。
<br/>　　此时，她一个蓦然抬头，天边飘来了一朵绮丽的绿色彩云，好看极了，宛如一幅美丽的诗画。
<br/>　　看着看着，涂小月有点惊讶，因为这片彩云就是朝着她的方向飘来，而且近了近了，她发现根本不是什么彩云，而是一只比鸡要大上一寸的飞鸟。
<br/>　　远远望去，就好像是拖地的长裙，曼妙优雅的飘来，很美的一副风景线。
<br/>　　几个呼吸之后，涂小月的眼眸慢慢的睁开，眼珠子都快掉下来了。
<br/>　　这飞来的大鸟，貌似是一只孔雀。
<br/>　　这只孔雀就这样在涂小月呆滞的目光下，顺着飞到了后院之中，降落在了她的面前。
<br/>　　“你是我见过飞的最慢的鸟，还百鸟之王，丢不丢人，丢不丢鸡”
<br/>　　一条金银蛇从孔雀的背上跳了下来，很嫌弃的开口。
<br/>　　青枭气的想跟这条蛇拼命：“你自己会飞，既然嫌我飞的慢，干嘛还要死皮赖脸骑在我背上”
<br/>　　从佘山高峰一路飞下来，青枭发现飞着飞着，背上的蛇居然跳了下来，悬浮在了它面前，不要脸的说：“你能不能飞快点，你Y的比麻雀飞的还慢”
<br/>　　青枭无语了，再次见识到这条蛇的恐怖之处，还会飞？
<br/>　　刚想说，你自己会飞，就自己飞呗。
<br/>　　可说还没说，这条蛇再次骑在了它的背上，趾高气扬的指挥它飞的快点。
<br/>　　现在一落地，又嫌弃自己飞的慢，气的青枭要原地爆炸了。
<br/>　　深吸了一口气，赶紧先梳理羽毛，被这条该死的蛇都弄乱了。
<br/>　　青枭现在是鸟在屋檐下，不得不低头，成了别人的宠物，什么气都要受着。
<br/>　　它不明白自己的人生是怎么了，好端端的吃几颗蛇胆，就沦为了一条蛇的宠物。
<br/>　　这莫非是蛇胆吃多了，报应？
<br/>　　涂小月看到蛇说话了，她表示理解，因为这就是她夜不归宿的弟弟，但这只孔雀也开口说话了，这是什么鬼。
<br/>　　她再次陷入呆滞，茫然，脑子像一张白纸。
<br/>　　“姐，你傻站着干嘛，你弟弟回来，不高兴？”
<br/>　　涂小安发现了姐姐的怪异之处。
<br/>　　小月指了指在用嘴巴梳理羽毛，一脸傲娇的孔雀道：“小弟，你能跟我解释一下，这孔雀是怎么回事，刚才它好像说话了”
<br/>　　“姐，不用大惊小怪，这是我从佘山顺手收的宠物，用来当坐骑的”
<br/>　　涂小安说实话还真不知道身为百鸟之王的孔雀，飞的那么慢，跟龟爬似的。
<br/>　　这飞行速度可以跟拖拉机赛跑了。
<br/>　　小月听到自己弟弟漫不经心的话，诧异起来。
<br/>　　在百蛇出没的佘山收一只孔雀当坐骑？
<br/>　　她脑子有点转不过来，只是一双眼珠子死死盯着这只美丽异常的孔雀看，越看越喜。
<br/>　　真是好漂亮的孔雀呀，这辈子还是第一次看到活生生的孔雀杵在自己的面前。
<br/>　　青枭看到小月一直看着它，看的痴迷，它神气起来了，昂着头，挺着胸脯，来回转悠，炫耀着自己的美丽，长长的尾巴拖地长裙一样，优雅高贵。
<br/>　　被一条蛇嫌弃了，它要从人的身上找回一点百鸟之王的尊严。
<br/>　　“太漂亮了，小弟，我好喜欢这只孔雀”
<br/>　　涂小安一溜烟的游到了老姐的肩膀上，像坐凳子一样弯曲的坐下来，看着青枭道：“来，给我姐来一个孔雀开屏看看”
<br/>　　青枭恶狠狠的瞪了金银色一条，一抹轻蔑闪过，微微扬起头，傲娇的不要不要的。
<br/>　　哼...
<br/>　　涂小安意念一动，一股主人的威严激荡出去，旋即，青枭就害怕起来，那种害怕好像是来自于灵魂迸发。
<br/>　　它乖了，安分了。
<br/>　　它的长长如拖地长裙的尾巴华丽的开屏了，顿时展开的羽毛鲜艳夺目，五光十色，使人眼花缭乱，散开的羽毛上闪闪发光，像是透亮的珍珠撒在它身上，非常美丽。
<br/>　　“哇...好美呀，简直太美了，孔雀开屏，只听别人说过。”
<br/>　　小月看的跟一个小迷妹似的，瞬间被这只孔雀给俘虏了。
<br/>　　青枭这次开屏就是很普通的开屏，没有使用自己那股特殊的力量，否则小月会瞬间陷进去，难以自拔，任人宰割。
<br/>　　很敷衍的开了次屏，青枭合上了，继续梳理自己美丽的羽毛。
<br/>　　就在这时，内屋中走出来一位妇人，边走边说道：“月儿，大早上你瞎叫什么”
<br/>　　温秋面色红润的走了出来，自从她的肺癌治疗好后，她的身子就一日比一日好，越活越年轻，每一天喝着杀人蜂的蜂蜜水，都觉得自己的状态要赶上年轻的时候。
<br/>　　“妈，你出来的正好，孔雀，快看孔雀”
<br/>　　一只傲娇的孔雀梳理着美丽的羽毛，此画面落入温秋的眼中，她也震惊了。
<br/>　　“这...这...”
<br/>　　“妈，你肯定以前也没见过孔雀吧”小月兴奋的指着青枭道：“孔雀孔雀，你在开一次屏，给我妈也看看”
<br/>　　青枭一脸轻蔑跟傲娇，直接无视，当它是什么，真拿它当动物园里的那些杂牌货？
<br/>　　扔点零食吃，我就给你开一次屏。
<br/>　　它可是高贵不凡的百鸟之王。
<br/>　　“那是我老娘，你要是不把她哄开心了，我不但拔光你的毛，我还给你煮了你信不”
<br/>　　一道冰冷的声音传入青枭的脑海中。
<br/>　　青枭顿时浑身冷颤，看了看这位中年妇女，看了看金银蛇，任它灵智在高，也弄不明白一个妇人怎么会是一条蛇的母亲。

<br/>　　温秋看着看着想去摸一下孔雀的羽毛，要搁平时，青枭怎会让人碰它华丽的羽毛，但此时它一个眼明手快，直接凑了上去。
