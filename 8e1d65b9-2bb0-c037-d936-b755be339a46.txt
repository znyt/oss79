        但是在下一秒……
<br/>
<br/>        童筱颖却拉住了傅夜沉的手。
<br/>
<br/>        “我爱过小卫，但现在爱的人是大哥。”
<br/>
<br/>        这是她的实话。
<br/>
<br/>        只是……她觉得，她对大哥的爱，不及落以柔对大哥的爱那么深。
<br/>
<br/>        她觉得自己爱得很迷茫，爱得不深不浅。
<br/>
<br/>        傅夜沉欣慰地扬起了嘴唇，冷淡的俊脸，终于有了一丝笑意盎然的生气。
<br/>
<br/>        有那么一个念想，让他燃起了一股私心，希望她能原谅他，他不是那么伟大也不是那么完美的男人。
<br/>
<br/>        他去救季雅珊，就必然会走上一条不归之路。
<br/>
<br/>        而接下来在没有他的日子，他却不希望她再次成为别的男人的女人。
<br/>
<br/>        “你起初说，要给我生个孩子，对不对？”傅夜沉旋过身，一张令人捉摸不透神情的俊脸，瞬间凑到了童筱颖的跟前。
<br/>
<br/>        童筱颖忽然间一愣，完全不明白他现在的用意。
<br/>
<br/>        明明是她心事重重，但此刻，他才是深不可测的那一个。
<br/>
<br/>        对，她总是摸不透他的心思。
<br/>
<br/>        就在她分神之际，他的大手已经来到了她的腿上。
<br/>
<br/>        没有任何前兆，她娇软的身子就被他从床上捞到了床边站着。
<br/>
<br/>        他单手解开了他自己的腰带。
<br/>
<br/>        童筱颖只觉身上发凉，仰头看向站在自己面前，身材高大的傅夜沉时，他的瞳仁，深似潭水。
<br/>
<br/>        傅夜沉眸子里暗晦不明，大手则强劲有力地托住她的细腰，扶持着她站稳了身子。
<br/>
<br/>        “好好抚养我们的孩子长大成人，将来让他继承傅氏集团。”他的这句话像是别有用意，但是语重心长。
<br/>
<br/>        “记住，我傅夜沉让你痛的时候，是什么滋味。”这个时候，他却邪肆地笑了。
<br/>
<br/>        依旧是那么坏……
<br/>
<br/>        童筱颖没有闭眼睛，漂亮的眸子，全程直勾勾地凝视着傅夜沉。
<br/>
<br/>        以前，她总会娇羞，生怯。
<br/>
<br/>        现在，她的心情却很复杂。
<br/>
<br/>        落以柔坐立不安地寻上来时，路过房门外，刚好看到这一幕。
<br/>
<br/>        她心痛地背靠着门边的墙壁，捂住了自己的嘴，不让自己发出声来。
<br/>
<br/>        看到自己最爱的男人，在跟另一个女人做最亲密的事情，心脏就好像是被五马分尸。
<br/>
<br/>        可是，她心里又觉得有些不可思议。
<br/>
<br/>        这大概是她第一次见到他傅夜沉，也有如此邪魅的一面。
<br/>
<br/>        他一身黑色风衣，衣着完整地贴附着他健硕颀长的身躯。
<br/>
<br/>        明明周遭散发着浓郁的荷尔蒙气息，但他的侧面却冷静得骇人。
<br/>
<br/>        落以柔的脑海中一片混乱，转身逃也似地跑开，只觉心痛得几乎窒息。
<br/>
<br/>        一抹黑影从余光里掠过，童筱颖分了心，偏头望向了房门口，才发现刚刚大哥抱她进来的时候没关门。
<br/>
<br/>        “大哥，门没……唔——”童筱颖的话还未说完，脸颊被一只大手捧住，唇上一热，话被堵住。
<br/>
<br/>        童筱颖惊愕地瞪大了眼睛，愣愣地看着傅夜沉，她刚刚的分神，换来的是他更深层次的惩罚。
<br/>
<br/>        童筱颖不知不觉地闭上了眼睛。
<br/>
<br/>        楼下，落以柔跑回了客厅，在沙发上心慌意乱地坐下。
<br/>
<br/>        傅夜沉那高大修长的身形，英俊邪魅的侧脸，那只充满阳刚的大手……
<br/>
<br/>        落以柔身心忽然痉孪起来，意识到自己竟然幻想着傅夜沉怀中的女人是自己时，沮丧地抬起双手，抱住了自己的脑袋。
<br/>
<br/>        天啦！
<br/>
<br/>        她真的是太丢脸了！
<br/>
<br/>        竟然对傅夜沉产生那种污秽的思想，难道她真的是爱到无可救药了吗？！
<br/>
<br/>        “收拾一下你的私人物品，我送你离开私宅。”
<br/>
<br/>        不知过了多久，一个低沉透着磁性的嗓音在落以柔背后响起。
