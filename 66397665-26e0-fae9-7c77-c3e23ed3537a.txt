　　苏重速度飞快，一闪一闪的前进。
<br/>　　比杨德的速度同样不慢，他双腿结构发生改变。变得好似水桶般粗细，大小腿肌肉鼓起俩个疙瘩。好似两个橄榄球，被细细的膝盖链接在一起。
<br/>　　每一次发力，地面上就出现一个大坑。一下子就能窜出去数百米。
<br/>　　苏重在后面紧追不舍。这一会儿他也看清楚了敌人的长相。知道这是尼特罗的儿子比杨德。
<br/>　　“怪不得看着那个魔像熟悉，这不就是模仿自百式观音的招式吗？”苏重眼中闪过一道火热。
<br/>　　尼特罗可是称霸一个时代的强者，即便是现在，那衰老的身体内，依旧蕴藏着纵横世界的力量。作为他的招牌，百式观音的强大毋庸置疑。
<br/>　　他的魔龙七段进展飞速，正是力量爆棚的时候。碰到堪比百式观音的招式，苏重心底顿时涌起火热。
<br/>　　砰！
<br/>　　一阵巨响，漆黑地面陡然炸开一个大坑。苏重速度再快一分，即便是气魄都无法消除声音，沿路发出呜呜响声。
<br/>　　比杨德脸色一变，即便不回头看，他都知道苏重速度猛增。心中估算，不等他跑出去，就会被苏重追上。
<br/>　　当机立断，比杨德不仅没有加速，反而猛然跃起，双腿好似钉子。砰的一声钉在了一根世界树根须上。
<br/>　　他就这么头下脚上，双眼死死盯住飞射而来的苏重。眼中战意飞腾，杀气铺天盖地的冲向苏重。
<br/>　　漆黑色烟气轰然炸开，魔像巨人眨眼间耸立。
<br/>　　魔像·千斩！
<br/>　　大片大片的紫黑刀光飞出，瞬间布满身前空间。照着迎面而来的苏重，当头劈下。
<br/>　　“没用的，相同的招式一点儿用都没有，还有什么招式，都用出来吧！”苏重怡然不惧。
<br/>　　暗杀拳·崩山！
<br/>　　轰！
<br/>　　加入了气魄的崩山更加强劲，控制力暴增。所有的震劲全被苏重收敛进一米范围之内。
<br/>　　一个黑红色光柱骤然飞出，所过之处，所有紫黑刀光全被碎裂，化作点点荧光。
<br/>　　苏重不闪不避，一头扎进刀网之内，顺着暗杀拳开出来的通道，直接飞了进去。
<br/>　　定睛一看，可哪里还有比杨德影子。
<br/>　　这家伙竟然跑了！刚才还一脸杀气，战意爆发。谁知道竟然放了一个大招，转身就跑！
<br/>　　苏重顿时气节，他还想近距离接触魔像，想要偷学这一招呢！
<br/>　　这不是妄想。苏重这段时间一直在琢磨魔龙七段，除了傀儡之线，没怎么开发其他技能。他的技能面板可还有两个空槽呢。
<br/>　　当初就是和傀儡之线的主人交手，让他偷学成功。苏重相信，只要和对方近身交手。抓住机会吸收对方一缕念力，绝对能把这招偷学过来！
<br/>　　这可是类似百式观音的强大招式啊！
<br/>　　“不要跑！不要跑，你给我站住！”苏重双眼满是狂热，就像是色中饿鬼见了绝世大美女。道种震颤之下，一身浑厚念力，骤然从双肩、后腰、脚下等地方喷发。
<br/>　　本就速度飞快的苏重，竟然把速度再次提高。这真成了一动黑色闪电。半空中竟只能看到一道模糊黑影！
<br/>　　比杨德丝毫不敢停留，冷峻的脸上满是肃然。来之前，他以为此行不过是手到擒来。即便见到苏重沐浴岩浆，他依旧充满自信。魔神之像给了他强大的信心。
<br/>　　可苏重的强大超出他的预料。全力斩击，竟然连苏重的光膜防御都破不开。
<br/>　　而且，两人对攻一击。自己竟然完全落入了下风！这还怎么打？
<br/>　　他不是纯粹的武者，没有那种在生死夹缝间，寻找胜利机会的勇气。看到情形不对，他哪里还会停留。
<br/>　　顺着来时道路，一路飞奔。半路上，不时出手攻击周围墙壁，把本来就不宽的通道砸的破破烂烂。不断有大石头坍塌下来堵住通道。
<br/>　　地下的石头，在压力和高温作用下，非常的坚硬。堵在通道里面，寻常人根本就打不开。
<br/>　　不过苏重不一样，整条通道都是他开的，比杨德明白。这仅仅只能拖延一些时间罢了。
<br/>　　……
<br/>　　地面之上，裂缝不远处有一个小山坡，山坡后面是一个狭小山谷。小到可以把它称为一片洼地。
<br/>　　这小山谷深处，数个水潭热气蒸腾。一股淡淡的硫磺味道弥漫在半空。这是一处天然温泉。
<br/>　　水潭不是死水，有一条小溪流流出山谷。而温泉底部也不是泥土，是一些鹅卵石。清澈可见底，是一个泡温泉的绝佳场所。
<br/>　　帕里斯通身上裹着一条大毛巾，坐在温泉中，舒服惬意的靠着一大石头。
<br/>　　“真是一个不错的地方。可以考虑把这里圈起来，专门用来泡澡。”帕里斯通乐呵呵的想着。
<br/>　　他一点儿都不担心比杨德。和比杨德合作那么久，即便精明如他，依旧没有探出这个老家伙的底。
<br/>　　可即便是他知道的，也足够让人震惊。反正自从两人相识以来，只要他出手，就没有失败的时候。
<br/>　　别看对方七十多岁的老头子，一身实力却正在最强的巅峰状态！比杨德可是尼特罗的儿子。而尼特罗的心源流，尤其擅长保持长久战力。
<br/>　　即便是不知道活了多久的涅特罗本人，依旧有着横行当世的强大力量！
<br/>　　轰！
<br/>　　一声巨响，好似一道闷雷骤然炸响。吓得帕里斯通一个哆嗦，赶紧睁眼看去。
<br/>　　小山坡后方，一个十多层楼高的巨人突兀出现。即便有山坡遮挡，他依旧可以看清那带着盔甲的紫黑头颅。
<br/>　　黑烟环绕巨人身周，好似火焰一般不断舔噬着四周空间。
<br/>　　他正要仔细看去，一股邪恶意念骤然传来。饶是他这个十二地支的猎人，实力强大，却依旧被震的浑身发冷。意识出现瞬间的空白。
<br/>　　“这是什么鬼东西！怎么那么邪恶！”他终于看清了那遍布整个巨人头颅，不断哀嚎挣扎的头颅。不断蠕动纠缠，让帕里斯通脸色刷的一下雪白。
<br/>　　这就是比杨德的底牌？好邪恶！好强大！
<br/>　　轰！
<br/>　　一多蘑菇云骤然升腾而起，巨大魔像炸开消失，好似刚才的一瞬间是幻觉一般。
<br/>　　呼！
<br/>　　一阵劲风刮来，平静的水潭顿时被激荡起数米水花，一道飞快身影带着狂风骤然停止。
<br/>　　“快走！”比杨德瞬移一样出现在帕里斯通身边，也不管帕里斯通衣衫不整，抓住对方肩膀，使劲一踹地面，猛地窜进了密林之中。
<br/>　　轰！
<br/>　　几乎在比杨德消失的瞬间，山坡后方再次升起一股蘑菇云。苏重冲天而起。强大的圆悍然横扫而出，片刻间就捕捉到了比杨德的踪迹。
<br/>　　念从身后骤然间喷发，苏重快若闪电，奔着比杨德就追了过去。
<br/>　　比杨德脸色大变，回头一看。发现只是这一会儿的功夫，苏重竟然已经快要到他身后！
<br/>　　“看你往哪里跑！”苏重大吼一身，身上黑红光膜骤然绽放。
<br/>　　暗杀拳，崩山！
<br/>　　一道黑红色光柱，从天际直直攻击而下。朝着比杨德骤然杀来。
<br/>　　比杨德瞳孔猛然收缩。紫黑色烟气瞬间从他身边炸开，一面恶鬼脸圆盾将他牢牢挡在身后。
<br/>　　八尺镜！
<br/>　　嗤嗤嗤！砰！
<br/>　　圆盾紧紧抵挡了三个呼吸，就被一击二溃败。比杨德顺势倒退，扬起手臂把帕里斯通往前一扔。
<br/>　　“帮我抵挡片刻！”说完他就盘膝坐在地上，竟然在这关键时刻闭上了眼睛。
<br/>　　帕里斯通想死的心都有。一直穿着严谨正派，恨不得被自己打上特效的他，此刻竟只在腰间裹了一条毛巾？都这样了，还要和别人打架？这也太坦诚了吧？
<br/>　　可抬头看向对面，帕里斯通脸色顿时古怪起来。他隐约看到了一个比他还坦诚的家伙。
<br/>　　“那个……咱们能不能停一停，这么打，有些尴尬啊！”帕里斯通一脸的讪讪。
<br/>　　苏重也郁闷的要死。两个大男人坦诚相见，可别提多尴尬了！虽然都是男的，可真要面对面站在一起，心里腻歪的要死！
<br/>　　可惜，还是要死！
<br/>　　毫不犹豫，苏重挥手就打出一道黑红光柱！
