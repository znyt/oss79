        郑忠威有间接地向傅炎廷推荐落以柔，傅炎廷也确实很看好落以柔。
<br/>
<br/>        但是，傅炎廷却误会了郑忠威的意思，以为郑忠威向他推荐落以柔，是因为落以柔很适合当傅夜沉的贴身秘书。
<br/>
<br/>        所以，才导致傅炎廷对落以柔的定位是自己长孙的秘书，并命令郑忠威好好地栽培。
<br/>
<br/>        其实，最主要的是，傅炎廷心里已经有了自己更好的长孙媳候选人。
<br/>
<br/>        新上任的李市长的千金——李忆莎。
<br/>
<br/>        这个女孩，傅炎廷并不陌生，可以说他是看着这女孩长大的。
<br/>
<br/>        李忆莎与姚明娜、傅昼景和傅夜沉，四个人算得上是青梅竹马，从小就认识。
<br/>
<br/>        姚明娜喜欢傅昼景，李忆莎喜欢傅夜沉。
<br/>
<br/>        这些孙儿辈的小恋情，傅炎廷也知道得一清二楚。
<br/>
<br/>        只可惜，他的两个孙子，都是“妾有意而郎无情”。
<br/>
<br/>        姚明娜是地利置业财务部的副部长，自从在他傅炎廷那儿得知，傅昼景结婚了，和他的妻子一起移民去了澳洲定居，她便对傅昼景死心了，并在傅昼景去澳洲的那年辞职嫁了人，现在已经是两个孩子的妈妈了。
<br/>
<br/>        那么，李忆莎呢？
<br/>
<br/>        她十二岁那年，情窦初开，自信满满地拿着自己做了一晚上的爱心巧克力和认认真真写了不下十遍的粉色情书，在二月十四号的情人节那天，鼓足了勇气当众向大了自己一岁的傅夜沉表白。
<br/>
<br/>        可结果，却被傅夜沉当成空气一样给无视掉了。
<br/>
<br/>        十四年过去了，那日表白的情形，对李忆莎来说，仍旧记忆犹新。
<br/>
<br/>        而对于傅夜沉而言，他几乎一点印象也没有，甚至都不记得还有她李忆莎这么一个女孩曾经出现在他的生活圈过。
<br/>
<br/>        “爷爷，我知道。”傅夜沉淡淡地应了声。
<br/>
<br/>        “新上任的李市长的千金李忆莎，你应该还记得她吧？”傅炎廷叹了口气，接着问道。
<br/>
<br/>        傅夜沉凝眉想了一会儿，硬生生地没想起这个女人是谁。
<br/>
<br/>        “李市长跟我说，她女儿暗恋了你十四年，为了等你，一直不肯交男朋友，所以才主动上门来跟我攀亲的。”傅炎廷感慨道。
<br/>
<br/>        傅夜沉仍旧没想起这个女人到底是谁。
<br/>
<br/>        “爷爷跟你说话了，你到时吱个声！”傅炎廷听闻傅夜沉那边没动静，又忍不住抱怨了起来。
<br/>
<br/>        傅夜沉这才开口回答道：“我不记得她是谁了。”
<br/>
<br/>        “李忆莎你不记得，那姚明娜，你总记得吧！她结婚的时候，是爷爷让你代我去参加的婚礼。”傅炎廷接着说。
<br/>
<br/>        傅夜沉漫不经心地应了声：“记得。”
<br/>
<br/>        “李忆莎是姚明娜的闺蜜，读初中的时候，你们不是总在一块上下学的吗？”傅炎廷真想打开自己这长孙的脑袋看看，里面到底装了些什么。
<br/>
<br/>        傅夜沉对于李忆莎这个女人，仍旧没想起什么来。
<br/>
<br/>        若是真想起来了什么，好像她曾向他的弟弟傅昼景表白过吧！
<br/>
<br/>        她喜欢的人应该是他弟弟傅昼景才对，怎么到了爷爷口中，就成了暗恋他傅夜沉十四年呢？
<br/>
<br/>        难道，这当中有什么误会？
<br/>
<br/>        “小沉，爱一个人得不到回应，是什么痛苦，你应该比任何人都清楚。既然如此，换位思考，李忆莎这个女孩为了你守身如玉十四年，又是有多么的痛苦？你就不能看在她对你如此痴爱的情分上，跟她见上一面？撇开她不说，你也得从筱颖的阴影里快点走出来，是不是？你已经熬了三年了，难不成，你还想再熬一个三年吗？小景的孩子都能满地跑了，小沉，你就不要再跟自己过不去了。不要让爷爷为了你的婚事操心。爷爷年级大了，只希望能在有生之年，抱到我家小沉的孩子。”傅炎廷苦口婆心地劝说。
<br/>
<br/>        小景的孩子都能满地跑了……
<br/>
<br/>        是啊！
<br/>
<br/>        筱颖的孩子都能满地跑了……
<br/>
<br/>        他们都已经有孩子了，他还在这傻傻地痴等着什么呢？
<br/>
<br/>        “我明白了，爷爷。”傅夜沉的口吻恭敬又诚挚。
<br/>
<br/>        “我需要的不是你说‘我明白了’这句话。”
<br/>
<br/>        “我知道，您是为我好。”
<br/>
<br/>        “……”
<br/>
<br/>        “好，我去。”
<br/>
<br/>        “……”
<br/>
<br/>        “订好地方、安排好时间，我再向您报告……”
<br/>
<br/>        “……”
<br/>
<br/>        “是，爷爷，我会尽快。”
<br/>
<br/>        傅炎廷在电话里说了什么，落以柔没听到，倒是只听到以上傅夜沉唯命是从的回答。
<br/>
<br/>        “小落，帮我订一家最高级的法国餐厅。”挂断电话后，傅夜沉交代道。
<br/>
<br/>        “你和傅老太爷？”
<br/>
<br/>        “不。”傅夜沉摇头，“我和那位市长的千金。”
<br/>
<br/>        “市长的千金？”落以柔有点不太明白，跟他通话的人不是傅老太爷吗？
