两个人又睡了一个回笼觉，这一睡竟然就到了下午，这时陆寒霆觉得鼻尖痒痒的，他没有睁眼，而是抓住了那只小手，坚毅的下颚抵到她柔软的脸蛋上蹭了蹭，“醒了？再睡一会儿。”
<br/>
<br/>        他下颚上有一圈青渣，蹭的她好痛，夏夕绾在他怀里“咯咯”笑着躲避，“陆先生，不要睡了，都下午了，太阳都要落山了。”
<br/>
<br/>        陆寒霆这才睁开眼，现在真是下午了，他二十多年的人生里别说睡到下午，连个懒觉都没有睡过。
<br/>
<br/>        夏夕绾乖软的趴在他的怀里，“陆先生，你是不是…太累了？”陆寒霆收紧健臂，力道大的恨不得折断了她这堪堪一握的小蛮腰，“陆太太，我是不是没有将你喂饱，要不要我向你证明一下我的体力？”夏夕绾迅速举双手投降，“陆先生，我错了，你饶了我吧，我现在身体好像散架了一样，需要休息一下。”
<br/>
<br/>        他的体力她已经见识过了，虽然是她主动撩他的，但是他的体力好到近乎变态。
<br/>
<br/>        陆寒霆见她巴掌大的绝色小脸都拧在了一起，眼下还一片乌青，求饶的样子真是可怜兮兮的，他勾唇，“现在知道求饶了，你胆子不是挺大的吗？”
<br/>
<br/>        夏夕绾俏皮的眨了一下纤长的羽捷，骄傲的抬起了小巧的下颌，“陆先生，你要记住了，你的处可是我破了！”
<br/>
<br/>        陆寒霆英气的眉心蹙了一下，“陆太太，你为什么总抢我的台词，这话应该我来说。”
<br/>
<br/>        “我不管，陆先生，都说男人会永远记住自己的第一个女人，你要永远记住我，可不许忘了我哦。”
<br/>
<br/>        陆寒霆伸手捏了一下她的小脸蛋，“你这话怎么怪怪的，陆太太，你是不是有什么事情瞒着我？”
<br/>
<br/>        他像是察觉到了什么，一双狭眸半眯，打量了她一眼。
<br/>
<br/>        夏夕绾有点伤心，她垂下纤长的羽捷，“再等个几天我们就要分别了，人这一生这么的漫长，以后会有别的女孩儿出现在你的生命里，渐渐的取代我的位置，让你欣赏和喜欢，只要想到我调教的这么好的陆先生要拱手让给别人，我就很不开心。”
<br/>
<br/>        他在帝都城还有一个未婚妻，等他回去，不但可以继承财产还有如花美眷，这样惬意的人生他应该很快就会将她忘记吧？
<br/>
<br/>        陆寒霆敛下俊眸，原来她说的是分别这件事，这一场蜜月之旅只有五天，现在都过了两天了，她的温柔乡总能一下子将他抛上云端，忘记外间的一切。
<br/>
<br/>        “绾绾，除了你，不会用别人了。”
<br/>
<br/>        会有别人的！
<br/>
<br/>        夏夕绾没有将话说出来，他这个甜言蜜语让她还是很受用的，她一张绝丽的小脸阴转晴，“陆先生，我们起床吧，晚上再睡。”
<br/>
<br/>        ……
<br/>
<br/>        两个人起来洗漱，陆寒霆换了一身衣服出来时已经看到夏夕绾在收拾凌乱的床单了，床单上绽放了一朵血梅。
<br/>
<br/>        夏夕绾脸皮薄，这样的床单自然是不能让那些婶儿们抢着洗的，她要自己洗。
<br/>
<br/>        陆寒霆走过去，从后面抱住她，大手放在她平坦的小腹上，“绾绾，要不要…吃一下避孕药？”
