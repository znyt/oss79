　　突破修为之后，苏莫又静修了两日，巩固了一番修为，便离开了火元峰。
<br/>　　现在还有一些时间，他要再做些准备，增加一些手段和底牌。
<br/>　　很快，他便来到了外府，来到了木璃导师的院落之前。
<br/>　　木璃导师是外府的导师，平常都住在外府，来之前苏莫用灵识在火元峰上扫了一圈，对方不在火元峰，那肯定是在住处了。
<br/>　　咚！咚！咚！
<br/>　　落在精美的院落之前，苏莫轻轻的敲了敲门。
<br/>　　院落中寂静无声，约莫过去了几个呼吸之后，脚步声响起，而后院门打开了。
<br/>　　“木璃导师！”苏莫向对方抱了抱拳。
<br/>　　“何事？”木璃凝声问道，言语不冷不淡。
<br/>　　“木璃导师，我来是有事求助。”苏莫沉声说道。
<br/>　　木璃目光凝视着苏莫，没有开口，等待苏莫继续说。
<br/>　　苏莫略作沉吟，道：“木璃导师，你是炼器师，不知道能不能弄到炼器材料？”
<br/>　　“你需要什么炼器材料？”木璃问道。
<br/>　　“血纹籽，空意木。”
<br/>　　苏莫说道，他是准备再炼制一批本命灵剑，核心的材料之中，通神玉他还有很多，这两样材料则是稀缺之物。
<br/>　　“你要炼制本命灵剑？”木璃惊讶的问道，她只听这两种材料的名字，便知道苏莫要炼制何物！
<br/>　　她知道苏莫是本命剑客，以前在战界和冷江一战之时，动用了二十八柄本命灵剑。
<br/>　　但已经拥有了二十八柄本命灵剑，数量也不算少了，居然还想炼制？
<br/>　　“嗯！木璃导师能弄到这两种材料吗？”苏莫点了点头，并未隐瞒。
<br/>　　木璃略作沉吟，问道：“你要多少？”
<br/>　　苏莫闻言眼眸一亮，对方果然能弄到这两种材料，血纹籽在苍穹世界基本绝迹了，但在这外界肯定不会绝迹。
<br/>　　“有多少要多少？”苏莫急忙说道。
<br/>　　“你要这么多干嘛？“木璃闻言皱眉，本命灵剑并不是越多越好，也要能催动的起来才行啊！
<br/>　　一般的武帝境武者，最多也就能完美的催动四五十柄本命灵剑而已，就算是精神力超强的天才，也就只能催动百余柄而已。
<br/>　　至于传闻之中，那种能催动几千、上万柄灵剑的人，那是天赋异禀的特殊妖孽。
<br/>　　“木璃导师，我要自然有用，麻烦你了！”
<br/>　　苏莫手掌一番，出现了一枚储物戒，递向对方，道：“这是购买材料的灵晶。”
<br/>　　木璃没有拒绝，接过了储物戒问道：“就这两种？没其它的了？”
<br/>　　“若有其它炼制本命灵剑的材料，那就更好了！”苏莫说道。
<br/>　　“嗯！”木璃点了点头，随即关上院门，闪身离开了。
<br/>　　苏莫见此松了口气，本来他还担心对方不愿意帮他呢，毕竟这个女人不是热情的人，没想到对方还是挺乐于助人的。
<br/>　　随后，苏莫就跑到了学府中的交易坊市，购买了少量的炼器材料。
<br/>　　最后他才回到了火元峰的洞府中，思量着到哪里收集齐炼制本命灵剑的其它材料。
<br/>　　神武学府中的交易坊市，他也只买到了十几种炼制本命灵剑的材料，还不足以凑齐所有的材料，看来要离开学府，到附近的城池中购买。
<br/>　　时间上有些紧迫，怕是不一定能来得及了。
<br/>　　就在苏莫准备离开神武学府，到附近的城池去转转之时，木璃导师来了。
<br/>　　唰！
<br/>　　一袭白衣的木璃，身形飘落在了洞府门前。
<br/>　　“呐，这是炼制本命灵剑的所有材料！”木璃玉手一抛，一枚储物戒飞射而出。
<br/>　　“所有材料？”苏莫闻言一怔，伸手接过了储物戒，随即他意念向储物戒中一探，顿时彻底怔住了。
<br/>　　储物戒中全部是炼器材料，全部是炼制本命灵剑的材料，五百多种材料一样都不缺。
<br/>　　而且，每一样分量都很足，足以炼制几百柄本命灵剑。
<br/>　　“这……！”
<br/>　　苏莫大喜，他没想到木璃导师居然给他找来了所有的材料，而且还这么快，这还不足半个时辰的时间。
<br/>　　有了这些材料，他就能马上着手炼制了，不用再去找寻材料了。
<br/>　　“哈哈！木璃导师，你真是太可爱了！”苏莫大笑着说道，他现在突然觉得看对方非常的顺眼。
<br/>　　“你胡说什么？”木璃闻言顿时面色一冷，不过她白皙的脸庞上，却是不由自主的升起了一抹红晕。
<br/>　　“呃……我是说你真好，这次太感谢你了！”苏莫神色一僵，略带尴尬的说道。
<br/>　　“哼！”木璃没有再理会苏莫，身形一转，便飘飞而去。
<br/>　　苏莫见此轻笑一声，这个木璃导师，虽然平时一脸的冷漠，一副生人勿近的模样，但为人还是挺不错的。
<br/>　　少倾之后，苏莫进入玉镯空间中，开始炼制本命灵剑。
<br/>　　取出所有材料，他先行将所有材料整理一番，分成一分又一份，这个过程不能大意，每一份的量都极为讲究。
<br/>　　“这些材料好像不同！”苏莫看着地上的血纹籽、空意木等很多材料，发现这些材料与他在苍穹世界所用的那些材料，明显的不同。
<br/>　　拿起一枚血纹籽，苏莫仔细端详了起来，这血纹籽比在苍穹世界的血纹籽略大，脸色更加的深。
<br/>　　“是年份不同！”苏莫眼眸闪亮，这种血纹籽，是一种植被的种子。
<br/>　　而这些血纹籽的年份，比他在苍穹世界得到的血纹籽，年份要长远的多。
<br/>　　这就好像是那些灵草灵药，比如说百年人参，明显无法和千年人参相比。
<br/>　　苏莫又检查了一些其它不同的材料，发现基本上都是品质更高，质地更好。
<br/>　　“这些材料练出的本命灵剑，应该比以前的更强吧！”苏莫心中暗喜。
<br/>　　能练出更高品质的本命灵剑，他当然高兴，这意味着威力更强。
<br/>　　不过，他本来的那些本命灵剑，虽然品质没有多高，但经过了这么多年的温养，品质早已大大的提升。
<br/>　　他那些本命灵剑，现在任何一柄，品质和威力都不下于帝级的宝剑了。
<br/>　　这些灵剑是能跟他一起成长的兵器，足够使用很久了。
<br/>　　良久之后，苏莫取出炼器熔炉，开始了炼制。
<br/>　　由于很多材料有些不同，年份更久或者是品质更高，导致苏莫没有掌握好火候，连续失败了三次。
<br/>　　直到第四次之时，他才炼制成功第一柄本命灵剑。
<br/>　　灵剑出炉，寒光四溢，犀利的锋芒仿佛能洞穿一起。
<br/>　　“品质的确是高！”
<br/>　　打量着炼制成功的本命灵剑，苏莫暗暗点头，即便是刚炼制出来的灵剑，威力和品质都在他温养了多年的灵剑之上。
<br/>　　而后，苏莫开始了一刻不停的炼制，时间有限，他必须加快进度。
<br/>　　炼制本命灵剑，对于他来说已经炉火纯青，而且由于改修了尊级中品的火属性功法《三千炎焱诀》，他的火属性玄力，威力巨大，融炼材料的速度大大提升。
<br/>　　从而，让他炼制灵剑的速度，也是大大的提升。
<br/>　　一日又一日，一柄柄本命灵剑炼制而出，苏莫连续的炼制了四十余柄本命灵剑之后，停下来休息了一日。
<br/>　　实在是他坚持不住了，每一柄本命灵剑，都需要一滴他的精血，连续消耗了四十多滴精血，就算是武帝境武者，都疲惫不堪。
<br/>　　休息了一日之后，苏莫继续炼制本命灵剑，又过去了几日，他再次炼制出了三十余柄本命灵剑。
<br/>　　至此，加上他本来的本命灵剑，他的灵剑数量达到了整整一百柄之多。
<br/>　　“够了！”苏莫满脸疲惫的坐在地上，心中苦笑，真不知道远古传说之中拥有万柄灵剑的人，是怎么做到的？
<br/>　　应该不是本命灵剑，只是普通的灵剑吧！不然只是消耗的精血，就不是一般人能够承受的。
<br/>　　“还有两天，就要出发去真圣界了！”
<br/>　　少倾之后，苏莫叹了口气，盘膝而坐，打坐调息，恢复精力。
<br/>(本章完)
