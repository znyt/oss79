        &lt;h3&gt;第三卷第二百二十八章  宁雅的路&lt;/h3&gt;
<br/>
<br/>        宁雅随着陈子昂回到北京，也立刻投入到工作中，她的目标已经树立起来了，现在只要按照计划进行就好，陈子昂已经重新规划了耗材推向市场的节奏，原本的计划全部打乱重新制定。
<br/>
<br/>        在上班的第一天陈子昂就召开了全体中层会议，布局了今年的计划，整整将耗材上市提前了一个月的时间，工作流程全部要重新安排，在这短短的一个月里要实行全店联动，虽然之前工作做好了，但是突然提前还是给了大家一个措手不及的感觉。
<br/>
<br/>        宁雅也忙得一塌糊涂，她不但要联系各部门进行配合，她还要重新联系各职能部门进行协商沟通。
<br/>
<br/>        会议上有些中层还是想按照原来的计划进行，但是陈子昂的固执是出了名的，没有人能反驳得了她的意见。
<br/>
<br/>        “原本计划在四月初，你这提前一个月，让大家都措手不及。”钱喵喵嘀咕着。
<br/>
<br/>        “是啊，这样我们整个工作计划都得打乱，你真是个疯子！”苟江陵没好气的附和。
<br/>
<br/>        “既然大家都知道我是疯子，做事情没有任何规矩而言，那么大家就不要去想着我会按照规矩办事，我意已决，无需多言，你们只管执行就好，计划是计划，无非就是提前了而已，前期我们已经做了那么多的工作，我觉得没有任何问题！”陈子昂不顾钱喵喵和苟江凌的唠叨：
<br/>
<br/>        “小美、凌菲，你们俩个准备得如何了？”
<br/>
<br/>        “我们这里随时待命，都已经准备好了。”倪小美和李凌菲同时回答，经过一段时间的磨合，他们俩个人配合默契，工作进展得很顺利。
<br/>
<br/>        “宁雅你有什么问题吗？”陈子昂问宁雅，宁雅要负责公关。
<br/>
<br/>        陈子昂知道宁雅也在做自己的事情，作为她来说，宁雅一天不辞职，她一天就是公司的人，该干什么就得干什么。
<br/>
<br/>        宁雅也知道陈子昂的意思，怕她一心两用顾不过来。
<br/>
<br/>        “我这里没有任何问题，一切都听陈总安排！”宁雅坚定的回答，陈子昂想早点将耗材推出去肯定有她的原因，她不会辜负她的信任，现在的她们已经不是上下级的关系，而是战友的关系，她们是在一起奋斗一起努力的。
<br/>
<br/>        陈子昂微微一笑，总有那么几个人懂她的，她并没有通知江俞轩和寒伯安，因为江俞轩在西城，最近西城的事情比较多，寒伯安在美洲，落雪和寒冰玉在美洲遇袭虽然没有造成很大的伤害，但是事情诡异，肯定要参入解决。
<br/>
<br/>        她和郝景文打电话商量了一下，郝景文说全力支持她的决定。
<br/>
<br/>        “那么就这样决定了，大家各负其责，加快进展，凤盛夏和陈思宇一定要把好财务的关。”陈子昂接着叮咛凤盛夏和陈思宇。
<br/>
<br/>        “陈总放心，我们这里一切正常，随时配合大家的步伐。”凤盛夏和陈思宇也是很多年的老搭档了，经过那次危机之后，她们带领着财务部兢兢业业，一刻也不敢懈怠。
<br/>
<br/>        .......
<br/>
<br/>        “我将计划提前，最辛苦的就是你了，你得重新去跑外围。”陈子昂将宁雅叫到办公室。
<br/>
<br/>        “无所谓，原本已经都说好了，无非就是我再跑一趟，将时间改过来。”
<br/>
<br/>        “你的美容事情准备怎么办？”
<br/>
<br/>        “场地在装修，装修好了再晾一段时间，就可以营业，工作人员基本都能到位。”
<br/>
<br/>        “你这能量还是蛮大的，这才多长时间，说干就真的干起来了，不愧是宁雅。”
<br/>
<br/>        “跟着疯子就会变成疯子一样的人。”宁雅微微一笑，现出些许调皮的样子。
<br/>
<br/>        “不怕我这疯子把你带歪啊？”陈子昂也笑了，生活无论如何暗潮汹涌，该笑还是得笑。
<br/>
<br/>        “我愿意被你这个疯子带歪，这一生跟定你了。”
<br/>
<br/>        “哈哈哈，我怎么感觉是示爱的味道，好酸！”
<br/>
<br/>        “想什么呢？跟你示爱？我也酸了，我取向正常。”
<br/>
<br/>        “那就等耗材推向市场之后，你就辞职吧。”
<br/>
<br/>        “这么快就赶我走？”
<br/>
<br/>        “有句话怎么说呢，女儿大了总得嫁人嘛，总不能让你一直呆在我身边吧，况且你还有事情要干。”陈子昂的声音里竟然透露着一股子悲凉。
<br/>
<br/>        天下之事，那能没有聚散，就像江俞轩说的，天下没有不算的宴席，来来往往，我们不过都是凡夫俗子，你来了他走了，匆匆而来匆匆而去，能留下的有几个？
<br/>
<br/>        “咦，你还想给我当长辈的，美得你！”
<br/>
<br/>        “别闹，我说的是真的，另外，上次和你爸爸谈妥了，准备一起合作。我想安排你回去和他们商讨合作的事宜。”
<br/>
<br/>        “陈子昂，你疯了吧？”宁雅惊得站了起来。
<br/>
<br/>        “你出来做事情，一部分的原因是因为你的家庭，你让我陪你回家有一部分也是因为你的家庭，凌辰去你家用意很明显，但是你爸爸却让你妹妹跟着凌辰去了山西。宁雅，每个人的出发点不同，考虑事情的结果也不同，我觉得我让你回去是正确的，另外，你会马上面临着自己创业，将会更加困难！”
<br/>
<br/>        “你不怕我向着宁家？”
<br/>
<br/>        “原本我们合作就是向着宁家，世界这么大，只要利益得当，和谁合作我都可以，为什么我偏偏和宁家合作呢？”
<br/>
<br/>        “子昂，我.......”
<br/>
<br/>        “别一副感动涕零的样子，我们都是女人，女人当自强，只有你自己坚强起来了，你的世界才能你自己主宰，不过这个道理适用于任何人，不论男女。命运是掌握在自己的手中，就看你怎么走。”
<br/>
<br/>        宁雅从陈子昂的办公室出来，还是一脸懵圈，她不是不明白陈子昂的心意，但是这个代价是不是太大了？
<br/>
<br/>        陈子昂就不怕她会签一份不平等的条约回来？
<br/>
<br/>        陈子昂是在帮她也是在逼她，逼她跨出去一步，逼她做她自己。
<br/>
<br/>        宁雅长长的叹了口气，路，就在自己的脚下，怎么走真的要看自己的选择。
<br/>
<br/>        既然陈子昂放心她，那么她就代表公司去宁氏一趟，不过不知道父亲知道是她代表公司回去和他谈判又会作何感想？
<br/>
<br/>        想一想真是可笑，当初卸了她的职位，如今她回去谈判，真是好讽刺！
<br/>
<br/>        陈子昂这个女人还真是睚眦必报，斤斤计较呢，不过宁雅是又欢喜又忐忑，欢喜的是有人为她出头露面，忐忑的是父亲也不是一般的人，陈子昂能想到的，父亲自然也会想到，两家能不能合作？
<br/>
<br/>        她会和谁来谈论合作的意向？这一段时间忙的都顾不上想起凌辰了，被陈子昂提起来心里还有点涩涩的感觉。
