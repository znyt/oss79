
<br/>　　<br/>　　    “别闹，我要送你礼物，也得是郑重其事的，一拿出来就金光闪闪的那种，绝对不会把这么个破盒子给你，我丢不起这个脸”王惊蛰的反应很快，一看小草眼睛放光的落在了盒子上，就顺着坡往下接了一句。<br/>　　<br/>　　    茅小草撇了撇嘴，拿起一杯鸡尾酒嘬着吸管一连冷笑了几声：“鸡贼！”<br/>　　<br/>　　    王惊蛰干咳了一声，说道：“是沈君给我的，算是谢谢我的援手之情，我也没好意思拒绝，就给收下了”<br/>　　<br/>　　    菜刀文挺有兴趣的给拿了起来，说道：“我看看是什么东西，我跟你说，这袍哥大佬看起来也不像是抠搜的样子，我估计看包装不咋样的话，内里可能别有乾坤呢”<br/>　　<br/>　　    这盒子看起来挺普通的，四四方方的，木头材质，上面刻着一些花纹，看起来很旧很有年代感。<br/>　　<br/>　　    林汶骐皱眉看着菜刀文把盒子拿过去，说道：“你可能走眼了，里面是啥我不知道，但东西绝对价值不菲，搞不好你一看都能把你的眼珠子给惊掉了”<br/>　　<br/>　　    菜刀文一愣，问道：“怎么说的呢？”<br/>　　<br/>　　    林汶骐指着他手里的盒子说道：“那是南海黄花梨，看上面的花纹应该还是名家出手的，但我一时也没认出来是谁的活，要说年代的话估计还得是明末清初那个时期的，以前我在墓里倒腾过两件，我们是卖死货的，当时还出手了几十万的价，如果是正当途径卖出去的，估计能上百万了，黄花梨价值是一方面，雕刻手艺也值钱，这东西保守估计你卖两三百个也不难，如果碰到喜欢这个的藏家可能还会高”<br/>　　<br/>　　    林汶骐这么一说，王惊蛰当即就皱眉了，倒头专家给品鉴出来的绝对不会走眼，他就觉得沈君送的这东西，确实可能有些份量了。<br/>　　<br/>　　    “唉，这交往的艺术，真是处处有门道啊”王惊蛰叹了口气，他是看出来了，沈君当面给自己的话对方怕他不要，所以就从司机那转手送过来了。<br/>　　<br/>　　    “让你们这么一说，我都不敢卡了，太重，容易砸了脚面子”菜刀文伸手打开了一道缝隙，往里面瞅了一眼，狐疑的叫了一声，然后才把盖子给拿下来，从里面用两根手指拿出一块被黄布包裹着的东西。<br/>　　<br/>　　    布子掀开，上面躺着的东西看起来也不太起眼，那就是块乌黑发亮，拿在手里显得有点沉的木牌，四边见方有棱有角，瞅着特别有厚重感。<br/>　　<br/>　　    菜刀文拿起木牌正反看了几眼，木牌一面光滑什么也没有，另外那面刻着几行小篆，除此以外也没什么稀奇的地方了。<br/>　　<br/>　　    这玩意真没看出哪值钱了，你就是扔在农村的路上，都没人愿意捡回去烧火，看起来跟小孩玩的那种积木没多大的区别。<br/>　　<br/>　　    “我是没看懂，你们品品？”菜刀文把木牌扔在桌子上，林汶骐顺手就捡了起来，翻看了几眼后也摇头说道：“这木头看着可能是阴沉木，但这上面的小篆我没瞧出是什么意思”<br/>　　<br/>　　    王惊蛰也接到手里观望了片刻，同样没瞧出什么一二三来，三个男人一人过了一手后，完全都没啥见解。<br/>　　<br/>　　    这时候，小草忽然皱着眉把鸡尾酒放在桌子上，然后擦了擦手上的水渍，说道：“给我看看……”<br/>　　<br/>　　    “您老给长眼了呗？”王惊蛰笑着就递了过去，然后就跟菜刀文和林汶骐闲聊着。<br/>　　<br/>　　    茅小草接过木牌，一连翻看了好几遍，脸上接着闪过好几道表情，诧异，狐疑还有点惊讶，足足过了六七分钟，这块木牌在她手里来回翻了能有好几遍，她才吐了口气。<br/>　　<br/>　　    “沈君给你的？”<br/>　　<br/>　　    “啊……”<br/>　　<br/>　　    “你接了，那这个人情可就欠大了”<br/>　　<br/>　　    “看你说的，好像他都要把她妹子给卖了似的，能有多大的人情啊？”<br/>　　<br/>　　    “从某种角度和意义上来讲，这东西可要比那个女明星值钱多了，并且有钱你也未必能买得到”茅小草把东西放在桌子上，很郑重的说道：“这确实是阴沉木，上面刻着的小篆应该是出自鬼谷子之手，这些小篆的意思……鬼谷子是干什么的，你应该知道吧？”<br/>　　<br/>　　    王惊蛰说道：“大秦帝师，据说秦始皇一统，除了他手下的那些大将以外，他背后站着的鬼谷子相当于是顶下了半壁江山，据说在争抢另外几国的国运之时，都是鬼谷子亲自出的手，论风水一术的话，古往今来他说第二，谁也不敢认第一”<br/>　　<br/>　　    华夏上下五千年，历经二十四朝，每一次的王朝更替，其实都涉及到一国之运的抢夺，每个朝代的都城都是建造在风水龙脉上，所以除了沙场上见胜负，最关键的是你在覆灭一个朝代的时候，也得把这一王朝的气运给截断了。<br/>　　<br/>　　    不然王朝下的王孙贵族皇子公主们，只要流落在外，那就总有在建朝的可能。<br/>　　<br/>　　    而鬼谷子，就是秦始皇背后，那个专门为他斩断另外几国气运的帝师，当时世间人都说，有鬼谷子一人，几乎可当千军万马来用了。<br/>　　<br/>　　    “这个小篆刻的是鬼谷子的风水要诀……”<br/>　　<br/>　　    王惊蛰足足呆愣了半晌，才有些不可置信的问道：“你是怎么知道的？”<br/>　　<br/>　　    “天道峰上有过记载，说是鬼谷子有一风水法器，就是阴沉木打出来的，那块木牌如果在布置风水法阵的时候，被充当阵眼所用，可以稳住运转的风水阵，想要破了这阵几乎就不太可能了”<br/>　　<br/>　　    王惊蛰豁然一惊，看着那块木牌忍不住“咕嘟”一声咽了口唾沫，这玩意儿对一般人来讲那真就是块木头，但对他这种通风水走阴阳的人来说，无疑就是一块大肥肉，眼巴巴的馋得很。<br/>　　<br/>　　    “沈君对你这是有所图啊，扔过来这么一份大礼，光是谢谢的话，也不太可能吧”菜刀文斜了着眼睛说道。
