　　　　涂小安并不知道这只猫精海棠经历了什么，但它身上强暴的蛇类气息，应该跟它忽然的消失，有着密切的关系。
<br/>　　这阴差阳错的，他还将陈高阳的“女朋友”给找到的。
<br/>　　只是这猫精现在的状态充满了杀戮，有点像是失去了理智。
<br/>　　不待涂小安在多想，一道粗大的蛇尾排山倒海的席卷过来，实验室的空气顿时阵阵爆响，充满了阴冷肃杀之意。
<br/>　　涂小安只能先原地闪身离去，轰然一声，这面墙壁如豆腐渣一样细碎的簌簌而落。
<br/>　　直接将对面的房间给打通了，对面的房间是一间电脑室，莫约有五六十台的电脑林立其中。
<br/>　　“海棠，先别动手，你可知道陈高阳”
<br/>　　既然算是间接的熟人，涂小安想先屈人之兵，他总觉得这只猫精的身上有什么诡异的故事。
<br/>　　它身上的蛇类气息不可能是与生俱来，否则系统的洞察就会扫描出来。
<br/>　　蛇尾落空，猛兽般的猫精准备再次攻击，可一闻陈高阳三个字，戛然而止，那双血红色的红目不在散发妖艳的光，好像有点黯淡下来。
<br/>　　还没等涂小安露出心喜之色。
<br/>　　仅仅停滞片刻。
<br/>　　“嗷...”它发出一声震耳欲聋的咆哮声，双目妖艳的光芒大亮，如黑暗中的吸血鬼，唯有见血。
<br/>　　这道咆哮，看似威风凛凛，让人心底发颤，可隐晦间，涂小安却将这道咆哮定义成悲恸嚎叫，那双妖艳的红目惊心动魄，可仅仅是表象，其中蕴含着害怕，不安，徬徨，身不由己。
<br/>　　心智混乱，难以自控。
<br/>　　又是一道蛇尾携惊天之力横扫而来，涂小安的人形态不能在受损坏，所以根本不敢硬接，抽身继续闪身避让。
<br/>　　“就让我帮你冷静一下吧”
<br/>　　涂小安跟猫精保持一定的安全距离，妖治的双眸如光如电，体内雷电属性的异象迸发出来，周身缭绕电光，空气中出噼哩叭啦的滋滋电流声。
<br/>　　“嗷...”
<br/>　　猫精再次一声咆哮，宛如饿狼扑食，几乎是一个眨眼，就如若幻影的出现在了涂小安的头顶，凶煞可怖。
<br/>　　“雷霆之怒”
<br/>　　幽幽然的声音带着一股天生的冰冷，涂小安含词轻吐。
<br/>　　顷刻间，实验室内充斥着一股刺耳的白光，只见还在空中的猫精直接被笼罩在一片雷海之中。
<br/>　　紧接着，一道悲恸嚎叫声迸发，让人毛骨悚然。
<br/>　　半晌后，轰...
<br/>　　猛兽般的猫精直接砸落在涂小安的面前，浑身炸立的毛发软绵绵塌陷，它的体型已肉眼可见的速度迅速缩小。
<br/>　　涂小安冷淡无波的看着，只见猫精满是妖艳的红目开始慢慢的淡化下来，但是它的气息却没有变弱。
<br/>　　二十九级，它的实力应该根本没有施展开来，或许它是有意为之？
<br/>　　“海棠，现在你清醒了没有”涂小安淡淡的开口。
<br/>　　恢复原形的猫精眼神清澈起来，没有妖艳的光，它的眼眸宛如宝石一般的璀璨，好看极了。
<br/>　　“你是谁，你怎么认识我”
<br/>　　猫精张了张嘴巴，发出了略带沙哑的声音，但却是女音。
<br/>　　“看来是清醒了不少，可以交流了”涂小安展颜一笑：“我认识一位叫陈高阳的人，他说他的女朋友叫海棠”
<br/>　　“高阳哥哥...”闻言，这只猫精好似晴天霹雳当头一击，反应极大。
<br/>　　果真如此，涂小安笃定了。
<br/>　　涂小安也不拐弯抹角，直接问：“告诉我，你身上发生了什么事情”
<br/>　　“我...”
<br/>　　猫精刚要开口，忽然，实验室的门口两道身影出现，伴随着一道戏谑的调侃声：“好一只口吐人言的猫精，果真不凡”
<br/>　　声音一出，涂小安蹙眉的放眼望去，一男一女，男的不知道哪根葱，女的涂小安还到认识，正是那个被他下午教训过的女警员。
<br/>　　而猫精立刻浑身毛发炸立，虎视眈眈的警惕起来。
<br/>　　“多谢你，帮我们找到了猫精，辛苦你了”
<br/>　　白楚艳一副螳螂捕蝉黄雀在后的胜利者姿态，得意洋洋的看了涂小安一眼。
<br/>　　“你的皮又痒了？莫非那么快就忘记了下午的事情”
<br/>　　涂小安微微抬着下巴，举手投足间流露出的冷静跟孤傲，根本没将她放在眼中。
<br/>　　“你...”白楚艳闻言，冷若冰霜的瞪了涂小安一眼，道：“现在我不是来找你的麻烦，我们的帐日后慢慢算，将这只猫精交给我，我立刻就离开”
<br/>　　“这只猫精与你何干，你为什么要它”涂小安眉峰间仿佛郁结着化不开的冰霜。
<br/>　　她跟猫精跟有什么联系？
<br/>　　白楚艳冷冷的说：“这只猫精不知死活，偷了我们白家的东西，我们自然要带它回去”
<br/>　　“它偷了什么东西”涂小安好奇的问，顺便看了警惕的猫精一眼。
<br/>　　“这与你无关，你只管将猫精交给我，或许你退开，猫精我们自己动手”
<br/>　　涂小安还真的不是看不起白楚艳：“就凭你，你还想拿下这只二阶猫精？”
<br/>　　此话一出，涂小安目光看向白楚艳身后的那位不苟言笑的男子，她的底气在于他？
<br/>　　“敢对我们二小姐不敬？你找死”
<br/>　　任海龙大步踏前，吃喝一声，准备一言不合就动手。
<br/>　　“海龙，等等”
<br/>　　白楚艳见过这位看似有畜无害少年的厉害，心中还揣着一点小心思，道：：“这只猫精确实是偷了我们白家的东西，你要是不多管闲事，你伤我之事，我可以不跟你计较”
<br/>　　“它到底偷了什么”
<br/>　　“这无可奉告”
<br/>　　涂小安冷笑：“这猫精是我先找到的，既然你不说清楚，那么你们可以滚了”
<br/>　　况且涂小安并不认为猫精是白楚艳跟那个男子可以对付的，要知道这只猫精可是二十九级呀，绝对不是软柿子。

<br/>　　居然变身后，长出一条蛇尾出来在。
<br/>　　这一切都非常的蹊跷，涂小安还必须要弄清楚不可了。
<br/>　　猫精海棠见这两人想抓它，清澈宝石般的眼眸既然慢慢的又有妖艳的光要闪烁出来，蛇类气息开始沸腾起来。
<br/>　　涂小安见此，心中一想，莫非是那股特殊的蛇类气息，让这只猫精变得失去理智？
<br/>　　它这明显是要发飙的迹象。
