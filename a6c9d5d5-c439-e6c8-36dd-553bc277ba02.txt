　　可荒兽一族的数量却是众多，大量的荒兽，就连先天神兽都不愿意对抗。
<br/>　　在古老的蛮荒时代之中，荒兽是最为棘手的那个敌人。
<br/>　　眼前这古籍之中记载的人影，居然是以一人之力，去挑战荒兽群？
<br/>　　楚风眠脑海之中的景象虽然是数十万年，甚至是数百万年前的图像。
<br/>　　可是看到那一尊尊荒兽的时候，楚风眠都有些不寒而栗，以他现在的实力，面对一些强大的荒兽，都未必是那些荒兽的对手。
<br/>　　而眼前这一道身影，面对的可是一整个荒兽族群。
<br/>　　成千上万的荒兽，随着那荒兽王的命令，疯狂的看着那一尊身影冲了过去。
<br/>　　可那一道身影，却是冷静无比，只是缓缓的抬起手中剑锋。
<br/>　　一剑。
<br/>　　剑锋所过之处，化为一片片的火海。
<br/>　　成千上万冲过去的荒兽，皆是在这火海之下，灰飞烟灭。
<br/>　　这些荒兽，甚至许多的实力都要超越了九劫古帝，甚至是都接近到了天人境的地步。
<br/>　　可是在这一道人影的面前，却是如同蝼蚁无异，一剑之下，尽数灭杀。
<br/>　　“此人，好强！在荒古时代，居然有这样的人物？”
<br/>　　楚风眠目瞪口呆的看着这一幕，心中都是无比的不可思议。
<br/>　　荒古时代，人类还未崛起，荒兽，妖神，先天神兽等，才是那个时代的霸主。
<br/>　　楚风眠还没有想过，在荒古时代居然有着如此人物，这种实力，只怕是已经站在那个时代的巅峰了，接近到了先天神兽的级别了。
<br/>　　“怪不得敢叫第一，后人之中，只怕都未必可能有人达到这一步了……”
<br/>　　楚风眠喃喃自语道。
<br/>　　他还想要看下去，但是在这一剑结束之后，脑海之中的画面，也就全部消失了。
<br/>　　看来接下来的战斗，这古籍之上，并没有记载。
<br/>　　这让楚风眠也有点可惜，他也想要看看这一战的结果，到底如何。
<br/>　　不过在楚风眠观看完了这些图像之后，古籍之上，也出现了无数的文字，脉络，记载的，就是刚刚那人影，施展出的剑术。
<br/>　　“三阳烈炎剑？”
<br/>　　楚风眠看完古籍，也了解了一下，这一招名为三阳烈炎剑，也正是这古籍的主人，就是刚刚看到的那一尊身影，所创造而出的一招剑术。
<br/>　　这一招的大成，一剑之下，足以点燃星空，将星空都化为一片火海。
<br/>　　“这一招剑术，正是晦涩难懂。”
<br/>　　楚风眠可以说是剑道天才了，他的剑道资质，要超越剑道之主，但是参悟这三阳烈炎剑，也给楚风眠带来了不小的麻烦。
<br/>　　幸好楚风眠因为得到了剑道图腾，参悟了其中残缺剑术的关系，他已经是触及到了剑道本源。
<br/>　　这三阳烈炎剑，虽然难懂，但是楚风眠慢慢参悟，也可以将其逐渐参悟出来。
<br/>　　参悟剑术并不难，但是想要融会贯通，并不容易，饶是如此，楚风眠花费了两日的时间，才将这三阳烈炎剑参悟出来。
<br/>　　他运转灵力，剑锋一闪，一抹火焰便是从剑锋上浮现而出，这火焰，除了三阳烈炎，还有着一团无色火焰，附着在其中。
<br/>　　楚风眠是冰火神体，足以掌控九阳天火，他便是将九阳天火，也融入到了这一招剑术之下。
<br/>　　楚风眠将火焰压下去，只留下了千分之一的力量，对着面前斩了过去。
<br/>　　轰隆！
<br/>　　炙热的火焰，席卷了整个洞府，一瞬间洞府之内，变的炙热无比。
<br/>　　“不错！”
<br/>　　楚风眠嘴角露出几分笑容，这千分之一的威力，足以达到这一步，如果威力全开，只怕是不亚于九玄剑术。
<br/>　　而且这还是楚风眠第一次施展这三阳烈炎剑，等楚风眠将三阳烈炎剑，融汇贯通的话，这一招的威力还要更强几分。
<br/>　　在楚风眠的面前，剑道古籍凝聚而出，最新的一页，便是被三阳二字所占据，这三阳烈炎剑，也被楚风眠融入到了九域剑术之中。
<br/>　　“三阳烈炎剑？你从哪里得到的这剑术？”
<br/>　　在楚风眠的面前，一道身影突然间的飞了出来，正是星玄。
<br/>　　他盯着楚风眠，眼神之中露出几分惊讶道。
<br/>　　“你的运气真不错，这种剑术都能让你碰到？”
<br/>　　“星玄，你知道这三阳烈炎剑？”
<br/>　　楚风眠听到星玄的语气，突然询问道。
<br/>　　听星玄的语气，他应该是知道这剑术，不然的话，不至于看见就喊得出名字来。
<br/>　　“你不知道？”
<br/>　　星玄听到楚风眠的询问，才有点奇怪。
<br/>　　“你没有得到第一剑帝的东西？那你这三阳烈炎剑是从何而来？”
<br/>　　“第一……剑帝？”
<br/>　　听到这个称呼，楚风眠脸色有些古怪。
<br/>　　他看到这古籍上的“第一”二字，还以为这剑术，被名为第一了。
<br/>　　结果是有人，叫第一剑帝？
<br/>　　“我得到了一本古籍，还有这玉符，还有一把剑柄。”
<br/>　　楚风眠倒是也没有什么隐瞒，将他得到这古籍，玉符，剑柄的事，告诉了星玄。
<br/>　　“看来那老者，应该是跟第一剑帝有些关系，甚至可能是第一剑帝的后人，所以才有这些东西的。”
<br/>　　星玄听完的话，喃喃自语道。
<br/>　　“星玄，这第一剑帝，是什么人物？”
<br/>　　楚风眠听到这陌生的名字，不由的向着星玄询问到道。
<br/>　　在九域的古籍之中，楚风眠可没有听说过第一剑帝。
<br/>　　不过九域的古籍，对荒古时代的事情，都是记录的甚少，甚至是没有留下荒古时代任何一位人类强者的名字。
<br/>　　所以这第一剑帝，楚风眠还真不清楚是什么人物。
<br/>　　“第一剑帝，是天下间第一位剑道仙帝，也是无数岁月一来，剑道的真正第一人，自号第一，却无人可以摘夺这头衔。”
<br/>　　星玄想到了什么，突然出声道。
<br/>　　“对了，你得到了三阳，那四方，六合，八荒，九转这四招，你有没有得到？”
<br/>　　“四方？六合？八荒？九转？”
<br/>　　楚风眠摇了摇头道。
<br/>　　听起来，这也应该都是剑术，跟三阳烈炎剑一样。
<br/>　　不过楚风眠得到的这一本古籍之中，只有记载了三阳烈炎剑，没有记载其他剑术。
<br/>　　“可惜了，本以为你有好运，可以得到上古十大至天仙术之一，只有三阳烈炎剑这一招，也就只能跟寻常仙术媲美了。”
<br/>　　星玄有些可惜的说道。
