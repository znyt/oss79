<br/>　　<br/>　　    “尼玛！”胡胡差点破口大骂。<br/>　　<br/>　　    王惊蛰贱嗖嗖的看着午桥说道：“你不能眼睁睁的看着我被这帮玄门的人给扣下吧？到时候你想要分一杯羹可能就没什么机会了，怎么样好汉，是不是该拔刀相助了？”<br/>　　<br/>　　    “我助了你，有啥好处啊？”午桥皱眉问道。<br/>　　<br/>　　    “我要是落到他们手里，可能被各种逼供，满清十大酷刑什么的都用到我的身上来了，到时候我要是挺不住呢，就有可能把该说的不该说的全都给吐出来了，就比如你感兴趣的那些东西，然后人家万一在杀人灭口什么的，是不是什么都没了·······”<br/>　　<br/>　　    午桥哦了一声，点头说道：“你说的好像挺有道理”<br/>　　<br/>　　    王惊蛰两手一摊，说道：“我现在能脱身了，你以后就还有机会，我现在要走不了，你以后可就什么机会都没有了”<br/>　　<br/>　　    “你说的好像确实挺有道理”<br/>　　<br/>　　    胡胡顿时气急败坏的说道：“你别听他胡扯，午桥你可以和我们联手，从他身上挖出来什么东西，我们平分就是了”<br/>　　<br/>　　    “不好意思，我这人喜欢吃独食，跟你们分不了”午桥硬邦邦的怼了一句后，跟王惊蛰说道：“我拦住他，你走，但你记住我一句话，这次欠我个人情，下次再碰上了，你得拿出等值的东西来交换才行”<br/>　　<br/>　　    王惊蛰干脆利索的点头说道：“必须靠谱！”
