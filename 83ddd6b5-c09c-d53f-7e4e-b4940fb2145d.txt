        &lt;h3&gt;第三卷第六十五章  春风吹又生&lt;/h3&gt;
<br/>
<br/>        陈子霖听了江俞轩的话，点点头。
<br/>
<br/>        他站起来头也不回的走了。
<br/>
<br/>        “子霖。”江俞轩跟在陈子霖的身后。
<br/>
<br/>        “我们回西城！”
<br/>
<br/>        “好！人都带走吗？”
<br/>
<br/>        “为什么非要将人带走？”
<br/>
<br/>        江俞轩看着气呼呼的陈子霖，“你都差一点掐死人家老婆了，你还留人在这里干嘛？”
<br/>
<br/>        “生意照做，我看他傅海生能把我怎么样？”
<br/>
<br/>        “子霖？”江俞轩刷新了对陈子霖的认识，刚刚和人大打出手，现在又要和人谈生意？
<br/>
<br/>        “江俞轩，这件事情先不要和我妹妹说。”陈子霖说
<br/>
<br/>        “为什么？”江俞轩不明白了。
<br/>
<br/>        “因为牵扯到了我二嫂！”
<br/>
<br/>        “你是想保护你二嫂？”
<br/>
<br/>        “不，我要先去问问我二嫂是怎么回事，如果真的是我二嫂干的......”真的是二嫂干的怎么办呢？
<br/>
<br/>        陈子霖颓废的往路边马路牙子上一坐，他不走了。
<br/>
<br/>        “二嫂啊，二嫂，你让我拿你怎么办啊？”
<br/>
<br/>        “子昂啊子昂，如果你知道这些事情你该怎么办呢？”陈子霖陷入了两难的境地。
<br/>
<br/>        “子霖，是这样的，你现在也不要去找你二嫂了，也不要和傅海生起冲突了，我也不告诉子昂了，子昂最近事情多，她还要去甸城。我们就不要给她添堵了。”江俞轩劝解着陈子霖。
<br/>
<br/>        陈子昂也真是命运多舛，被自己的亲二嫂给卖了，要不然也不会碰见张函吧？不会碰见张函也就不会当未婚妈妈，也就不会这么辛苦吧？
<br/>
<br/>        可是没有二嫂的出卖，也没有今天的陈子昂不是？陈子昂放在傅海生他们那一堆的同学里面应该都是佼佼者吧？
<br/>
<br/>        如果从体制上来说，可能陈子昂是悲催的，但是从个人发展的眼光来看，塞翁失马焉知非福呢？
<br/>
<br/>        “也好，刚刚子昂说让我一定要拿下洛城的这个项目，我得回去找傅海生去。”陈子霖又站起来了。
<br/>
<br/>        “子霖？”
<br/>
<br/>        “大丈夫能屈能伸，我们做我们正当的生意，不取不义之财，不做不义之事，我今天心急，动了手，是我不应该，至于夏颜他们做的那些个事情，自然有法，律来惩罚他们。我为了我的鲁莽去给他们道歉！”
<br/>
<br/>        江俞轩才知道什么叫能屈能伸了，陈家果然个个是汉子啊！
<br/>
<br/>        他又陪着陈子霖敲响了傅海生的门。
<br/>
<br/>        夏颜还没有从惊慌失措里走出来，看见陈子霖又不好了。
<br/>
<br/>        她从小到大都是被捧在手心的，哪里见过今天这样的场面啊，脖子还隐隐作疼，呼吸还困难。
<br/>
<br/>        傅海生也是一惊，不知道陈子霖为什么去而复返。
<br/>
<br/>        “傅总，傅总夫人，我回来是给你们道歉的，我着急动手是我的不对，你们所做的事情当然会受到惩罚的，但不是我私人惩罚你们！”陈子霖真诚的说。
<br/>
<br/>        “这哪里是道歉啊？这简直就是真诚的赤/裸/裸的威胁。”江俞轩诽腹。
<br/>
<br/>        “子霖，是我们的不对，夏颜，你赶快给子霖赔不是。”
<br/>
<br/>        “子...子霖，我当初年少不懂事，我现在也是后悔了，我做梦都梦见子昂要拿回属于她的东西，我......”夏颜哭了。
<br/>
<br/>        “其实，子霖，我们当初都是不知道的，是，是大人们干的事情。”傅海生也着急的说。
<br/>
<br/>        “我不管是什么人干的事情，但是她肯定是知情的，我不会追究你们的，但是有人会追究你们，我回来只是为了我刚才的事情给你们道歉。不打扰了！”陈子霖转身就走。
<br/>
<br/>        “去哪里？”
<br/>
<br/>        “工作，去看看他们看得怎么样了。”
<br/>
<br/>        江俞轩站住看了好一会陈子霖的背影，那紧绷的挺拔的背影，高抬着头坚定的朝前走着。
<br/>
<br/>        在上次吃饭的时候，他见过陈子昂其他的哥哥姐姐们，他们都是淳朴的，柔和的，每个人都给人一股温良的感觉，而上一次的陈子霖也是柔和的，不露一丝锋芒，偶尔和他闹闹小脾气，展示着他宠妹的特制，别的他倒是没有看出来，他以为，陈子霖只是一个傀儡，是陈子昂放在西城的一个自己人。
<br/>
<br/>        可是，这一次的陈子霖给了他不一样的感觉。
<br/>
<br/>        陈家，会是怎么样的一家人呢？
<br/>
<br/>        农村的人都与他们一样吗？
<br/>
<br/>        “你怎么不走了？”陈子霖发现江俞轩没有跟上来，转过身发现江俞轩在发呆。
<br/>
<br/>        “嗯，走!”
<br/>
<br/>        “江总，你说这个项目我们该怎么做？”陈子霖直接问起了工作上的事情。
<br/>
<br/>        “常规做法，这个项目目前对我们来说没有什么难度，轻车熟路，你在西城也跟了很多这样的项目，不过，你和傅海生的这一闹，也许会影响后期的合作。”
<br/>
<br/>        “傅海生不可虑，主要是夏颜。”
<br/>
<br/>        “怎么说？”
<br/>
<br/>        “傅海生这个人我们也见了几次，他是能将私人和工作上的事情分开的，只要我们方案做的好，能拿出他们合适满意的方案来，后期合作的机会不是没有，在西城这种项目能和我们pk的公司不多。
<br/>
<br/>        至于夏颜，她明明知道她是顶替我妹妹上的学还那么心安理得，而且昨晚说了那么多话，就知道她一定不是省油的灯，她压根就不会悔改，相反，会成为我们在这个项目上，或者是我们西城公司发展的绊脚石。”
<br/>
<br/>        “陈子霖，你知道夏颜吗？”
<br/>
<br/>        “上次回去了就了解了一下傅海生的喜好和他的家庭，他和夏颜的关系不怎么好，所以他先到的洛城，夏颜后去的西城，夏颜现在是某部门信息部老总，刚好和我们有接触。我也知道他们和子昂是同学，但是不知道他们竟然有如此深的纠葛。”陈子霖叹了一口气。
<br/>
<br/>        “江总，你跟着我妹妹一起工作了很多年了吧？”
<br/>
<br/>        “是的，六年多了。”
<br/>
<br/>        “她一个人过得很辛苦吗？”
<br/>
<br/>        “还好，公司的人都是齐心协力的。”
<br/>
<br/>        “当初，你和我妹妹一起回来，我还以为你是我妹妹的男朋友呢。如果，她上了大学也就不认识张函了吧？如果不认识张函她现在应该不会是一个人吧？”
<br/>
<br/>        “事情都有两面性，如果她上了大学，可能她现在的位置也就是和夏颜差不多，正因为她没有上大学，她凭着自己的努力，才有了现在的成就，她现在在业界的成就可不是傅海生和夏颜能比得上的。而且，她现在的志向还大着呢。”
<br/>
<br/>        “哈哈，你倒是会安慰人。”陈子霖终于笑了。
<br/>
<br/>        江俞轩说的很对，无论什么样的结局，都是自己奋斗的结果，处在不一样的环境也是会有不一样的结果。
<br/>
<br/>        他们兄妹就是路边的野草，春风吹又生！
