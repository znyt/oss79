　　唰唰唰！
<br/>　　叶尘一连数剑攻出，逼得血天海只好放弃收回血日妖轮，全力抵抗他的攻击。
<br/>　　而他这一分神，便再也无法操控血日妖轮，
<br/>　　嗡嗡嗡~~
<br/>　　血日妖轮在一阵剧烈晃动之后，
<br/>　　嗖！
<br/>　　一颗血色珠子，从血日妖轮中窜了出来，飞入吞天鼎之中。
<br/>　　而噬血妖珠脱离之后，血日妖轮瞬间黯淡了下来，变成了一堆废铁坠落地面。
<br/>　　哗啦啦！
<br/>　　周围的血海大阵，没有了血日妖轮的支撑，开始迅速消退，一转眼便消失的无影无踪，原本被血海包围的黑岩城，也得以重见天日。
<br/>　　“不！！这不可能！”
<br/>　　血天海看到这一幕，顿时不甘的大吼了一声，两眼死死盯着叶尘，几乎要喷出火来，
<br/>　　“你到底用了什么手段？我的血日妖轮，乃是仙器，怎么会变成这个样子？”
<br/>　　叶尘哈哈一笑，
<br/>　　“我说过，我的手段，可不是你能揣测的！给我去死吧！！”
<br/>　　叶尘大喝一声，再次举起紫琼剑，飞扑了上去。
<br/>　　血日妖轮被废，血天海本来就惊恐不已，再加上没有了血日妖轮提供的血煞之力，他本就不是叶尘的对手，顿时开始节节败退。
<br/>　　轰！
<br/>　　叶尘再次一剑斩出，血天海直接被轰飞了出去，落在数千米外的地面上之后，又足足滑行了数百米，方才稳住身形。
<br/>　　“等等！或许我们可以谈一谈，今日之事，就此作罢可好？”
<br/>　　眼看着叶尘如影随形般追了上来，血天海自知不是对手，开始主动求和。
<br/>　　可惜，叶尘理都不理，直接一剑劈落下来！
<br/>　　轰隆！
<br/>　　地面上顿时出现了一道，长达上千米的恐怖深坑！
<br/>　　“噗！”
<br/>　　血天海虽然再次勉强挡下，可是却忍不住喷出了一口老血，
<br/>　　“小子！你若是杀了我，你觉得血月宗会放过你吗？你一个人就算实力再强，难道还能一人对抗一个宗门不成？我血月宗之内，可还有两位不弱于本座的存在，我劝你最好还是给自己留点后路的好……”
<br/>　　血天海见叶尘不理不睬，顿时有些急了，开始搬出血月宗，希望能让叶尘心生忌惮。
<br/>　　可惜，叶尘依旧不为所动，
<br/>　　“聒噪！”
<br/>　　轰轰轰！
<br/>　　叶尘一剑接着一剑，每一剑轰出，都能逼的血天海倒退吐血。
<br/>　　一连七八剑下来，血天海的气息已经变得十分虚弱，只能依仗防御法宝竭力的抵挡，根本没有任何还手之力。
<br/>　　“罢了罢了！就当是本座向你求饶还不行吗？只要你放过本座这次，什么条件你尽管提，这样总行了吧？”
<br/>　　血天海无奈之下，只好又开始苦苦哀求起来。
<br/>　　远处围观的灵山等人，看到这个场景，感觉像是在做梦一般，
<br/>　　凶名赫赫的血月宗太上长老，居然在向一个少年求饶？
<br/>　　这事若是说出去，只怕都没有人会相信！
<br/>　　然而，
<br/>　　面对血天海的苦苦哀求，叶尘却根本不为所动，
<br/>　　因为他很清楚，像血天海这种人，就算今天自己放过了此人，他也不会感激，将来反而会更加疯狂的报复！
<br/>　　所以，要么不做，既然做了就要做绝，将这种后患，彻底扼杀！
<br/>　　“死！！”
<br/>　　叶尘抓住血天海虚弱的空隙，突然动用缩地成寸的神通，直接冲杀到血天海的跟前，然后狠狠一拳砸落。
<br/>　　彭！
<br/>　　血天海身体周围的外道神像，顿时被叶尘这一拳直接轰碎，整个人再次狂喷一口鲜血，倒飞了出去。
<br/>　　“结束了！”
<br/>　　还没等血天海的身体落地，叶尘已经如影随形般追了上去，硕大的金色拳影，犹如一座小山般，朝下方的血天海狠狠砸落！
<br/>　　血天海顿时瞳孔一缩，似乎也知道自己今日在劫难逃，眼眸中顿时闪过浓浓的疯狂之色，
<br/>　　“既然你不给我活路，那我们就同归于尽！”
<br/>　　还没等叶尘的拳头落下，血天海竟然自己主动冲了上来！
<br/>　　继而，
<br/>　　血天海的身上，陡然爆发出一道璀璨的血色金芒，同时身体开始迅速膨胀，皮肤上出现一道道恐怖的裂痕！
<br/>　　“他这是要——自爆？”
<br/>　　叶尘见状，也顿时反应了过来，不由得眉头一皱，当即收起拳劲，再次施展出缩地成寸神通，火速暴退！
<br/>　　一名化神天君自爆的威力，足以威胁到返虚强者，以叶尘如今的实力，根本不足以抵抗。
<br/>　　更何况，这是血天海拼着性命的最后一击，只要傻子才会往上面凑。
<br/>　　“彭！！”
<br/>　　就在叶尘的身体闪电般退到百米之外的同时，血天海整个人轰然爆炸！
<br/>　　轰隆隆！
<br/>　　一股令人心悸的狂暴能量，以血天海为中心，朝四面八方狂涌而去！
<br/>　　咔嚓嚓~
<br/>　　这一刻，连数百米之内的空间，都被这股狂暴的能量，震成了一片虚空！
<br/>　　“噗！”
<br/>　　叶尘虽然反应奇快，第一时间果断选择了倒退，可是一名化神天君自爆的能量，实在太过强大！
<br/>　　纵然他逃遁的速度已经极快，而且第一时间开启了琉璃金身和护体真元，可还是被那股狂暴的力量所波及，喷了一口血，受伤不轻。
<br/>　　不过，相比于血天海，叶尘的这点伤，却也不算什么了。
<br/>　　“没想到啊！没想到这样都没有没能杀死你！我恨！我好恨呐！！”
<br/>　　血天海自爆之后，最后一道残魂，目睹了这一结果，顿时发出了一道不甘的凄厉吼叫。
<br/>　　可惜片刻之后，那一缕残魂也随风消散，彻底消失在这个世间。
<br/>　　血月宗太上长老血天海，彻底陨落！
<br/>　　“赢了？”
<br/>　　“叶前辈居然真的击败了血天海！”
<br/>　　“我们得救了！”
<br/>　　……
<br/>　　黑岩城的众高手，忍不住欢呼起来。
<br/>　　然而就在众人无比激动兴奋之时，不知道谁突然冒出一句，
<br/>　　“现在说这个，恐怕还为时尚早吧！血天海被叶前辈杀了，血月宗岂会善罢甘休？”
<br/>　　原本还真激动不已的众人，听到这话，顿时一个个没了笑容，心情瞬间跌入谷底。
<br/>　　接着又有人开口道：
<br/>　　“是呀！倘若叶前辈一走了之，那我们黑岩城，必然会被血月宗屠城的！”
<br/>　　“为今之计，我们也只能将希望，寄托在叶前辈的身上了！”
