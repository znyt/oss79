　　　　当年的赌约，虽然只是口头之言，但他们两个都非常的在意，成为毕生所追求。
<br/>　　陈默本以为对方先行离世，那么赌约也就不复存在了。
<br/>　　可现如今，这条所谓的涂家家蛇，是何等的超凡脱俗，要不是涂兄培养，总不可能凭空出现吧。
<br/>　　而且如此觉醒类的蛇种，也没有义务去守护涂家。
<br/>　　只有一种可能，这条蛇，其实是涂兄他生前培养的。
<br/>　　另外最关键最可怕的是，这条蛇都已经化为人形，成为了真正的蛇王了。
<br/>　　“哈哈...涂兄，原来是你赢了，小弟陈默现在甘拜下风”
<br/>　　只见陈默抬头遥望星空，大声的咆哮，浑身隐隐颤抖，显的非常的激动。
<br/>　　这十几年前的赌约，今日划上了一个完美的句号。
<br/>　　涂小安被整的莫名其妙，这人身体残废了，莫不成脑子也残废了？
<br/>　　这都什么跟什么呀。
<br/>　　涂小安到不介意陈默怎么想他的身份，他将自己当成老爸所养的蛇，到也说的过去。
<br/>　　本来他也并不想将自己真实的身份说出来，因为那样太过离奇。
<br/>　　这般误打误撞，貌似也不错，一切还都能解释的过去。
<br/>　　陈默一声咆哮之后，还流下了一滴真挚的眼泪，很是感性，涂小安只是在旁静静的看着。
<br/>　　他并不太清楚这个人跟自己的父亲之间隐藏着什么样的故事，不过如此一说开，涂小安到是误会了蛇博士。
<br/>　　而陈默也算是误会了他。
<br/>　　最起码他们第一次有一个共识，就是不想涂家的两位女人再受到伤害。
<br/>　　半晌后，陈默冷静了下来，看着涂小安的眼神也柔和了不少，准备将心中的疑问一一的问出来。
<br/>　　“现在白镇的蛇王就是你吧”
<br/>　　“是”
<br/>　　“吸血蛇也是你杀的”
<br/>　　“是”
<br/>　　“那我徒弟想必也死了吧，还是你杀的”
<br/>　　“不是”涂小安转口。
<br/>　　陈默诧异：“那他还活着”
<br/>　　涂小安摇了摇头：“死了，死无全尸，被吸血蛇一口吞了”
<br/>　　陈默闻言，顿时一滞，随后叹了一口气，梦呓似的说：“泽儿，你为什么就是不肯听师父的话，那条蛇你控制不住，控制不住”
<br/>　　其实他们师徒，都是玩火自焚，最后都折在了变异吸血蛇的身上，但最后真正收益的，却是涂小安。
<br/>　　吞了变异吸血蛇，他登基坐殿，成为一代蛇王，君临天下。
<br/>　　陈默复杂的看着眼前这位少年，脑海中却是一条蛇的形象。
<br/>　　他说涂兄赢了，自己输了，这点陈默现在是输的心服口服。
<br/>　　吸血蛇虽然最后被他培养成绝世蛇种，但骨子里却是一条邪蛇，他玩火自焚。
<br/>　　而涂兄培养的这条蛇，在他的印象中，没做过一件坏事，不然白镇人民岂会奉为蛇王，今日的蛇王节之隆重也足够说明一切。
<br/>　　正蛇，行善举，保一方太平，邪蛇，无恶不作，伤天害理。
<br/>　　不管从那个层面，他陈默都输的一败涂地。
<br/>　　“博士，虽然你之前抓过我，但我也不曾恨过你，既然我们之间还有这段渊源，我想，我们应该不算仇人吧”
<br/>　　涂小安本来对陈默已经起了杀心。
<br/>　　但现在看来，完全没有必要再下杀手，况且对方还是自己父亲的故友。
<br/>　　于情于理，他都没那么做的必要。
<br/>　　“你是涂兄养的正蛇，守护涂家，我心甚慰”
<br/>　　陈默此时含笑的点了点头。
<br/>　　这也算一笑泯恩仇，开始不过是大水冲了龙王庙。
<br/>　　在白镇这个地界上，但凡是捕蛇人，相互之间都会很容易撤出一点渊源跟干系。
<br/>　　话说开之后，两人的关系明显变的微妙起来，涂小安推着陈默的轮椅，两人在这夜晚随意的开始散步闲聊。
<br/>　　陈默不禁的想，要是当初这条蛇是自己培养，成长到如此地步，那该多好呀。
<br/>　　将一条蛇培养化为人形，简直是一件无比荣光的壮举。
<br/>　　可惜他陈默一生与蛇为伍，却没这个命。
<br/>　　而他故去的老友有这个命，却没那个福。
<br/>　　“小子，你既然成为蛇王，化为人形，那么就要多修善果，为世人排忧就难，不可残害生灵，一正一邪皆在一念间，人也好，妖也罢，怀揣一颗善心，便是苍天之福”
<br/>　　陈默开始已一个长辈的心态面对涂小安。
<br/>　　涂小安笑了笑，继续推着他的轮椅向前走，道：“博士，你可知道今晚中毒晕迷事件，是何人所为吗”
<br/>　　“你知道了？”陈默微微扭头看了他一眼：“莫非真如你说的，他们中的是蚊子毒？”
<br/>　　涂小安道：“是蚊子毒不假，但是却是人为，还是你们人类觉醒者”
<br/>　　陈默闻言，好像并不是特别的意外，：“你说是人类觉醒者，利用特殊品种的蚊子伤人是吧”
<br/>　　“他们利用特殊品种的蚊子吸噬人的精血，制造血精，助人修为”
<br/>　　这时，一颗散发妖艳光芒的血精，出现在了陈默的面前。
<br/>　　陈默将这颗充满生命气息的血精拿在手中，脸色骤变，勃然大怒：“混帐，如此阴邪之术也用的出来”
<br/>　　“博士莫气，那人已经被我杀死”涂小安风轻云淡的说。
<br/>　　陈默微微变色，一位人类觉醒者说杀就杀，显然这条蛇王的实力，已经不是他可媲美。
<br/>　　“杀的好，这种败类杀一个不算多，杀两个都是福报”
<br/>　　涂小安没想到这博士还有点疾恶如仇的性子。
<br/>　　“博士，人类社会中，是不是隐藏着一些修炼的古武家族”
<br/>　　陈默诧异：“你连这个都知道了”
<br/>　　“被我杀的人类觉醒者说的，他也是被人指使，他背后的势力是所谓的白家”
<br/>　　此言一出，陈默脸色骤变，显然是知道什么。
<br/>　　“博士知道白家？”
<br/>　　涂小安之所以跟陈默说这个，就因为陈默本身是人类觉醒者，肯定知道很多他不知道的事情。
<br/>　　“嗯...温城白家，隐藏的修炼家族，聚集了大量的觉醒者”陈默看了他一眼，继续说：“现在世人只知道山野集团是温城的龙头老大，但那不过是表面现象，其实背地里，真正的龙头老大是隐世白家”
<br/>　　“这是一个拥有数百年历史的大家族，说起来，这个家族还是起源发家与白镇”
<br/>　　涂小安有点意外：“白镇，白家？”
