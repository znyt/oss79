　　第459章
<br/>　　天已亮了。
<br/>　　从东方跃出一轮红日，光明普照。程大雷没有休息，倚着大树往山坡下望。看见了一张张麻木憔悴的脸。
<br/>　　战争持续到现在，打的人很疲惫，挨打的人也很疲惫。大家都靠一口气撑着，这口气泄了，战争便也就结束了。
<br/>　　咚咚咚！
<br/>　　三军擂鼓，经过一夜的调整，帝国军已准备完毕，将要发动总攻。
<br/>　　程大雷将大旗交给银眸，道：“扛好它！”
<br/>　　银眸无声接过大旗，将义字旗扛在肩头。
<br/>　　程大雷骑在黑牛背上，举起了手中的鬼面斧，目光扫过身后的六百残兵。
<br/>　　“今日便是今日，我带你们冲出去，或者，一起拼光。”
<br/>　　“吼！”
<br/>　　六百人举起兵器，发出一阵狂啸。
<br/>　　尉迟离眉头颦起，道：“事到如今，竟还如此猖狂，来人，谁去山前叫阵。”
<br/>　　一匹快马出阵，来到半山坡叫骂：“呔，逆贼程大雷听真，某乃尉迟将军座下大将郑武，速速来吾马前受死。”
<br/>　　三军不发，大部分的视线都盯着郑武。李乐天微微眯眼，脑海中思索着什么。
<br/>　　郑氏一族，乃是帝国有名的武将家族。家中大部分男人都在军中任职，简单说，哪怕是郑家的一条狗，都可能是军犬，吃朝廷的狗粮。
<br/>　　李乐天心头寻思，若是能将这郑武收入麾下，郑家在军中的势力便会为自己所用。
<br/>　　这郑武是郑家着力培养的年轻人，过去几年，也堆积出一些名气。他使一杆方天画戟，在军中有美将之称，深受一些世家小姐的追捧，据说去他家提亲的人，将门槛都踏破了。
<br/>　　今日郑武抛头露面，未必没有再积累一些名气的意思。胜固可喜，但就算败了，也能在收获勇武之名。
<br/>　　李乐天正思索着，忽然见一头『怪兽』从山林上扑出，四蹄踏地，卷起烟尘动地。及到近了，李乐天才看清楚是一头黑牛。
<br/>　　程大雷骑在黑牛背上，一手拉缰绳，一手提大斧，宛若一阵风般出现在郑武面前。半程举起斧头，一斧挥下。
<br/>　　人与马，化为两截，郑武的血如被捅破的气球一般裂开。
<br/>　　程大雷风一般来，风一般而去，地上徒留郑武被分成两半的尸体。
<br/>　　帝国大军，雅雀无声。程大雷动作太快，一切只在电闪雷鸣之间，郑武还没亮起他的花戟，便已丢掉了姓名。
<br/>　　尉迟离扶额，心中暗道了一声可惜。这郑武出阵，未必没有看程大雷之前激战六将时没有杀人，错误判断了程大雷的人性。不料，今非昔比，程大雷出手有时候比林少羽更狠。
<br/>　　得，现在好了，帝国少一员大将，郑家年轻一辈少了一个继承人。
<br/>　　程大雷去而复还，砍掉郑武于他而言，并不费太多力气。真正的困难，在于如何从这里厮杀出去。
<br/>　　山坡四周，都是帝国军，两眼放光，目露凶色，想要突围，无非就是寻找薄弱处，全力攻击，至于能不能活着离开，也只有六个字：尽人事，听天命。
<br/>　　鼓声越来越急，程大雷知道，这是帝国军要进攻的节奏。他深吸一口气，低声冲举大旗的银眸道：“跟紧我。”
<br/>　　“杀！”
<br/>　　山坡下，帝国军发出一阵嘶吼，借助弓箭掩阵，大股士兵冲杀了过来。
<br/>　　“程大雷骑着黑牛，也同时发起冲锋。只是他没有向帝国军正面冲锋，手里拎着斧头转开了圈，寻找敌人薄弱处进攻。
<br/>　　黑牛宛若某种洪荒怪兽，冲进了敌阵之中。黑牛发出一声咆哮，撞翻了七八个人，程大雷拎着大斧开路，身边有扛旗的银眸帮他提防暗箭。身后六百残兵，也明白今日是绝路，到此种境地，除了放手一战，当真无任何办法。
<br/>　　人人一心，当真是锐不可挡，如一杆长矛，狠狠扎入敌人中央。可身边的敌人实在太多，一层层，一片片，杀不尽杀。
<br/>　　片刻间，起义军就有数十人被围杀。
<br/>　　程大雷抡着斧头战斗，鲜血淋漓，他从牛背上跃下，一个俯冲砍翻了一人的脑袋。
<br/>　　口中紧跟着一声呼哨，黑牛冲到他面前，程大雷翻身上了牛背，继续冲杀。
<br/>　　尉迟离在很远的地方观战，口中啧地一声：“世间竟有如此莽夫，若能收为己用，必是日后一大助力。”
<br/>　　“想要收下他？”李乐天道：“当初我父王都未做到的事，你觉得你能做到哦？”
<br/>　　尉迟离心中腹诽，心道：我也没被人一枪捅了。
<br/>　　他心中这么想，口中自然不会这么说，而是道：“臣才疏学浅，怎敢和明帝陛下相提并论。不过试试嘛，试试总是没有错的。”
<br/>　　李乐天摇摇头，看着程大雷厮杀的身影道：“没用的，有些人生来便注定不会寄人篱下，你要么杀掉他，要么被他杀掉。至于招降，哪怕他会痛快的答应，肯定回身就是给你一刀。”
<br/>　　尉迟离想了想，觉得李乐天说得也有道理。
<br/>　　“那么就杀掉吧。”
<br/>　　程大雷一阵厮杀，身边尸堆如山，然而越来越多的敌人靠近，数不胜数，杀不尽杀。
<br/>　　“逆贼，还我兄弟命来！”
<br/>　　郑家大郎郑雄，挥舞着一杆大棍杀来，他与郑武乃是一母同胞的亲兄弟，早已憋着一股劲儿，要杀程大雷报仇。
<br/>　　“好，我就送你们黄泉相会。”
<br/>　　程大雷其实也不知道这大将是谁，反正这种境地，杀了便是。一斧抡出去，将郑雄的脑袋砍了下来。
<br/>　　铛！
<br/>　　斧切喉咙，顺滑得紧，没有碰到任何阻隔。过程中，程大雷似乎听见嘟地一声，收到了一条系统提示。
<br/>　　眼前光景，程大雷也分不出空隙来查看系统提示。但只感觉杀掉郑雄后，身体有了改变，本已有些力竭的身体，重新获得体力补充。
<br/>　　简单来说，从前程大雷就像一口池塘，里面虽然存了很多水，但却是有出无尽，用完便算是用完了。
<br/>　　而现在的他，则像一条小溪，虽然水流很细，但却是有了源头，周而复始，往来不休。
<br/>　　程大雷在厮杀的过程中，终于抽时间看了一眼系统提示，时间紧张，只来得及看清几个字。
<br/>　　嘟，获得隐藏属性……
<br/>(本章完)
