　　楚风眠来到了白萱的洞府前，便是将身上的气息，给显现了出来，没有在隐藏。
<br/>　　这个时候，白萱才突然发现了，在她洞府前的楚风眠。
<br/>　　“哦？”
<br/>　　白萱的脸上，都是微微露出了几分惊讶。
<br/>　　“你的境界，果然突破了，没想到你居然可以闯过这云雾山脉的禁制，看来你在御风境的积累，不可小视。”
<br/>　　这云雾山脉上的禁制，就算是白萱自己，想要偷偷潜入其中，都不容易。
<br/>　　如今楚风眠，居然是无声无息之间的可以进入其中，这倒是大大超出了白萱的意料。
<br/>　　“小有突破罢了，我的实力，早就可以踏入神力境，只是在积蓄，厚积薄发罢了。”
<br/>　　楚风眠随意的开口道，倒是没有多谦虚，毕竟一味谦虚，只会让人看不起。
<br/>　　既然是打算跟这白萱一起去探寻虚空海，去寻找那颜古龙帝的洞府，楚风眠至少也要展现出足以跟白萱平起平坐的实力才行。
<br/>　　如此一来，才有这合作的余地。
<br/>　　“踏入神力境，以你现在的实力，晋升内门是时间问题了，差不多也足以位列地榜前十了。”
<br/>　　“如此也好，本来有些人对你的实力，还有些微词，不过现在你的实力，已经完全足以跟我们比肩了。”
<br/>　　白萱笑着开口道。
<br/>　　“你跟寒月漓，简直就是两个怪物，都不知道你们到底是从哪里冒出来的，一个比一个妖孽。”
<br/>　　“月漓那边怎么了？”
<br/>　　听着白萱的话，楚风眠一愣，随后询问道。
<br/>　　他倒是没有打听到多少关系到寒月漓的事情，不过看如今白萱的话，她却是知晓一些什么。
<br/>　　白萱毕竟是内门弟子，在北邙学院中也算是一个老人了，知道情报的渠道，自然要远远超越楚风眠。
<br/>　　楚风眠在北邙学院中，虽然打听过一些，但是知道的毕竟不具体，毕竟寒月漓现在是长老弟子，地位崇高。
<br/>　　很多事情，楚风眠也都不曾知道。
<br/>　　“你不知道？看来她应该还在秘境中修炼，所以没有来得及告诉你。”
<br/>　　白萱听到楚风眠的话，先是有些惊讶，随后开口解释道。
<br/>　　“其实我也是从我是师尊哪里听来的，我的师尊，是北邙学院的鸿君长老，跟冰璃长老是故交，所以我才知晓一些事情。”
<br/>　　“其实寒月漓师妹一来北邙学院，便是被冰璃长老，送入到了北邙秘境之中去修炼。”
<br/>　　北邙秘境。
<br/>　　这个词，楚风眠也是从拓跋鼎哪里听说过的，这北邙秘境，是北邙学院真正的修炼圣地。
<br/>　　比起圣山，都要神秘的多，圣山，只要是有着足够的功勋值，也可以兑换资格。
<br/>　　但是若是想要前往北邙秘境的话，只有一些位高权重的长老，才可以去向学院申请才行，而且也需要付出极大的代价。
<br/>　　看来冰璃长老，是真的想要好好培养寒月漓，居然是将她送入到了北邙秘境之中去。
<br/>　　也怪不得楚风眠打探不到寒月漓的消息，这北邙秘境，绝非是现在楚风眠可以接触的到的。
<br/>　　楚风眠心中暗自想着，一旁的白萱却是继续开口道。
<br/>　　“就在昨日，寒月漓便是在我们北邙学院的北邙秘境之中，度过了生死大劫，踏入了生死境。”
<br/>　　“而且寒月漓所度过的，还是千雷皇劫。”
<br/>　　这白萱的两句话，顿时如同一颗惊雷，令楚风眠脸上都露出了几分震惊。
<br/>　　寒月漓居然是突破了，踏入到了生死境，还引来了千雷皇劫。
<br/>　　这千雷皇劫，虽然不如万雷帝劫，但是能够引起的，也都只有天才中的天才。
<br/>　　寒月漓的资质果然不错，引出了她身躯中的玄寒之力，令她一跃引来了千雷皇劫，一旦度过，只怕对寒月漓的将来，都有着极大的好处。
<br/>　　踏入到了生死境，以寒月漓的资质，绝对不会弱于一些生死境三重，四重的强者。
<br/>　　在北邙学院中的实力，只怕已经是超越了寻常长老了，地位一下子暴涨。
<br/>　　“现在寒月漓的地位，可以说已经不再许多长老之下了，甚至是得到了学院一些老古董的重点看重，已经宣布晋升种子弟子了。”
<br/>　　白萱看着楚风眠，看着楚风眠脸上没有任何的担心，只有喜悦的表情，不由的惊异一声道。
<br/>　　“你不担心？”
<br/>　　“我担心什么？”
<br/>　　楚风眠随意笑了笑开口道。
<br/>　　寒月漓的实力越强，楚风眠自然也就越开心，起码她在这北邙学院中，地位水涨船高，也会安全许多了。
<br/>　　毕竟寒月漓可是玄寒之体，先天神体，对许多武者来说，可都有着极大的诱惑力。
<br/>　　寒月漓的实力越强，现在楚风眠的心中也就越是安心。
<br/>　　“寒月漓现在的地位，已经是超过你我了，被北邙学院的许多老古董看重，重点培养，只怕是很有可能会晋升核心弟子。”
<br/>　　白萱看着楚风眠，眼神中露出几分异样光芒。
<br/>　　“你的实力虽然突破，但是毕竟还只是神力境四重，距离生死境还有着不小的差距，最多只能是晋升成为内门弟子。”
<br/>　　“据说现在北邙学院中的许多核心弟子，都对寒月漓追求有加，甚至是核心弟子中的第一人，圣党的首领，夜天君都在追求她。”
<br/>　　“你跟寒月漓的关系，不止是朋友那么简单吧，你就不担心？”
<br/>　　白萱开口之际，也在观察着楚风眠脸上的表现，楚风眠表情中的任何变化，都瞒不过她的眼睛。
<br/>　　但是令她有些失望的是，楚风眠的表情并没有担心，并没有嫉妒，甚至是没有任何的变化。
<br/>　　只有极为平静，自信的笑容。
<br/>　　“月漓的地位水涨船高，我只有开心，至于其他人，要是他们敢去打寒月漓的主意的话，那就只有死路一条。”
<br/>　　楚风眠平静的开口道。
<br/>　　但是在其中，却是听见了一种令人不寒而栗的杀意。
<br/>　　就算是白萱，感觉到这杀意的一刻，都是心神不由的战栗起来。
<br/>　　楚风眠的境界，明明只有神力境四重，但是白萱听到这声音，都内心极为恐惧。
<br/>　　楚风眠不是不会担心，嫉妒，但是楚风眠有着自信，谁敢去打寒月漓的主意，楚风眠便是杀了谁。
<br/>　　就算是核心弟子中的第一人，夜天君，楚风眠也迟早杀了他。
