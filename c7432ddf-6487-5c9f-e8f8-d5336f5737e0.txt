　　至于之后幻境的幻灭，这还真是个意外，叶少阳本以为还得费劲寻找出口，结果直接就出来了，在幻境泯灭的瞬间，叶少阳生怕自己魂魄被拉走，急忙定身念起了清静经，一睁眼，发现自己坐在地上，周围一片漆黑，只有一点亮光，于是抬头望去。
<br/>　　“你醒了？”是谢雨晴的声音。
<br/>　　叶少阳朝她看去，谢雨晴也睁大了眼睛看着他，“你没事吧？”
<br/>　　“我怎么了？”叶少阳看见四周，黑咕隆咚的，就问是什么地方。
<br/>　　“还是在山洞里啊，我们一路跑过来，结果不知道为什么，出去的路没了，然后你就昏迷了，我也不敢乱走，只好在这守着你，你真的没事吗？”
<br/>　　“没事没事。”叶少阳在心里回想了一下，原来之前从看到出口开始，就都是幻觉了，之前那个谢雨晴也是假的，想到这，心中一阵后怕，这幻境与现实，还真是有无缝对接啊，而且模拟的也太真实了，连周围的景物、游人什么的全都有，如果不是瞳孔投影这个极小的漏洞，自己八成现在还被蒙在鼓里，后面还会发生什么，还真不知道。
<br/>　　这下恐怕真遇到棘手的对手了。
<br/>　　叶少阳起身缓了缓，从谢雨晴手中接过灯笼，朝四周照去，发现是个石室，四周什么都没有，也没有灯，怀疑这还是幻境，毕竟从景区工作人员考虑，不大会弄这么一个黑咕隆咚的山洞出来，而且什么景观也没有。
<br/>　　这是谢雨晴表示，刚他们一路跑过来的时候，她一扫眼看到有条岔路，于是叶少阳拉着她，小心翼翼地往回走，走了有二十米远，就看到了右手边的山洞，里面有幽绿色的灯光，是安全出口的标志，叶少阳本想检查一下，但之前那些人形妖尸喷出来的雾气，已经蔓延地哪里都是，妖气弥漫，只好让谢雨晴跟在自己后面，小心地走进去。
<br/>　　越往前走，空气越冷，但好在有安全出口的标志指引，往前走了有十来分钟，前面看到亮光，两人快步走过去。
<br/>　　“出来了！”
<br/>　　谢雨晴长出一口气，整个人几乎瘫倒在叶少阳身上，看叶少阳凝神望着四周，没什么反应，问他怎么了。
<br/>　　“之前我走过一次了。”叶少阳默默说道。
<br/>　　这条路，的确是之前他们进洞走的路，石碑、垃圾桶什么的都还在，游人也还是来来往往，一切看上去都很正常的样子。
<br/>　　如果没有之前的经历，叶少阳会毫不犹豫地认为这一切是真的，但是……在刚才的幻境里，这一切已经经历过一次了，再次看到，叶少阳一时间也判断不出真假。
<br/>　　谢雨晴听说了之后，走到路边，伸手拍了拍栏杆和树枝，又揉碎了几片树叶，道：“这种感觉好真实的，看上去不像是假的啊，对了少阳，不是说咬舌头能破梦魇，你……”
<br/>　　“那只能破普通幻境，对这种没用。”
<br/>　　叶少阳走到她身边，装作不经意地从她脸上扫了一下，幻境是其次，他现在最想确定的是，眼前的谢雨晴到底是不是真的，这才是最关键的。
<br/>　　瞳孔有自己的投影，看上去没问题。
<br/>　　“你看我干嘛，现在怎么办？”谢雨晴被他看得有点不自在。
<br/>　　叶少阳沉吟起来。
<br/>　　幻境，大致有三种，第一种就是类似鬼打墙的那种一般邪物用自身气息创造出来的结界，因为实力有限，不可能大范围模拟现实，最多在一些路口的地方模拟出假象，让人误入歧途，进入的人，走的路其实还是真的，破这种幻境最容易，最简单的办法就是拿出手机，找到方向，不用管脚下的路，一口气往前走就行，只要破了结界，就走出去了。
<br/>　　第二种是制造大范围的幻境，人在里头所看到的一切，都是假的，只有自己是真的，因为结界闭环，整体影响了内部的气息和磁场，仪器什么是没用的，就连法器也没用，这种幻境，普通人遇到是怎么也走不出去的，如果是法师，那就只有找到阵眼所在，然后硬闯。
<br/>　　第三种幻境更加高明，同样一切都是按照现实模拟出来的假象，连自己的肉身也是假的，只有元神或魂魄才是真的，就像自己刚才所经历的：以为自己走出了山洞，真身其实还在山洞中昏迷不醒。这种环境，也有不同，一类是只模拟真实，不能模拟细节。
<br/>　　简单说就是，假如一个邪物制造了幻境，用来迷惑一个人，幻境的一切，靠的都是邪物自身对这件事物的了解，山水什么的最好模拟，人也没问题，但如果这邪物不会玩手机，那一定模拟不好，最多模拟出外形，内容什么的不可能模仿好。
<br/>　　如果被困的人拿出手机一看，就会露馅了。
<br/>　　不光手机这类精细的高科技，就算是山水人物，细节上也有问题，这种环境就像电脑游戏一般，乍看上去很像真的，但放大局部，去观察细节，就会发现很多地方有问题。像素越高，细节上就越真实，对比幻境，幻境操控者的实力，就是这决定环境真实程度的像素。
<br/>　　因为有这些漏洞可以识别，这还不是最高明的幻境，最牛逼的幻境，是利用被困者的神识自己模拟出一个世界，被困者所看到的一切，都是假的，但都是自己意识产生的结果，说白了，一切都是受困者自己幻想出来的，就像做梦一样，手机、电脑什么的，一切都是大脑根据自身知识幻象出来，都跟真的一样，就算受困者察觉不对，怀疑是梦境，也未必就能跳的出去，并且无法改变梦境。
<br/>　　这才是最牛逼的幻境，与真实世界也最难以区分，甚至电话都可以打出去，只是接电话的声音，也是自己在“做梦”而已。
<br/>　　叶少阳拿出手机摆弄了一下，没什么问题，于是收起来，又想了一下，对谢雨晴道：“我们还是下山吧，如果这是幻境，范围一定是有限的，我们先下山再说。”
