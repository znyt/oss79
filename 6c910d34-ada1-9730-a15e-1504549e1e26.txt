　　第258章
<br/>　　周霜起身而立，手握着伞柄：“你受伤了。”
<br/>　　程大雷肩膀的确受伤了，而且是惯用的右肩。
<br/>　　“你的确有些小聪明，不过想要活到最后靠小聪明是没用的，靠的是实力。”
<br/>　　程大雷撇撇嘴，轻轻弹了弹剑锋：“你知道我的天赋有多高么？”
<br/>　　周霜眼睛眯起，鱼龙卫收集天下人情报，曾专门调查过程大雷。程大雷从出世以来的短时间内，做了很多人一辈子都做不到的事，或许他本就是个擅长战斗的天才。
<br/>　　“好啦，其实我也没有什么天赋，但你知道我有多努力么？”程大雷横剑在身前。
<br/>　　周霜不得不警惕，有时候一个足够努力的人，要比天才更可怕，天才易夭折，而勤奋的人往往能走到最后。
<br/>　　“其实也没怎么努力啦，只不过，我开挂的你懂不懂？”
<br/>　　“开挂？”周霜微开双唇，在琢磨着这两个字。
<br/>　　程大雷出剑，身体由静向动，静若石，动若电。周霜一惊，剑芒如星，一点点向他逼近，在他瞳孔内愈发扩大。他从未见过这么快的剑，下意识撑伞去挡程大雷的剑。
<br/>　　这黑伞中夹了金丝，在阳光下会有龙鳞般的金光，普通的兵器根本刺不破，只要自己能将伞撑开，绝对能挡住这一剑。而伞尖的毒针也可以收割掉程大雷的生命，他知道自己小看了程大雷，所以一上手就是杀招。
<br/>　　前提是他能将伞撑开。
<br/>　　撑开一柄伞需要多长时间，刺出一剑又需要多长时间？
<br/>　　在周霜得到的答案是，撑伞需要的时间太长了，比刺出一剑更长。
<br/>　　空气中一抹电闪，他喉咙出现一个血洞，然后睁大眼睛，脸上浮现惊愕的表情，而这个时候，他手中的黑伞才撑开一半。
<br/>　　带着最后的惊愕与疑问，周霜仰面倒地，呼吸断绝，而于此同时，他的黑伞才因机括的弹力而打开。
<br/>　　李婉儿震惊的看着这一幕，跌跌撞撞，她的心情经历太多起伏，直到现在还未平静，唯一能确定的事情是，程大雷还活着。
<br/>　　程大雷看着水中的李婉儿，二人目光对视，李婉儿从他的眼里，看到狮子般的凶恶。
<br/>　　“瞧瞧你爹带出来的队伍，什么玩意。”
<br/>　　程大雷骂了一声，身体一下瘫倒在地上，肩上的伤口涌出大量的血。在负伤状态下使用快剑，对自己的身体是巨大的消耗。
<br/>　　他没有坐多久，便撑着身子站起来，给赤发鬼、周霜和杀人书生各补了刀，虽然三人活下来的可能已经不大，但现在程大雷没能力承受任何意外。
<br/>　　过程中他看到了他们的眼睛，死前的眼睛带着恐惧，绝望，哀求，将要沉入那个不透光的深渊，每个人都会恐惧吧。但程大雷很平静，假装没瞧着。
<br/>　　绝地求生就是绝地求生，你不能在一局绝地求生内，招呼人来一场斗地主。
<br/>　　他将周霜的伞捡起来，掂了掂发现是一件不错的武器，又从周霜身上搜出针囊，研究着如何装填。
<br/>　　“小心，有毒。”李婉儿提醒道。
<br/>　　“我明白。”程大雷将伞合起，转向李婉儿道：“我去马车上，你穿衣服出来吧，这里死了人，他们很快能找到这里。”
<br/>　　程大雷在马车上清理了伤口，肤色因失血过多而显得有些苍白。李婉儿看到他时，见他额头渗着汗珠，但他的目光依旧很平静。
<br/>　　“是我害的你，而我什么都帮不了你。”李婉儿。
<br/>　　“在这个游戏里，你就是个吉祥物。”程大雷揉揉她的脑袋，自顾自道：“我的任务是护送你离开，所有人都想争夺你，没有人是可以相信的。“
<br/>　　李婉儿听不懂程大雷的话，只是看着他。
<br/>　　“而你可以相信我。”程大雷笑笑，鞭子抽在毛驴身上。
<br/>　　“吉祥物是什么啊？”马车上传来李婉儿的声音。
<br/>　　“就是那种很可爱，很好看，但却没什么用的东西。”
<br/>　　夕阳西下，草原荡起一片红色的金，毛驴拉着马车，从车上时不时传来笑声。
<br/>　　……
<br/>　　在夕阳最后一抹温暖都沉入西山下时，程大雷停留过都水泡子边出现两个人。
<br/>　　一人耷拉着眉毛，看上去像没有睡醒，在他脚边卧着一只黄色猎犬。
<br/>　　另一人蹲在地上，叼着一只烟袋，眼睛盯着地下，任何蛛丝马迹都不肯放过。
<br/>　　鱼龙卫，猎户与捕头。
<br/>　　“看出什么了么？”猎户打了个哈欠。
<br/>　　“赤发鬼，杀人书生，行者都是江湖上一流好手，现在都死了。”
<br/>　　“就算瞎子都能看出他们已经死了，程大雷杀的一流高手也不是第一个，你能不能说点什么我不知道的。”
<br/>　　“瞎子怎么看？”捕头道：“我知道一路以来已经有很多人死在他手上，乔家父子，雁北薛雄，可是，这次有些不一样。”
<br/>　　“那里不一样？”
<br/>　　“乔家父子，雁北薛雄，他们都是喉部中剑，一剑封喉，但这三人除了周霜，致命伤都不是喉部。”
<br/>　　“就是说程大雷最后杀死的只有周霜。”
<br/>　　捕头点点头：“我有种感觉，程大雷的剑越来越快了，这快草原，好像成了他的磨刀石。”
<br/>　　“以人头磨剑，果然是个魔头啊，要不我们撤吧，别让我们的脑袋也给他磨剑。”
<br/>　　“他好像受伤了。”捕头盯着草地上散落的血滴。
<br/>　　“那还等什么，赶快走啊，趁他病要他命。”
<br/>　　“他往那个方向去了？”捕头站起身，抽了抽烟袋，有白色的轻烟冒起。
<br/>　　猎户踹踹脚下的猎犬：“别趴着了，赶紧找人。”
<br/>　　那黄色猎犬爬起来，在地上嗅嗅，忽然向北方跑去。
<br/>　　“如果是想回到关内，他应该往南走，为何他要一直往北走。”捕头皱眉。
<br/>　　“大概是知道关内有人守着，想先去极北之地躲一躲避一避风头。”猎户。
<br/>　　“嗯呐，你说得也有些道理。”
<br/>　　“北方……北边守着的人是屠夫和赌鬼吧？”
<br/>　　“嗯，也许我们已经不用过去了。”
<br/>　　鱼龙卫，鱼卫三千，龙卫三十六，和尚，道士，屠夫，赌鬼……各人皆身怀异术，屠夫不是其中最强者，但一定是其中最残暴的一个。
