　　曹春辉满脸惊恐，不安的回答道：“我们已经在尽力研究了，那三种药剂我们也在随后的第一时间研究了出来，随时可以……”
<br/>　　“那不过是你根据我给你的原液复制调配的而已不是么？”异瞳男子打断曹春辉的话，很是不屑。
<br/>　　曹春辉一个哆嗦，赔笑道：“但我也是比狼牙先研究出了新人类爆发药剂不是么？能让新人类在短时间内战力提升好几个等阶，战斗力超群，这狼牙可没研究出来吧？”
<br/>　　男子冷冷的看了他一眼：“爆发药剂的确是个暴力的东西，副作用也大的很，用过这种药剂的人活不过一年，利用药剂透支他们的生命获取短暂的能量爆发，你这是想要我每培养一批人出来，用完就死？”
<br/>　　曹春辉不敢说话，低着头满脸惊恐。
<br/>　　男子继续冷笑道：“再看看狼牙，人家研发出来的是最低副作用的等阶提升剂，整个狼牙战团的大几万人都在用，也没发生过意外！”
<br/>　　随着这名男子的愤怒咆哮，身旁的楚云天和胡梦浩则是满眼异色，他们常年待在封闭的底下研究院内，对外界的情况一无所知，之前的南署基地大尸潮一战情况，已经让两人大为震惊，而眼下有关狼牙研究部研发而出这三种药剂的事，他们更是第一次听说！
<br/>　　丧尸引诱剂，新人类觉醒剂，等阶提升剂。
<br/>　　世上竟然有如此厉害的研究部，研发出了这么高端又是当下这个世界最需要的东西，完全甩了曹氏研究院好几条街啊！
<br/>　　楚云天对狼牙不了解，更对这个狼牙研究部的情况一无所知，只是在震惊之余感叹这个叫做狼牙的组织，不仅战团如此强势厉害，连后方供应需求的研究部都逆了天。
<br/>　　就是不知道这个逆天组织的首领，是谁呢？
<br/>　　楚云天有些好奇，更腾起了一股想要与之结交的心思。
<br/>　　对比楚云天的好奇与震惊，胡梦浩的情绪则更多的是打脸般的痛快！
<br/>　　他和楚涵有过约法三章，两人属于同一阵营，楚涵的狼牙越强，胡梦浩越高兴，他在这个扭曲的曹氏研究院早就待不下去，若不是有着坚持的信念，他甚至想结束生命。
<br/>　　而当漫无天日的等待中，忽然听闻外界的消息，楚涵一手建起的狼牙竟然已经发展到这个规模，这让胡梦浩如何能不兴奋？
<br/>　　“罗杰大人！请再给我一次机会！我一定会超过狼牙研究部，一定能研究出比他们更出色的东西！”忽的，曹春辉满脸泪痕的出声，双膝更是直接触地，就这么毫无尊严的跪在了异瞳男人的面前。
<br/>　　罗杰？
<br/>　　胡梦浩在心中默念了一遍这个名字，将这一细节牢记。
<br/>　　他知道他早晚有一天会出去，而那一天到来的同时，他也需要将这座研究院内的一切有用消息告知楚涵。
<br/>　　显然在这个名叫罗杰的男子出现之后，胡梦浩就已经知晓曹氏研究院并非属于曹春辉，其背后另有人在主宰。
<br/>　　这个异瞳男子自然就是神秘家族罗家的嫡系少爷罗杰，罗家在回归华夏之后，便同时进行了很多条线的发展。
<br/>　　除却明面上赞助的几家基地，亦或是找来的大批罗家代言人，其资源分配都不如这座底下的曹氏研究院来的多。
<br/>　　眼下还没有多少人知道这个研究院的所在，这也是罗家的底牌！
<br/>　　早在两个月前罗杰发现了分界线大战的不对劲时，就放弃了坐以待毙，选择亲自调查。
<br/>　　只是走访了白家无果，又亲自去往了战区调查后，罗杰便发现了狼牙战团为何如此强大的秘密。
<br/>　　一个战团如此多的人，怎么可能一如狼牙就变成了新人类，并且在短时间内升阶这么快？
<br/>　　神秘家族想要调查什么很简单，于是早在其他基地的首领知晓之前，罗杰就已经知道了新人类觉醒剂和等阶提升剂的存在！
<br/>　　并且他确定，这件事高家和白家也已经知道，只是白家竟然没有告诉他，白忧竟然在他面前选择避而不谈！
<br/>　　除却这种被动的不爽感之外，罗杰更愤怒的是他花了大价钱培养的曹氏研究院，在这么长的时间里都没有任何研究成品，而狼牙白手起家，却偏偏在各方面都超越。
<br/>　　于是罗杰第一时间弄到了这两种新型药剂的原液，交于曹春辉研究。
<br/>　　曹春辉毕竟也是文明时代门萨内智力超群的人，在短短两个月内就将这两支药剂研究了个彻底，配方和细节都已经调查的一清二楚，也能够随时调配出。
<br/>　　只是曹春辉心里很清楚，罗家早晚会来兴师问罪，而这一天，就是他的死期！
<br/>　　果然不出所料，当南署基地大尸潮一战结束，狼牙战团再次开挂了似得取得大胜之后，罗杰的怒火再也压制不住，亲自来到了研究院。
<br/>　　这也是曹春辉会如此恐惧，甚至不惜跪下求饶的原因。
<br/>　　罗杰望着跪在地上的曹春辉冷笑：“你倒是说说，我要怎么再给你机会，你还能研究出什么？”
<br/>　　“我，我……”曹春辉语塞，眼珠不停转动着思考，忽的他猛然双目一亮，慎重的抬头坚定道：“我会研究出解除丧尸感染的药剂！”
<br/>　　此话一出，旁边的胡梦浩和楚云天都一愣，而后皱起眉。
<br/>　　丧尸病毒他们一直在研究，结果却发现无论如何努力，都无解，这种感染是不可逆的。
<br/>　　曹春辉此刻说的话，没有根据和道理，也是在做最后的挣扎。
<br/>　　罗杰好笑的看了曹春辉一眼，冷嘲道：“感染的病毒解药啊？这种东西对我们罗家有什么用？你以为区区丧尸能伤的到我们？”
<br/>　　“不，不是……”曹春辉没想到自己用了最大勇气说出的话，却被罗杰当面反驳了回来，想到罗杰话中的意思，他连忙解释道：“我是说，大人您如此高贵，罗家又是名门贵族，丧尸这种杂种怎么可能需要罗家人亲自动手呢？罗家可不是狼牙战团，您更是高贵无比的存在，跟楚涵那种货色完全一个天上一个地下，您当然不用在意感染不感染的。”
