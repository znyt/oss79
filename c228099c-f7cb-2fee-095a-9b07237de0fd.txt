　　在苏莫向南洲而去的同时，整个苍穹世界也沸腾了，所有大势力之人，也都知道了，试炼者要来了！
<br/>　　霎时之间，无数的大势力纷纷立刻临时解散，化整为零。
<br/>　　无数的强者立刻逃窜，各自隐藏了起来，躲避杀身之祸。
<br/>　　面对试炼者，他们有的只是无尽的恐惧，哪里还敢现身！
<br/>　　一时间，整个苍穹世界，一片纷乱的场景。
<br/>　　无尽的海底，幽暗的洞穴之中。
<br/>　　一名黑袍人盘膝而坐，身上没有半点气息，仿若是一块朽木一般。
<br/>　　蓦地，黑袍人身形突然一震，而后立刻睁开了眼眸。
<br/>　　他的眼眸中，爆射出璀璨的精芒，将洞穴的岩壁都都洞穿了，发出嗤嗤的声响！
<br/>　　“哈哈哈！！！”
<br/>　　陡然间，黑袍人仰天狂笑了起来，面上充满了狂喜之色。
<br/>　　“终于来了！终于来了！等了数百年，终于等到了！”黑袍人状若疯狂，神情激动无比。
<br/>　　黑袍人自然便是巫森，他等了数百年，终于等到了巫族试炼者降临之日。
<br/>　　“哈哈！你们都随我离开吧！”
<br/>　　看了一眼手臂上的空间玉镯，巫森再次大笑一声，随即身形直冲苍穹，洞穿了厚实的岩壁，洞穿了无尽的海水，瞬间出现在了海面之上。
<br/>　　咻！
<br/>　　随即，巫森的身形，化为了一道黑光，划破长空，如风似电。向南洲的方向疾驰而去，速度快到巅峰。
<br/>　　……
<br/>　　南洲。
<br/>　　一片广遨而又荒凉的平原之上，虚空不断的震荡，传出一阵阵闷雷之声。
<br/>　　平原的上空，一处方圆数百丈的空间，虚空不断的扭曲，不断的变幻。
<br/>　　仿佛是有什么太古凶兽，在虚空之内肆虐，随时会破开空间冲出来一般。
<br/>　　时间缓缓流逝，约莫半个时辰之后，空间严重扭曲，形成了一个浩大的漩涡，开始向内塌陷。
<br/>　　轰隆隆！
<br/>　　惊天爆响不断，空间塌陷之处，很快便出现了一个巨大的空间通道，横贯虚空，不知通向何处！
<br/>　　空间通道很是虚幻，但随着时间的流逝，变得越来越凝实。
<br/>　　又过了一刻钟之后，空间通道彻底凝实，另一方也完全打通，通道中发出刺目的亮光。
<br/>　　而后。
<br/>　　嗖！
<br/>　　一道人影突然的从通道内飞了出来，出现在通道之外。
<br/>　　紧接着，破空声不断，一道道人影如同鱼跃龙门一般，从通道内鱼贯而出。
<br/>　　片刻之间，便有约一千人冲出了通道，出现在了高空之中。
<br/>　　这些人个个身上气息惊天，强大异常，而且这些人有两个共同的特点，其一便都是年轻人，有男有女，个个看起来二十岁左右，气度非凡。
<br/>　　其二，这些人的额头眉心之间，无一例外，全部都有一个三叉戟模样的标志。
<br/>　　这个标致，正是巫族的标致，这些人正是巫族的后辈试炼之人。
<br/>　　随着这一千人的到来，高空中的空间通道，逐渐变的虚幻了起来，几息之后便彻底的消失无踪。
<br/>　　“这就是苍穹世界？好脆弱的空间！”
<br/>　　一名身穿锦衣的高大青年，眼眸扫视四方，其微微摇头，手掌轻轻一挥，身前的大片空间便支离破碎。
<br/>　　“太弱了！这么弱的小世界，会出现武帝境的土著吗？”另一名穿着兽皮短衫的青年，满脸不屑之色。
<br/>　　“哎！苍云星支脉的人去了天澜世界，化灵星支脉的人去了真阳世界，我们天冥星支脉的人，却要来这个鸟不拉屎的苍穹世界！”
<br/>　　“这有什么办法？我们天冥星支脉实力最弱，自然只能选别人挑剩下的！”
<br/>　　“真是晦气啊！听说天澜世界和真阳世界，均是强者无数，珍宝众多，可惜我们没有这个机会！”
<br/>　　不少巫族之人纷纷开口，个个无精打采，语气中均是充满了羡慕和无奈。
<br/>　　他们巫族除了主脉之外，还有三大支脉，别的支脉的后辈天才，均是去往强大的小世界历练，而他们却被分到了孱弱的苍穹世界，这让他们非常的气恼。
<br/>　　来到苍穹世界历练，根本就没有任何挑战性！更别说得到什么上好的宝物了！
<br/>　　以他们的实力，挥手间便能横扫整个世界，屠戮众生，没有什么意思！
<br/>　　“好了，都别抱怨了！”
<br/>　　就在此时，一声冷喝陡然响起，充满了无上的威严。
<br/>　　霎时间，千名巫族天才，全部闭上了嘴.巴，再也没有一人出声。
<br/>　　众人的目光望向前方，均是落在了一名青年的身上。
<br/>　　这是一名身形挺拔，面容白皙，俊逸如女子一般的青年。
<br/>　　此人伫立在虚空中，仿佛就是天地间的主角，让人不敢直视。
<br/>　　而在青年的身边，还站立着一名女子，此女身穿兽皮短裙，五官小巧，面容娇美，黑发披肩。
<br/>　　但此女给人的气质，却并不柔弱，而是狠辣、果决、凌厉。
<br/>　　“苍穹世界虽然弱小，但我们的任务不能懈怠，必须要圆满完成！”面容白皙的青年再次开口，声音冷然。
<br/>　　“是，天御族兄！”众人闻言纷纷应声，不敢有丝毫反对之声。
<br/>　　面容白皙的青年，名为巫天御，乃是他们巫族天冥星支脉的后辈中，武帝境之下第一天才。
<br/>　　所以，众人自然不敢违逆。
<br/>　　他们的主要任务，便是猎杀苍穹世界强大的土著，然后收集高等级的武魂。
<br/>　　至于资源宝物，若是有的话，他们当然也不会放过。
<br/>　　“还有祖爷爷交代的事情，你们也别忘了！”
<br/>　　这时，巫天御身边的那名皮裙女子开口了，目光扫视众人，冷冷的道：“这个任务才是最重要的，完不成的话，你们知道后果！”
<br/>　　“娴姐放心吧！我们一定会找到大祭司的后人！”
<br/>　　“这个不难，有族印的话，轻而易举！”
<br/>　　“包在我们身上吧！”
<br/>　　众人纷纷表态，这个任务他们倒是不觉得有什么难度。
<br/>　　他们支脉的大祭司，年轻之时来到苍穹世界历练，在此留下了后人。
<br/>　　现在需要他们找到其后人，然后带离苍穹世界。
<br/>　　随后，一众巫族天才，纷纷拿出了一块海碗大小的玉石罗盘，开始查看了起来。
<br/>　　每一个玉石罗盘之上，均是有大量的红点，有的红点伫立不动，有的红点却在快速移动。
