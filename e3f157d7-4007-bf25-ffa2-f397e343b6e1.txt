        &lt;h3&gt;第三卷第四百零八章  陈天天的疑虑&lt;/h3&gt;
<br/>
<br/>        陈天天开着车将车停到地下车库，到了冯睿的家中，家里的一切都没有变动，陈天天悲伤的不能自已，冯睿再怎么不好都是她亲生的母亲，俩个人的关系刚刚缓和，冯睿就不在了。
<br/>
<br/>        陈天天在沙发上坐了一会，这个家里冷冰冰的，已经没有一点点烟火气息，陈伟冯估计最近也没有来这里，这里的东西还是以前的摆设。
<br/>
<br/>        “妈，你告诉我，是谁撞得你？”陈天天看着冯睿之前的照片，泪水长流。
<br/>
<br/>        “你告诉我啊，我一定要替你报仇，我绝对不会让撞你的人逍遥法外。一天找不到我一个月一个月找不到我一年，我一定要找到他！”
<br/>
<br/>        “妈，我不想失去你啊，你快点回来，我还想吃你做的饭呢。”
<br/>
<br/>        陈天天抱着冯睿的照片哭成了泪人。
<br/>
<br/>        “我去警局了，警局说他们正在找人，我也去三姑哪里了，三姑也说她在想办法，我昨天晚上梦见你了，你一定是不甘心吧？”
<br/>
<br/>        “妈，你放心，我一定一定要找到那个撞你的人。”陈天天语无伦次。
<br/>
<br/>        无论她怎么说话，照片上的冯睿还是微笑着一言不发。
<br/>
<br/>        陈天天哭了一会，站起来在房间里转了一圈，突然想起来冯睿给陈子建发的短信，她将冯睿的电脑打开，输入陈子建的生日。
<br/>
<br/>        冯睿的硬盘里有陈子建大量的研究性的文章，也有一些冯睿的日子。
<br/>
<br/>        冯睿详细的记录了她去看陈子建的经过，陈天天心里像刀搅一般疼。
<br/>
<br/>        陈天天一一的看下去，冯睿有写日子的习惯，里面写了她是怎么认识蔡少芬和弥清云的，怎么和弥青云他们来往的、冯睿是如何和夏颜来往的。
<br/>
<br/>        冯睿在写到夏颜的时候这样说：阴差阳错中我毁了一个人也成就了一个人，不过总体来说，这俩个人现在的成就都不一般，而夏颜还知恩图报，对我也很好，尤其是近一段时间，简直是比我亲女儿还好，给了我很多的东西，还将家里的钥匙给我了。
<br/>
<br/>        想起她说：冯妈，以后没事了你就过来给我做饭吃，我特别喜欢吃你做的饭，我这心里啊，竟然是满满的爱意，竟然只想余生天天给她做饭吃，这一段时间过得很充实，有了颜颜的陪伴，感觉不到那么可怕的孤独。
<br/>
<br/>        我也知道她很想认识更高一些层次的人，那么就将她介绍给蔡少芬和弥青云吧。
<br/>
<br/>        也许，我和她是同一路人，也许我们的心里对陈子昂都有那么一种敌意，我们极尽可能的黑化陈子昂，当夏颜对蔡少芬说出陈子昂的事情的时候，我有一种幸灾乐祸的感觉。
<br/>
<br/>        希望颜颜的成就能够比陈子昂大，这样也不枉当初的阴差阳错。
<br/>
<br/>        陈天天心里在滴血，她不知道冯睿是这样的人，冯睿两种性格的面孔在她的眼前飘来飘去，她关掉电脑，踉踉跄跄的朝楼下走去。
<br/>
<br/>        真相总是那么让人难以承受，那些文字就像是刀子一样剜在陈天天的心上。
<br/>
<br/>        陈天天走到地下车库，不小心拌倒在一辆车前。
<br/>
<br/>        陈天天蹲在地上哭起来，哭了好一会，扶着车头站起来，觉得车很眼熟，仔细一看，车头前有剐蹭的痕迹，车号也很熟悉，她突然想起来，这个是冯睿的车，她坐过一次。
<br/>
<br/>        陈天天给陈伟冯打电话。
<br/>
<br/>        “哥，妈妈的车一直在车库里放着吗？”
<br/>
<br/>        “是啊，车在车库里，车钥匙找不到了，我还说改天找个师傅去看看。那车放那里时间久了也是一笔不小的费用。”
<br/>
<br/>        “妈妈之前有出过车祸吗？”
<br/>
<br/>        “没有啊，我好像从来都没有听她说过，怎么了？”
<br/>
<br/>        “她的车头前好像被蹭了，也不知道是放再在车库蹭的还是在别的地方蹭的。”
<br/>
<br/>        “人都不再了，还管那么多干什么呢？你没事了去地下车库干嘛？”
<br/>
<br/>        “我今天来这里看了看，刚刚到车库取车准备回家。”
<br/>
<br/>        “早点回去吧，人死不能复生，别伤心了啊，你现在要好好读书，快快长大，过好自己的生活。”
<br/>
<br/>        陈伟冯因为冯睿的关系，和陈家人的关系一直比较疏离，对这个唯一的亲妹妹其实也没有太多的感情，但是从冯睿走了之后，他心里有了一点点变化，亲人不多，对陈天天也有了一点点的爱护。
<br/>
<br/>        “好吧，你也要好好的生活，哥，再见！”
<br/>
<br/>        陈天天看着冯睿的车，一丝疑虑涌上心头。
<br/>
<br/>        冯睿走之后，所以的东西都顺理成章的交给了陈伟冯，没有人关注冯睿生前用过什么东西，去过什么地方。
<br/>
<br/>        大家已经接受了冯睿离开的事实，人们已经回归到各自的生活状态中，没有人去寻找冯睿离开的蛛丝马迹，大家只是等着警局来寻找肇事者，谁也没有注意到冯睿的车有问题。
<br/>
<br/>        陈伟冯挂了陈天天的电话，陷入沉思中，他也知道冯睿是被车撞的，而冯睿的车钥匙不见了，前一段时间找了几天也没有找到，最近夏颜调走了，夏颜的位置空缺着，他很想要进一步，而和他能有竞争的人就是孙毅了。
<br/>
<br/>        夏颜和冯睿的关系不错，这么几年都是夏颜照顾着他，他才能到现在的这个份上，也不知道夏颜会不会看着冯睿之前的面子上再照顾他。
<br/>
<br/>        他准备去找一下夏颜，夏颜之前好像经常开着冯睿的车，陈伟冯决定见到夏颜的时候问问夏颜有没有见到冯睿的车钥匙。
<br/>
<br/>        陈伟冯试着给夏颜打了个电话，电话响了一会才接起来。
<br/>
<br/>        “夏姐，我是伟冯。”
<br/>
<br/>        “伟冯，有什么事情吗？”夏颜皱着眉头。
<br/>
<br/>        “我想见一下你。”
<br/>
<br/>        “我现在很忙，有什么事情你在电话里直接说就是。”
<br/>
<br/>        “其实也没有太大的事情，我妈的车钥匙不见了，之前有见你开过她的车，我就想问问你有没有见过她的车钥匙。”
<br/>
<br/>        “我没有拿过啊，我怎么会开她的车呢？”夏颜一口否认。
<br/>
<br/>        “哦，我想着你们关系很好，所以我就问问。”
<br/>
<br/>        “没有，你还有别的事情吗？”
<br/>
<br/>        “我......那个你走后，我们这里一直没有人管理，你看我........”
<br/>
<br/>        “你别着急，我会安排好的。”夏颜心里一动，洛城的项目当时其实有陈伟冯一起参加，那个时候陈伟冯还是一个普通的科员。
<br/>
<br/>        夏颜挂了陈伟冯的电话，给呼延临庆打了个电话：“准备四十，去找信息上的陈伟冯！另外，抽时间到我这里来一趟。”
