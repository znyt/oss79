　　绫裳离开终极神王殿之后，便径直向轮回神朝而去，前去寻找天尘宇。
<br/>　　她曾经也见过天尘宇，这个天赋绝世的无上神王，只不过，他当初认识天尘宇之时，对方还没有成为神王。
<br/>　　终极神王或许不知晓，天尘宇为何寻找苏莫，但是，绫裳心中却是非常的清楚，肯定是为了帝一魂。
<br/>　　所以，此行对于她来说，完全没有任何的意义，但是，她却是不能拒绝终极神王的命令。
<br/>　　绫裳的速度非常快，两日之后，便离开了终极神域的地界。
<br/>　　这在此刻，后方传来虚空呼啸崩溃之声，她转头一看，是紫霄追了上来。
<br/>　　绫裳见此，顿时停了下来，面色一沉，她已经猜到了对方的来意。
<br/>　　“绫裳，为兄有事和你商量！”呼吸之间，紫霄便来到了绫裳的身前，面带和煦的笑容。
<br/>　　终极神王命他暗中跟着绫裳，但他并没有这么做，因为他有自己的打算。
<br/>　　“紫霄，你有何事？”绫裳面色清冷如霜，淡淡的问道。
<br/>　　“你知道秦陨的底细，对吧？”紫霄微笑着问道。
<br/>　　“不知道！”绫裳摇了摇头，就算是知晓，她也不可能告诉对方。
<br/>　　“不知道？”
<br/>　　紫霄闻言，嘴角勾起一丝戏虐的笑容，道：“我知道你不会告诉我，但我想说的是，秦陨必须回来。”
<br/>　　“你什么意思？”绫裳美眸一眯，心中涌出不好的预感。
<br/>　　“你看这个！”
<br/>　　紫霄袖袍一挥，一个透明的水晶棺椁，出现在了他的身前。
<br/>　　白衣如仙的芊寻月，静静的躺在棺椁之中，仿佛亘古存在。
<br/>　　“这……棺椁怎么会在你的手中？”绫裳见此，清冷如霜的绝美面容，顿时大变。
<br/>　　她知晓紫霄是想通过她找到秦陨，但是，完全没有料到，芊寻月居然在紫霄的手中。
<br/>　　“这你就不必知晓了！”
<br/>　　紫霄再次一挥袖袍，收起了芊寻月的棺椁，微笑着道：“如果秦陨不回来，这个女人会怎么样，为兄可不敢保证。”
<br/>　　“你……紫霄，你居然如此的卑鄙？”绫裳的脸色，一阵青一阵红，对方的卑鄙手段，让她心中怒急。
<br/>　　“哎，绫裳，为兄也不想走到这一步！”
<br/>　　紫霄叹了口气，背负着双手，叹道：“你我当初一同成长，为兄的志向，你是明白的，荒界已进入末世，纪元终结已经不远，为兄只能不择手段了！”
<br/>　　紫霄的心中，有些无奈，对于面前这个曾经喜欢的女人，他也不想伤的太深。
<br/>　　只不过，为了证道，为了超脱纪元，他不得不这么做。
<br/>　　毕竟，只有自己强大，才是最重要的，女人不算什么。
<br/>　　“紫霄，我从未想过，你会变成这样！”绫裳的心很痛，上次紫霄以救芊寻月一事，要挟她以后成为其炉鼎，现在，又以芊寻月的生死，要挟她和苏莫。
<br/>　　这一切的一切，都让她愤怒。
<br/>　　她和紫霄，曾经一同成长，亲如兄妹，有数十万年的感情。
<br/>　　但是，今时今日，这数十万年的感情，就是一个笑话。
<br/>　　“绫裳，成全为兄吧，若是为兄能超脱纪元，一定尽全力保你平安！”紫霄面带深情的说道。
<br/>　　“我没有办法帮你，也没有能力帮你！”绫裳摇了摇头，清冷的面色，变得更加的冷漠。
<br/>　　面对绫裳的严词拒绝，紫霄并未生气，他缓缓转身，背对着绫裳，淡淡的道：“我给你时间，五年之内，秦陨不来找我，这棺椁中的女人，将缺手断脚；十年之内秦陨不来找我，这女人的那丝执念将会被抹杀，若是二十年之内，秦陨还不来找我，那这女人的尸体，只能拿去喂狗了；当然，这么绝色的女人，就算是拿去喂狗，事前也会丧失清白！”
<br/>　　言罢，紫霄背负着双手，脚踏虚空，大步离去。
<br/>　　“紫霄，你这个畜生！”绫裳脸色铁青，气的浑身发抖，心中的怒气无法抑制。
<br/>　　如此不堪的话语，居然从紫霄的口中说出，简直让她不敢相信。
<br/>　　但是，她却是毫无办法，巅峰时候的她，都不是紫霄的对手，现在更加不可能是对方的对手。
<br/>　　“记住，秦陨来找我，不能让任何人发现，可以事前给我传讯！”
<br/>　　幽幽的声音，从远处传来，随即一缕电光激.射而来，落在了绫裳的面前。
<br/>　　电光之中，包裹着一枚特制的传讯符，显然，是紫霄为苏莫准备的。
<br/>　　嘭！
<br/>　　绫裳的玉手，直接捏在了传讯符之上，顿时将传讯符捏成了齑粉。
<br/>　　她不可能收起这枚传讯符，因为这很可能是紫霄的手段，掌控她行踪的手段。
<br/>　　日后，就算真的让苏莫回来，也不会用这枚传讯符。
<br/>　　绫裳的脸色，非常的难看，虽然她已经不是宏青璇，但是心底深处，对于芊寻月还是有感情的。
<br/>　　她不希望芊寻月彻底死亡，更不希望芊寻月死后还要受辱。
<br/>　　“苏莫，你会在哪里呢？”绫裳低声自喃，她也不清楚苏莫会去哪里。
<br/>　　不过，她有宏青璇的记忆，也曾和苏莫同在玄荒星域，同在神路，对于苏莫了解很多，心中也有些头绪。
<br/>　　“先去轮回神朝，再去混沌神朝和天龙神朝！”
<br/>　　少倾之后，绫裳心中有了打算，苏莫显然不可能留在天池了。
<br/>　　那么，苏莫的去处，最有可能的就是混沌神朝和天龙神朝。
<br/>　　随即，绫裳不再耽搁时间，立刻全力赶路，向轮回神朝而去。
<br/>　　……
<br/>　　因终极剑道和终极神剑，掀起的轩然大波，远远没有结束。
<br/>　　不光是终极神王殿在追查苏莫的下落，天池神朝各大界域，无数的大势力，也开始追查苏莫的下落。
<br/>　　毕竟，无论是终极神剑，还是终极剑道，都是至宝，就算是对神王级别的强者，都有莫大的吸引力。
<br/>　　而天池神朝为之震动之时，始作俑者苏莫，经过两个多月的赶路，已经进入了天龙神朝境内。
<br/>　　苏莫来到神域，已经有不少年月了，对于天龙神朝，也有一些简单的了解。
<br/>　　天龙神朝虽然是龙之国度，但是龙族只占一小部分，大部分是妖类种族和异族，以及一些人族。
<br/>　　苏莫再次改变了气息和容貌，进入天龙神朝之后，随便一打听，便打听到了龙族的大本营所在。
<br/>　　天龙神朝的核心，在一个叫做‘龙都’的地方，换言之这是龙之都城。
