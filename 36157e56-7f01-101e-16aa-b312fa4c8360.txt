        现在回想起来，还真是一种缘分。
<br/>
<br/>        他小时候抱过亲过的那个干妹妹，长大后竟然真的嫁给了自己，虽然是被他强娶回来的。
<br/>
<br/>        “真不记得了吗？”童筱颖反问。
<br/>
<br/>        傅夜沉莫名其妙地被她质问得红了耳根。
<br/>
<br/>        童筱颖盯着傅夜沉，索性又朝他凑近了一些，她身上的气息，几乎全喷在了他的俊脸上，周遭的空气，瞬间有些暧昧起来。
<br/>
<br/>        “大哥，需不需要，我帮你回忆一下？”她嬉皮笑脸地说。
<br/>
<br/>        傅夜沉呛得“咳”了一声，故作淡定地挑眉：“你打算怎么帮我回忆？”
<br/>
<br/>        “当然是跟你接吻啊！”童筱颖眉眼间带着温柔的笑意。
<br/>
<br/>        傅夜沉会意地笑了笑。
<br/>
<br/>        他突然站起身，伸手握住她的腰肢，将她从圆凳上搂了起来，紧紧地贴入自己怀中。
<br/>
<br/>        童筱颖只觉自己的胸脯跟这个男人结实的胸膛撞了个满怀。
<br/>
<br/>        “我很喜欢你在我身上点火。”傅夜沉的嗓音变得格外好听，犹如琴弦的低音调，将空气振动出一波又一波的磁场。
<br/>
<br/>        童筱颖微微仰头，怔愣地盯着傅夜沉的薄唇。
<br/>
<br/>        屋内的荧光灯，在他的唇珠上泛起一点白色的星光，唇瓣微扬，如玉般莹润。
<br/>
<br/>        原来，一个男人的唇，也可以这么好看。
<br/>
<br/>        童筱颖踮起脚尖，咬住了傅夜沉的下唇。
<br/>
<br/>        “嘶”地一声，傅夜沉皱了一下剑眉，但是没有推开她，而是任由她咬着不放。
<br/>
<br/>        她的贝齿，只是咬拉着他的唇瓣往外轻翻，却没有咬破。
<br/>
<br/>        “嗯，一点都不好吃！”童筱颖松了口，碎碎念叨。
<br/>
<br/>        傅夜沉忍俊不禁地低头，薄唇微微凑到她的耳畔，轻语呢喃：“我这儿倒是有个好吃的东西，你要不要尝一尝？”
<br/>
<br/>        童筱颖双手抵在他的胸膛上，再次仰起小脑袋，怔忡地看向傅夜沉的黑眸，不禁兴师问罪起来：“你怎么这么坏？还把好吃的东西给藏起来了！”
<br/>
<br/>        “你想吃？”傅夜沉哑然失笑。
<br/>
<br/>        “当然！”童筱颖不假思索地回答。
<br/>
<br/>        她真的是个吃货啊！
<br/>
<br/>        傅夜沉只是笑着，有些无法继续把这个略带彩色的玩笑继续下去了。
<br/>
<br/>        这丫头变得单纯了……
<br/>
<br/>        还让他怎么继续下去？
<br/>
<br/>        “要不，我们去后厨偷吃吧！”傅夜沉不得不转开话题。
<br/>
<br/>        童筱颖倒是十分兴致勃勃：“好啊！这回，我要偷一只烤鸡回来吃！”
<br/>
<br/>        比起吃烤鸡，他更想吃她。
<br/>
<br/>        傅夜沉将这种想法，忍了下去。
<br/>
<br/>        吃完宵夜回来后，他和她仍旧同房不同床。
<br/>
<br/>        第二天，童筱颖一如既往地得陪着冷夫人。
<br/>
<br/>        今早刚从郊区的玻璃花房里送过来的各种鲜花，童筱颖坐在长木桌旁，将这些鲜花修剪分类。
<br/>
<br/>        冷夫人一边拿起童筱颖修剪好的鲜花插在营养泥上，一边话中有话地说道：“你这失忆了，是顺便把做妻子的基本义务也一并给忘了吗？”
<br/>
<br/>        “？！”冷夫人突然说这种话，让童筱颖一脸懵。
<br/>
<br/>        冷夫人瞥了童筱颖一眼，将一支牡丹花插在了营养泥的正中央，接着说道：“给你们整理内务的女佣跟我说，你和小沉是分床睡的。是不是床太小了，不够你两睡？”
<br/>
<br/>        “……”童筱颖被冷夫人说得语塞，一时间一句话也答不上来。
<br/>
<br/>        “怎么不说话了？”冷夫人不满地眯起了眼睛。
<br/>
<br/>        他儿子爱这个女人真的是爱到骨子里去了，现在是连做男人的尊严也没有了吗？
<br/>
<br/>        媳妇到底不是亲生女儿，冷夫人自然是向着自己的儿子。
<br/>
<br/>        童筱颖刚想说是大哥自己要跟她分床的，但话提到了嗓子眼，又被她自己一口气给咽了回去。
<br/>
<br/>        “我……我没有和大哥分床睡啊！”童筱颖强颜欢笑地咧开一个大大的笑容，虽然自己是在撒谎，但也要把谎话说得像是那么一回事。
<br/>
<br/>        “我给你准备了一件睡裙，晚饭后，我让佣人送你房里去，今晚你就穿给小沉看吧！我想我儿子一定会喜欢。”冷夫人面不改色，淡定地拿起另一朵牡丹花插上，“我既然已经不在乎你是否可以生孩子的这件事情，但是你总不能让我儿子当和尚吧？”
<br/>
<br/>        童筱颖听冷夫人说这话时的感觉怪怪的，但又不好反驳。
<br/>
<br/>        结果，真到了晚饭过后，女佣果然捧着一个粉色方形礼盒送到了她的房间里来。
<br/>
<br/>        女佣放下礼盒后，欠身便退了下去，离开前，还替她把房门给带关了。
<br/>
<br/>        童筱颖坐在桌前，好奇地打开了礼盒。
<br/>
<br/>        里面装了满盒的白色羽毛，羽毛上还有一堆黑色丝质的衣物。
<br/>
<br/>        纤纤玉手捏起这衣物后，童筱颖看到的是纯黑的丁字裤和两点式胸衣，以及几近透明的轻纱吊带蕾丝裙。
