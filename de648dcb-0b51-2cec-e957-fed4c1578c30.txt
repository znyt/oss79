　　“星陨阁，这尼玛怎么听的像是什么城中村洗头房的名字。”
<br/>　　林峰无语了。
<br/>　　白衣男子道：“自我介绍一下，我叫罗风，星陨阁成员，我之所以能够察觉到追魂索命蛊这种战门独有手段，是因为我星陨阁之前和战门交过手，当然，那一场战斗是我星陨阁失败了。”
<br/>　　“不过我星陨阁成员对于战门的任何一切都无比清楚。”
<br/>　　“你和我说这干什么？”
<br/>　　林峰皱着眉头道。
<br/>　　“原因很简单，我们是一伙人。有着共同的目的。”罗风道。
<br/>　　“没错啊烤肉小哥哥，古家、张家、还有王家，江城的三大家族，每个家族背后一个组织扶持，王家背后是尸傀门，张家背后是血尸堂，这两个都是是第一邪教的下属势力。”古月道。
<br/>　　罗风叹了口气：“这个第一邪教太强大了，末世前他们就是遍布全国，和传销一样，末世后，更是一发不可收拾，非常恐怖。”
<br/>　　“这第一邪教这么强？难道比战门还厉害？”林峰皱着眉头问道。
<br/>　　“战门在江城这个地方算是最强的组织了，他们战门门主是个异能者，他的异能者是图书馆异能者。”罗风道。
<br/>　　“图书馆异能者？这尼玛什么异能，难道变出一大堆黄色小说出来给自己的人看？”林峰道。
<br/>　　“烤肉小哥哥你好色啊，整天想着黄色小说。”古月笑道。
<br/>　　罗风摇了摇头，“图书馆异能，我也是听我老大说的，这个异能很厉害，你应该遇到过战门成员了吧，知道他们战门成员的手段了吧，实际上他们那都不是异能，那是叫做战技的东西，这些战技，就是他们老大用异能得到的。”
<br/>　　林峰顿时想起了之前第一个交手的战门外门成员，那什么迷幻步什么刀上喷火，当时林峰还以为这家伙两个异能。
<br/>　　如今看来，似乎这不是异能，而是两种不同的招式。
<br/>　　“你的意思是，这些战技，都是书？是书，就可以从这个图书馆异能之中得到，换句话说，他们战门的老大就是个大辅助，只要是他的部下，哪怕不是异能者，他也可以把他们打造成堪比异能者的存在？”
<br/>　　林峰震惊了，这个异能很牛逼啊，和他的这个空投都有得比了。当然了，实际上还是差了不少的，他的空投什么都可以捡到，日后搞不好，神都可以捡到，算是个超BUG级别的异能。
<br/>　　这个图书馆还算是情理之中，勉强可以算个BUG级别的异能。
<br/>　　“的确如此，据我所知，他的图书馆异能可以拿出许多战技，甚至一些功法，算是很强的存在了。不过当然了，他也是有很多限制的。不管怎么说，江城第一的民间组织就是战门，但他也比不过第一邪教，第一邪教是全国范围的，到处都有他的人。而他们的目的，就是要灭掉所有人，制造新人类，欢迎他们心中的神降临地球。最不好的消息是，我听说战门似乎开始和第一邪教的人合作了。”罗风道。
<br/>　　“这么说来，这王家和张家都是坏的，你们古家还真的是好的。”林峰道。
<br/>　　“烤肉帅哥，你这样相信我了吧。。”
<br/>　　古月委屈巴巴道。
<br/>　　“我看你穿得这么骚，我就以为是坏人。”
<br/>　　“额，摸摸屁股，不哭啊。”林峰捏了把古月大屁股。肉嘟嘟的非常有弹性。
<br/>　　古月顿时闹了个大红脸。
<br/>　　“烤肉小哥哥，你太坏了。”
<br/>　　“咳咳。”
<br/>　　罗风在旁边咳嗽了一下。
<br/>　　“那个，你叫什么。”
<br/>　　“林峰。”林峰道。
<br/>　　“林峰，我可以帮你去除掉你身上的追魂索命蛊，也算是一个见面礼。”罗风说道。
<br/>　　“当然，林峰我需要你帮我个忙。”罗风道，“你不必担心，你的实力足够应付，只要小心，不会有什么危险。”
<br/>　　“没问题，你先帮我弄掉追魂索命蛊再说。”林峰道。
<br/>　　“嗯。”
<br/>　　罗风道玩就直接念了段口诀，然后就是忽然朝林峰眉宇之间一点。
<br/>　　顿时林峰就感觉脖子上什么东西没了。
<br/>　　“已经祛除了。现在你可以跟我来了。”罗风说道。
<br/>　　“还真是够神奇的。”林峰摸了摸脖子。
<br/>　　“这是我的异能，可以清楚一些诅咒之类的东西，也就龙虎山天师门的人会类似的秘法了。”罗风道。
<br/>　　“龙虎山？莫非真的有道士？”林峰道。
<br/>　　“自然，这些都是末世前就已经修炼得到的道士了，会画符，会降魔，会捉鬼，不过这类道士很少。”罗风道。
<br/>　　“林峰，跟我去个地方，我需要你的鼎力相助。”
<br/>　　罗风说完就离去了他的速度很快就好像会飞一样。
<br/>　　来到罗风所说的地方。
<br/>　　这里荒郊野岭，什么都没有。
<br/>　　古月也跟了过来好奇的看着四周。
<br/>　　“地上什么没有，但地下却不一定。”
<br/>　　罗风来到一个地方，一脚把一个小土坡给踢到了，里面居然是一面旗帜，把这个旗子拔出来后，顿时整个大地上出现一阵阵黑气朝泥土外面喷射。
<br/>　　“这个地方，是血尸堂的炼尸之地，血尸堂的很多人，都是得了一种罕见血液怪病，必须要喝人血，因为这种怪病，他们在末世前都是被社会遗弃的对象，这群人私底下走得很近，也就成为了现在的血尸堂，这群人都是被社会伤害甚至遗弃过的人，所以也算上报复社会的心理变态吧。”
<br/>　　“他们不知道杀了多少幸存者埋在这个地方，我也是一次意外撞见，既然遇见了，那我就不能不管，必须要顺手铲除这个毒瘤。不然只会有更多的幸存者无辜遭受屠杀。这群幸存者都好不容易活下来，没死在丧尸嘴巴里，却死在了这群家伙手中。”罗风认真道。
<br/>　　“炼尸之地？这血尸堂莫非拿活人炼制尸体？”林峰震惊了，这就太过分了，简直惨无人道。
<br/>　　“没错，确切的说，这是他们的异能，他们异能就是可以把活人炼制成尸体，那个尸傀门也是一样。”罗风道。
<br/>　　“还真是够阴毒残忍的异能。”林峰道。
<br/>　　“是谁敢来我血尸堂地盘闹事！”
<br/>　　刷刷。
<br/>　　忽然不知道从哪里奔出了两个白衣青年，这两人也都不过二十七八岁上下罢了，但一人带着一个尸体，就好像是个保镖一样。
<br/>　　罗风正要开口说话，林峰却阻止了。
<br/>　　“是你战门的爷爷在此！”
<br/>　　罗风有些发愣的看着林峰似乎不明白林峰为什么要扯上战门。
<br/>　　“这一招我用了好几次了，反正战门厉害，让他多背背黑锅。”林峰无赖的说道。
