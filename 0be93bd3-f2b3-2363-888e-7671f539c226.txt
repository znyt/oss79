        看着大家的热情，童筱颖忽然觉得这世态炎凉果然是冷暖自知。
<br/>
<br/>        先前，她只是一个在校未毕业的实习生时，无依无靠，没有背景，到哪里都好像低人一等，工资不高，干的活却是最多的。
<br/>
<br/>        如今，她仍旧是一个在校未毕业的实习生，但因为多了一层大Boss妹妹的身份，所以，谁看她都顺眼，工资高，干的还是轻松的活。
<br/>
<br/>        可是，她并不只是满足于现状。
<br/>
<br/>        童筱颖不想仅仅只做设计图的审稿工作，还想自己亲自设计，于是主动跟部长说出了自己的意愿。
<br/>
<br/>        她想用自己的实力，让大家信服她。
<br/>
<br/>        傅氏集团总部里的设计部相对于地利置业的设计部来说，所有的活都比较轻松，因为，初稿设计都是由地利置业那边的设计部里的设计师完成，终稿完善都在总部这边。
<br/>
<br/>        部长是一个年过四旬，看起来很有经验的青年男人。他的鹰钩鼻上架着银框眼镜，西装革履，深红色领带，穿戴整齐。
<br/>
<br/>        童筱颖就站在部长的办公桌前，一脸认真的样子，让这位男部长刮目相看。
<br/>
<br/>        “那你把这初稿设计拿去深度完善，三天的时间，能做到吗？”男部长推了推鼻梁上的眼睛，随手从右边堆积如山的文件中抽了一份出来，递向童筱颖。
<br/>
<br/>        童筱颖双手接过文件，微笑着点了点头。
<br/>
<br/>        “那你去忙吧！”男部长礼貌性地点了点头。
<br/>
<br/>        童筱颖微微颔首，抱着文件，高高兴兴地回到了自己的办公桌前。
<br/>
<br/>        她打开了文件袋，从里拿出初稿设计图看了看，不禁会心一笑。
<br/>
<br/>        果然和她猜想的一点都没错，部长交给她的单子一点都不难。
<br/>
<br/>        忽然间，童筱颖又想起了地利置业设计部的副部长孙莉莉。
<br/>
<br/>        记得当初，副部长孙莉莉一开始就为难她……
<br/>
<br/>        算了……
<br/>
<br/>        童筱颖心底虽然在黯然神伤，但如今总算是雨过天晴，她得把眼前手下的事情做好才行。
<br/>
<br/>        中午，办公室里的下班铃声准时响起。
<br/>
<br/>        童筱颖听到这个铃声时，差点误以为自己是坐在高中的教室里。
<br/>
<br/>        当她从电脑后抬起头来时，同在一个集中办公室里的同事，一个个纷纷起身，走去大门口打卡下班。
<br/>
<br/>        “筱颖，你是回家吃午饭，还是和我们一起去楼下员工餐厅一起吃？”坐在童筱颖对面办公桌的女同事齐灵云从她的办公桌前站起身来问道。
<br/>
<br/>        童筱颖这才注意她，并放下手中的活，看向齐灵云微微一笑：“和你们一起去吃。”
<br/>
<br/>        “那我们走吧！”齐灵云微笑着说。
<br/>
<br/>        她笑起来的样子，两只眼睛，就像两弯可爱又动人的月牙，令人印象深刻。
<br/>
<br/>        童筱颖一下子就记住了齐灵云的模样，并下意识地看了一眼她胸前的工作牌，才得知她的名字叫“齐灵云”。
<br/>
<br/>        两人一番交谈之后，童筱颖改了口，管齐灵云叫“云姐。”
<br/>
<br/>        因为，她比她大了五岁，齐灵云今年二十三岁了，不过来这设计部工作才一年的时间。
<br/>
<br/>        齐灵云单身，没交男朋友，所以，她家里人一直在给她物色对象，但是她都没看得上。
<br/>
<br/>        童筱颖跟着齐灵云和设计部里的其他女同事一起，打了饭，坐在了一张大圆桌前。
<br/>
<br/>        大家一边吃，一边八卦，聊天聊地，就没有她们不聊的话题。
<br/>
<br/>        童筱颖感觉自己插不上话，只好默默地吃饭，静静地聆听她们谈笑风生。
<br/>
<br/>        本来，这话题不会落在她头上，但是不知道是谁先带头，主动向她童筱颖打听起傅夜沉来。
<br/>
<br/>        “筱颖，你哥哥，也就是我们的傅大Boss有没有交女朋友啊？”
<br/>
<br/>        “傅大Boss他平常都喜欢做些什么？”
<br/>
<br/>        “傅大Boss的家住哪儿？”
<br/>
<br/>        “傅大Boss的妈妈好不好相处？”
<br/>
<br/>        ……
<br/>
<br/>        还不过一分钟的时间，围着这一桌的女人，对傅夜沉的八卦兴趣，已经到了登峰造极的地步。
<br/>
<br/>        而童筱颖只回答了一个问题，那就是：“我大哥他已经有未婚妻了……”
<br/>
<br/>        一瞬间，大家全都安静了。
<br/>
<br/>        坐在童筱颖身旁的齐灵云忍俊不禁起来，刚刚也就只有她一个人，没有拉着童筱颖问傅夜沉的事情。
<br/>
<br/>        齐灵云的淡定，反倒让一旁的女同事们对她不约而同的起哄。
<br/>
<br/>        “灵云啊！听说跟你相亲的对象，你一个也看不上。倘若，换做是咱们傅大Boss那种，要钱有钱要貌有貌的男人，你那岂不是没得挑了！”
<br/>
<br/>        “别说灵云，如果我的相亲对象是这种高富帅型的男人，我立马就嫁了！”
<br/>
<br/>        “诶诶诶！我也是耶！我也想嫁给高富帅的男人！”
<br/>
<br/>        “有钱长得帅的男人，那还得品格好会宠女人，不然，嫁过去活受罪！”齐灵云见这群女人七嘴八舌，忍不住打岔。
<br/>
<br/>        “诶诶诶！你们是不知道，我有一发小，她就嫁了一个有钱又长得帅的男人。但是，那男人忒大男子主义，说什么会养我发小一辈子，我发小都不用出去工作的。结果，你们猜怎么着？我那发小的有钱帅老公出轨了！出轨的对象，还是她老公的初恋情人。我发小因为一直没参加过工作，没有经济来源，都不敢跟她老公离婚。只能活受那小三对她翻白眼的罪！”
<br/>
<br/>        “这么悲催啊！”
<br/>
<br/>        “真作孽啊！”
<br/>
<br/>        听到女同事们忽然聊到这个话题上，童筱颖心里莫名其妙地联想到了傅昼景。
<br/>
<br/>        傅昼景不仅有钱而且还长得帅，那么……他会不会也有一个她不知道的什么初恋情人吗？
<br/>
<br/>        “这男人啊！心里的白月光永远都是初恋！”
<br/>
<br/>        “不一定吧！我就不是我老公的初恋，但我老公现在非常爱我！”
<br/>
<br/>        “你老公爱你，那是你‘又作又骚’，懂如何抓住老公的心，还懂如何经营自己的婚姻罢了！”
<br/>
<br/>        大家各持己见。
<br/>
<br/>        童筱颖沉默在自己的臆想里。
<br/>
<br/>        如果，哪天，傅昼景的初恋情人突然冒出来，跟她抢傅昼景，她到时候该怎么办？
<br/>
<br/>        算了算了！不要再去想这些乱七八糟的事情了！
<br/>
<br/>        她只希望这样的事情，永远也不要发生在自己的身上！！
<br/>
<br/>        然而，午饭过后，童筱颖却有点儿鬼使神差般掏出手机，独自一个人躲在了安全通道里，给傅昼景打了通电话。
<br/>
<br/>        电话响了许久，傅昼景才接听。
<br/>
<br/>        听筒那边很吵，有男人高声阔谈的声音，也有女人盈盈嬉笑的声音，童筱颖听得出，他是在应酬，还在陪客人吃午饭。
<br/>
<br/>        “我现在有点忙，若是没什么事的话，我就挂电话了。”傅昼景的声音听起来很淡，而且还有点儿不耐烦。
<br/>
<br/>        童筱颖抿了抿唇，一时间不知道该说什么好，只能沉默不语。
<br/>
<br/>        “筱颖，你若是不说话，我就把电话挂了！”傅昼景接着说。
<br/>
<br/>        童筱颖急忙开口，心直口快地说出了自己的真心话：“昼景，我想你了。”
<br/>
<br/>        “嗯，我在外忙完这段时间，就回来陪你。”傅昼景的口气稍稍缓和了些。
<br/>
<br/>        童筱颖强颜欢笑地应了声：“好！我等……”你回来！
<br/>
<br/>        “嘟——”
<br/>
<br/>        她的话，还未说完，他便把电话给挂了。
<br/>
<br/>        此时此刻，童筱颖的心里就像打翻了五味瓶，很不是滋味。
<br/>
<br/>        “原来你在这儿啊！我找你很久了，还以为你迷路了！”身后忽然传来齐灵云的声音。
<br/>
<br/>        童筱颖身心一怔，连忙换了换心情，转过身去时，已经面带微笑着看着齐灵云。
<br/>
<br/>        “外边太吵，我来这儿，给家里人打个电话。”童筱颖温和地解释道。
<br/>
<br/>        齐灵云倒是一个心思敏感，会察言观色的女人，见童筱颖眼睛都红了一圈，咧嘴打趣：“我看你是跟你男朋友吵架了吧！”
