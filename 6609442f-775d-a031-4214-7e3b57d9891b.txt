　　对这个消息，就算是很多业内的人士也都在那里讨论，甚至感觉有些不可思议。

<br/>　　“都支持自家的战队这也正常么，等到比赛结束就好了，主要是双方的实力都太强劲了，双方的粉丝都想让自己的战队获胜，就算是关系再好，这个时候也要分的清啊。”

<br/>　　而在论坛上，越是临近比赛开赛的日子，双方的粉丝吵的也越发的热烈了起来，到了这个时候恐怕就算是叶帝他们出面也无法阻止这一情况。
<br/>　　这也是没办法的事情，如果说双方有一方的实力能够碾压对方，那么恐怕也不会出现这样的局面，不过谁让他们两边实力相当，而且粉丝群体都巨大呢。
<br/>　　“有大魔王在我们白队是绝对不会输的。”
<br/>　　“大魔王有那个实力我确实承认，不过下路的萝卜和茄子两人肯定是要被霄神他们制裁的，我们这边可是双人路，优势会更大的。”
<br/>　　“野区我们的夜雪也能够制裁你们的飞刀，说到底还是我们会赢的，大魔王和夜雪的中野联动，天下无敌。”
<br/>　　“2333333333333，笑死了，虽然暗夜逗逼，但是你还真的把他当做吃素的了，大魔王就算是实力强，想要在线上单杀暗夜那也是十分困难的吧。”
<br/>　　.......
<br/>　　而就在这种激烈的争吵当中，周日这一天终于是缓缓的到来。
<br/>　　这一天荣耀蓝白两支战队的比赛会作为压轴的比赛登场，这也是火神处于种种考虑安排的。
<br/>　　要不然荣耀蓝白两支战队比赛打完了，现场这些粉丝都撤退了，你这让后面比赛的那些人多尴尬，所以这也是最好的选择了。
<br/>　　而在1点钟开始比赛的OG和RG这两支战队，此刻一个个也都是跟着紧张了起来。
<br/>　　在这么多蓝白粉丝的关注下，他们真的是一点都不想在场上多待，如果能够早点结束，那自然是最好的，这种没有自家粉丝支持的比赛，才是最尴尬的。
<br/>　　当时间到达下午2点45分的时候，叶帝和蒋念他们终于是赶到了鸿雁体育馆，停好车子后，在叶帝的带领下，一身黑色队服的白队便是率先向着场内走去。
<br/>　　而在叶帝他们刚刚走进比赛场地的一刹那，那些一直在关注着后方入口处的粉丝便是发现了他们的踪迹，刹那间比赛场左边传来一阵震耳欲聋的欢呼声。
<br/>　　仔细一看这些粉丝穿的居然和叶帝他们的队服都是一样的衣服，这样的群体直接将体育馆的左半边全部给霸占了，而反观又半边，则是被一群身穿着白色队服的蓝队粉丝群体霸占着。
<br/>　　在叶帝他们进入比赛场地的时候，只有左边的人在欢呼，而右边的人则是寂静无声。
<br/>　　叶帝几人朝着左侧的粉丝们挥手过后，便是向着后台走去。
<br/>　　片刻后，身穿一些白色队服的蓝队队员，在秦霄的带领下终于是走进了比赛室，同一刻一股丝毫不弱于刚刚的呼喊声从右侧响了起来。
<br/>　　两次呼喊在隔音室内的OG战队和RG战队的队员都是清晰可闻。
<br/>　　“叶哥他们进场了吧，这欢呼声也忒大了，什么时候我要是能有这么大的人气就好了。”小西一脸艳羡的说着。
<br/>　　“还是敢接结束比赛吧，咱们今天可不是主角，估计咱们多待一分钟，那些粉丝都要将咱们给活剐了。”RG战队的ad跟着开口说道。
