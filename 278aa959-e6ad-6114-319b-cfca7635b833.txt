　　　　涂小安骑着虺蛟直接落到了陈家大院。
<br/>　　此时整个城市都是静悄悄的，真的要说时间，恐怕都凌晨二三点了。
<br/>　　落地后，虺蛟直接化为了张天师的模样，脸上表情很不好看，但是也不好发作。
<br/>　　没一会儿，一间房间的门开了，猫精海棠款款而来，恭尊的走到涂小安的面前：“恩公，原来是你回来了”
<br/>　　涂小安笑了笑：“不好意思，把你吵醒了”
<br/>　　“恩公那里话”
<br/>　　说着海棠打量了下旁边的那位身穿道袍的男子，这一打量，顿时娇美的面容上诚惶诚恐起来：“化形的三阶妖灵？”
<br/>　　张天师看到海棠，也是一脸的吃惊，这猫精显然还没有证道，但是为何能是人形。
<br/>　　涂小安这条没证道的蛇精如此便罢，谁让他变态，但区区猫精凭什么也可以化为人形。
<br/>　　要是都那么好化形，他修炼了五百年，都修炼到狗身上了？
<br/>　　天色已晚，涂小安简单的将这虺蛟跟猫精的身份介绍了一下，就各自回房休息。
<br/>　　有什么明天在说。
<br/>　　次日。
<br/>　　涂小安起床，想着明天就要陪着山野集团的大胖儿子去古学院了，今日就带着人妖恋情侣跟张天师出去吃一顿好的。
<br/>　　因为他将张天师也安排在了陈家大院，所以有点不好意思。
<br/>　　海棠由于知道了张天师的虺蛟身份，有点害怕他，之后又要相处在一个屋檐下，有点胆战心惊。
<br/>　　所以，涂小安趁着自己还在，帮着化解一下尴尬。
<br/>　　另外涂小安也是心情好，这趟出去两天，收获太大，不但多了一种元素能力，还得到了一条三阶虺蛟，值得好好庆祝一下。
<br/>　　古街上，四人同行，涂小安看到味鲜阁再次开业了，就选择了这家吃。
<br/>　　里面客源不少，看来苏羽丢了龙蛇，也并什么多少影响店里的生意。
<br/>　　进店之后，迎面而来一位漂亮的迎宾，涂小安本想要一个包厢，但是迎宾说今天客人太多，包厢满了。
<br/>　　只能选择在大厅中就餐。
<br/>　　张天道如今又是一幅仙风道骨的样子，来到这个味鲜阁眼睛四处看，喃喃自语的说：“这个店的风水不错，可惜出现了衰退的迹象，这个店的生意也好不了多久了”
<br/>　　涂小安道：“之前这个店有一条龙蛇镇店，可惜后来被人偷走了”
<br/>　　“天师，我很好奇，你一条虺蛟怎么做起了道人”
<br/>　　他一条虺蛟修的哪门子道呀。
<br/>　　张天师却道：“进入这红尘人世中，自然要有一技之长，本天师还想着城里买房呢”
<br/>　　“当神棍好赚钱？”
<br/>　　涂小安收服虺蛟，自然看的出它不是为非作歹的妖人，没想到他能如此的融入人类世界。
<br/>　　还想跟普通人一样的买房，还真的让他有点诧异。
<br/>　　莫非他想在城中买房，将平安村的村花跟孩子接过来住。
<br/>　　或许也是想告诉村花小凤，他虽然不是人类，但是化形后，也可以跟人类一般无二。
<br/>　　但可惜，村花跟他的孩子已经没了。
<br/>　　坐到一个大厅上的桌上，菜单在前，涂小安直接拿起来看。
<br/>　　五步蛇肉，知了，野猪肉，竹林野鸡，皇家礼炮，穿山甲。
<br/>　　这是最新的野味菜单。
<br/>　　看来苏羽为了店里的生意也是煞费苦心，除了蛇肉，弄了不少别的野味，怪不得生意暂时还不见影响。
<br/>　　“安哥，顿时让小弟来请吧”带着墨镜的陈高阳忽然开口。
<br/>　　他一直找不到感谢涂小安的机会。
<br/>　　“这地方我找来的，自然我请，下回你来”涂小安笑了笑。
<br/>　　张天师也看了看菜单，：“这店居然还有穿山甲可吃呀，不错，来只尝尝”
<br/>　　“你现在是道人，注意身份”涂小安白了一眼。
<br/>　　对于穿山甲这种动物，涂小安是没真正见过的，那是既熟悉也非常陌生，平日很难见到。
<br/>　　第一次听说穿山甲，还是在动画片《葫芦娃》里，善良的穿山甲，为了帮助老爷爷逃跑，惨死在妖精手下，记得小时候看到这一幕还哭的稀里哗啦。
<br/>　　长大以后，还真没有机会亲眼见到穿山甲。
<br/>　　“道人曾经吃过？”涂小安看着他。
<br/>　　张天师点了点头：“曾经平安村一带，有不少穿山甲，后来渐渐的被你们这些该死的人类给抓光了”
<br/>　　穿山甲现在为什么不可见，就是因为快被人类给抓光吃光。
<br/>　　几乎快成了一种被人类吃的灭绝的动物。
<br/>　　在古代，穿山甲黑甲披身，鳞角四足，足五爪，色青黑，长可五尺，是被视为龙一般的奇珍异兽。
<br/>　　遗憾的是，如今的穿山甲，却因数量骤减被列为国家二级保护动物。
<br/>　　没想到这味鲜阁那么大胆，直接保护动物，自然放到了菜单上，怪不得店里生意那么多。
<br/>　　敢情杀死穿山甲的不是妖精，而是人类。
<br/>　　“道长想吃，那便来一只”
<br/>　　涂小安现在连人都吃了，更别说这些了，他现在是百无禁忌。
<br/>　　他叫了迎宾点菜，点到穿山甲，对方却道：“尊敬的客人，不好意思，今天店里最后一只穿山甲，被你旁边的那桌客人点走了”
<br/>　　旁边那桌是六七个样貌堂堂的年轻人，看着打扮，应该是家世不凡，都是有一定的身份。
<br/>　　“明天就要去学院报道了，今天我们要好好的吃一顿，我点了一只穿山甲，各位好好尝尝，味道不错”
<br/>　　“杨兄客气了，山珍海味吃了不少，但这穿山甲，我等还真没吃过”
<br/>　　旋即，一位穿着厨师小伙端了一个水盆就来到了这桌，然后又在桌边架了一个火盆。
<br/>　　只见水盆中，一只身披铠甲，体态修长的动物蜷局在水盆里，顿时吸引了不少客人。
<br/>　　水盆中的动物样子另类，看着有点像特殊品种的毒蛇，但它的头部膨大，和蛇又有不少差异。
<br/>　　“各位，这是本店最后一条长尾穿山甲，属于这桌客人了”
<br/>　　穿山甲，现代人很多都没见过，这只长尾穿山甲跟毒蛇还真的有几分的神似，体态修长，全身布满铠甲鳞片。
<br/>　　都说穿山甲身披铠甲，擅长穿山凿洞，但其实穿山甲并不会穿山，虽然为了捕食猎物——蚂蚁，穿山甲会在一些松软湿润的土层上挖洞，可也就仅限于此，凿穿一座山这样的事它是做不到的，甚至它的挖洞能力还不如普通的兔子。
<br/>(本章完)
