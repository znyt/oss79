        &lt;h3&gt;第三卷第一百九十八章  探望钱喵喵&lt;/h3&gt;
<br/>
<br/>        陈子昂送走了张函一家人，收拾好心情，陪着俩个孩子在家里安安宁宁的呆了两天，孩子在她的细心照料下，感冒也好了，财叔和春姨已经在陈家呆习惯了，回去和自己孩子住了几天，就回到了北京。
<br/>
<br/>        财叔和春姨一回来，陈子昂就感觉轻松了很多，这都多少年了，俩个老人无怨无悔的帮着她操持着家里的事情，包括孩子的学习，她都没有怎么去管，只是一门心思的忙着她的事业，如果不是这两个老人那里有她陈子昂的今天呢。
<br/>
<br/>        陈子昂见财叔和春姨回来，满心欢喜，晚上一家五口就坐在客厅聊开了。
<br/>
<br/>        嘟嘟最是嘴快话多：“爷爷，你知不知道我们家过年的时候来人了？”
<br/>
<br/>        财叔：“你爸爸来了？”
<br/>
<br/>        嘟嘟：“爷爷，你能不能不要那么聪明啊？”
<br/>
<br/>        财叔：“那爷爷就这么聪明有什么办法呢？”
<br/>
<br/>        春姨：“嘟嘟，你爷爷在年前见过你爸爸呢，知道你爸爸要来。”
<br/>
<br/>        团团：“爷爷，你知道爸爸要来为什么不提前告诉我们？”
<br/>
<br/>        财叔：“当时不确定他要不要来啊。”
<br/>
<br/>        陈子昂：“他们一家除了陆玉没有来，其余的三个人都来了。”
<br/>
<br/>        财叔：“陆玉在甸城，一时半会儿不会回来的。子昂啊，我回去上海的时候见了你落伯伯，你要将孩子送到国外，就送到欧洲吧，我到时候和你春姨一起过去，十几年前我也在欧洲呆过，也算是比较熟悉。”
<br/>
<br/>        陈子昂：“你和落伯伯说过了？”
<br/>
<br/>        财叔：“是啊，你落白哥哥也同意你先将孩子送走，不过这不是长远之计，瞒得了一时怎么可能瞒得了一辈子？”
<br/>
<br/>        嘟嘟：“为什么要瞒呢，我们不是见过爸爸了吗？那边的爷爷奶奶也见过了啊，谁规定爸爸和妈妈不在一起，我们就要偷偷摸摸的了？”
<br/>
<br/>        春姨：“不是要瞒什么，只是让嘟嘟和团团去国外更好的地方生活。”
<br/>
<br/>        团团：“没有妈妈的地方怎么可能是好地方？”
<br/>
<br/>        团团已经听出来了，爷爷和奶奶要带着他们去国外生活，而且还是借住在落爷爷家的庄园里，心里莫名的失落和难过。
<br/>
<br/>        三个大人都沉默了，是啊，没有妈妈的地方怎么可能是好地方呢？
<br/>
<br/>        良久，陈子昂搂着团团和嘟嘟：“你们先过去啊，我等这边的工作做得差不多了，我就过去陪你们，妈妈的耗材是要卖到全世界呢，你李凌菲阿姨和倪小美伯伯可是专门负责国外市场的，团团和嘟嘟先过去，给我们先将市场占领了，后期我们过去就一炮打响，你们说好不好？”
<br/>
<br/>        团团：“我知道你忙，我也知道你是为了要保护我们，但是妈妈，如果你不告诉我们真相，我们什么都不知道那才叫危险呢，我和妹妹知道真相了，最起码我们会有防范意识啊。”
<br/>
<br/>        陈子昂看着团团，孩子的眉眼青涩，透露着一股子坚毅和倔强，似乎他在她的眼里一瞬间长大了，可是她该如何给孩子讲这世道艰难？
<br/>
<br/>        财叔：“团团你的想法真不错，是个小男子汉了，爷爷告诉你，你爸爸的老婆家里很强大，他们是不会允许你们存在的，如果你们在国内会有危险，妈妈不可能时时刻刻在你们身边，所以我们暂时去别的地方，等你妈妈将这里的事情安顿好之后就会和我们在一起的。”
<br/>
<br/>        嘟嘟：“光天化日之下他们能干什么？”
<br/>
<br/>        春姨：“光天化日之下他们不能干什么，可是光天化日之后，我们也不知道他们能干什么，嘟嘟你和哥哥还小，还没有到能保护自己的地步，和爷爷奶奶一起去别的地方，等你们长大了就可以保护妈妈了。”
<br/>
<br/>        其实他们要担心的不但是陆家，还有媒体，唾沫星子能淹死人，陈子昂这几年的形象一直都是阳光的单身女人，她现在也是一个公众人物。
<br/>
<br/>        嘟嘟哭唧唧：“可是我不想离开妈妈。”
<br/>
<br/>        团团：“妈妈，你放心，团团一定快快长大，长大了可以保护妈妈和妹妹。嘟嘟，我们去欧洲庄园可以见到雪儿阿姨呢。”
<br/>
<br/>        嘟嘟：“好吧，陈虞，你要记得，我是你妹妹，你得保护我。”
<br/>
<br/>        团团：“你个小憨憨，我是你哥哥，我不保护你，我保护谁啊？”
<br/>
<br/>        俩个孩子很听话，只是让陈子昂的心里更是揪心和痛苦，她以为她的努力会给孩子和自己带来幸福带来安宁，可是她还是把现实想得太好了，张函和财叔都担心他们母子，她也不得不面对现实，好好思考一下后期该怎么应对。
<br/>
<br/>        “子昂，那就这样决定了，过一段时间你忙完了，我们就去欧洲，你也可以去休息两天，放松一下。”
<br/>
<br/>        “好的，我也是想去那边看看，看看雪儿呆了几年的地方。财叔，你们回来了，我明天就去公司看看值班的人。原本想着过年的那一天和大家一起吃顿年夜饭，可是张家来人了。”
<br/>
<br/>        “行，让你春姨做些吃的，我明天送你过去。”
<br/>
<br/>        .......
<br/>
<br/>        这个新年钱喵喵在公司值班，他们公司经理以上的人员都会有一套两居室的房子，所以这个新年他就将自己的父母也接到了北京，一家人在一起不孤单。
<br/>
<br/>        父亲和母亲都有工作，平时也没有时间来北京，刚好他也想父母亲过来看看，等上班了之后他也是可以休几天假的，到时候带着父母亲去转转，偌大的厂房就他们十来个人值守，责任重大，值班这几天是哪里都不能去。
<br/>
<br/>        钱喵喵正坐在房间里打着游戏，有人敲门，打开门，陈子昂站在门口：“陈总，你怎么来了？”
<br/>
<br/>        “来看看叔叔阿姨和你啊，怎么不欢迎吗？”
<br/>
<br/>        “欢迎欢迎，赶快进来。”钱喵喵打开门，将陈子昂和财叔迎进了家。
<br/>
<br/>        “叔叔阿姨，您们来这里还习惯吗？”陈子昂看着财叔将手中的礼盒放下，然后和财叔一起坐下和钱喵喵的父母亲聊了起来。
<br/>
<br/>        “习惯，习惯，喵喵在这里能得到陈总赏识，看着他现在的样子我们也很安心。”
<br/>
<br/>        “叔叔和阿姨都是老师，教育出来喵喵这么有责任心的孩子，是我们公司的幸运呢。”陈子昂感慨的说。
<br/>
<br/>        钱爸爸：“呵呵呵，还是陈总领导有方啊，给了他用武之地。”
<br/>
<br/>        “叔叔阿姨谦虚了，家庭教育是第一位。”
<br/>
<br/>        “哈哈哈，你这个说法我赞同，别看现在兴起了很多的补课班，孩子的素质教育和思想教育还是得靠自己的家里。”
<br/>
<br/>        “确实，最近兴起的补课班很多，叔叔和阿姨有没有退休后办个补课班？”
<br/>
<br/>        “如果有这个精力我倒是愿意，只是怕我们年龄大了，干不动。除非喵喵能回来帮忙。”
<br/>
<br/>        “想一想就行了，退休了就好好享受生活，办什么补习班呢，你们不嫌累我都看着累。”钱喵喵是不赞同他的父母亲退休之后再工作。
<br/>
<br/>        陈子昂还要去探望公司其他的人员，便告辞了钱家三口，和财叔一起走了。
<br/>
<br/>        钱喵喵没有想到这一次的谈话，给他父母留下了深刻的印象，几年之后，他放弃了自己在陈子昂公司的待遇，回去和父母亲一起办了个培训班。
<br/>
<br/>        “喵喵，你们这个老板很是平易近人啊。”钱爸爸看着财叔拿过来的礼品。
<br/>
<br/>        “是啊，你看，还有做好的熟食，太贴心了。咦，这里面还有红包，这红包也太大了吧。”钱妈妈翻看着礼品，将熟食放进冰箱。
<br/>
<br/>        “红包是给你们俩人的，我们的福利年前都发给我了。”钱喵喵并没有看到有红包有多大，他在公司干到这个份上，一年工资也不少拿，这个红包肯定是陈子昂给他爸爸妈妈的。
<br/>
<br/>        陈子昂对员工一视同仁，和他们在一起似乎没有上下级的那种威严，但是他们每个人却从心底尊敬她，认可她，愿意和她一起奋斗，他们的中层以上都是和她同甘共苦过来的，对于她的行为，大家已经是习惯了的。
