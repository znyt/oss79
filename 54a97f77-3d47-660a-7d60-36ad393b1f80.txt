
<br/>　　<br/>　　    水里的人被王惊蛰喊得一懵，迷茫的仰着脑袋往船上看，有点不知所措了。<br/>　　<br/>　　    黄詹讶然的问道：“怎么了？”<br/>　　<br/>　　    王惊蛰在水里那人身上来回又瞟了几眼后，低头说到：“黄教授，你要让我说为什么，我给你解释了，你肯定问我有什么科学依据，但依据我肯定说不出来，我只能用一句我对他下水的感觉不太好来解释了”<br/>　　<br/>　　    “胡闹！”张生利在旁边背着手呵斥了一句。<br/>　　<br/>　　    王惊蛰冷冷的瞥着他，直接扭头跟黄九郎说道：“你之前为啥同意我领导，你自己心里有数，话我放这了，听不听就不关我的事了”<br/>　　<br/>　　    “嗯”黄九郎点了点头，跟黄詹说道：“黄老，惊蛰的话我觉得你最好慎重考虑下，不行就让他上来，换个人下去？”<br/>　　<br/>　　    黄詹当即摇头说道：“让他上来，那我们所有的方案和计划就得重新来做了，最少两三天内又不能下湖了，而再过几年抚仙湖地区天气多变，一连阴雨好几天，这一耽搁搞不好就得要十天半个月了，影响太大，除非有什么不得已的因素，否则很难换人……”<br/>　　<br/>　　    之前，黄詹的学生还有研究所两个团队制定的计划是，分两批人下水，一批六个人，第一批下水的人负责外围勘测每个人都负责一个方位和角度，各个角色都有自己的区域任务，第二批的人则是直接进入古城内部。<br/>　　<br/>　　    所以两批人训练和计划的目标都是不同的，有针对性的，这样每批人负责一部分，能效率最大化，如果这时候突然把人换掉的话，这和临阵换将也差不多了，肯定会把计划全都给打乱的。<br/>　　<br/>　　    黄詹有疑虑，张生利不同意，那眼镜青年就冷言冷语的瞅着王惊蛰说道：“你乱指挥什么呢？我们的计划都已经制定一个星期了，团队里每个人负责的任务都不一样，现在突然被你一句话叫着要换人，以后出麻烦和疏漏你负责啊，扯淡！”<br/>　　<br/>　　    “文峰，注意下你的措辞，王先生肯定也有自己的考虑”黄詹呵斥了他一句，然后和颜悦色的抬头说道：“惊蛰啊，你能不能跟我说说，为什么不让他下水？刚才你的说辞，我总怕很难接受啊”<br/>　　<br/>　　    王惊蛰叹了口气，缓缓的说道：“你非得让我说的话，那我就一个意思，他在水下很有可能会出现意外，别的我解释不了了”<br/>　　<br/>　　    黄詹顿时就皱眉了，王惊蛰这话明显就跟一人走在街上，忽然有个算命的拉住他说，唉你今天会有血光之灾，赶紧回家呆着去吧，别出门了，十个人里至少得有九个人不会信，没准还得有人甩他一巴掌。<br/>　　<br/>　　    人就是这样，祝你恭喜发现步步高升你愿意听，说你要倒霉，你肯定不乐意。<br/>　　<br/>　　    文峰鄙夷的说道：“你还能掐会算啊，我就呵呵了，你换个理由也行啊，搞什么呢？”<br/>　　<br/>　　    张生利也言语不善的跟黄九郎说道：“黄主任，我觉得你是不是要考虑下，换个合作对象了，有这么毫无道理的危险耸听的么？”<br/>　　<br/>　　    “唰”王惊蛰直接转身就走，回头的时候扔下一句话，说道：“不用换了，我暂时退出，你们随意发挥吧”<br/>　　<br/>　　    王惊蛰的脾气就属于那种顺毛驴子，你好声好语的跟他商量，他就会和颜悦色，但你要顶风来，那他真不惯着你，小脾气嗷嗷倔强，几头牛都拉不回来。<br/>　　<br/>　　    王惊蛰走到另一头，靠在栏杆上抽着烟，陈三岁无语的说道：“这队友确实挺坑，得回没混到一起去”<br/>　　<br/>　　    王惊蛰淡淡的说道：“本来我还心存善念有点愧疚呢，这下好了，我一身轻松了！”<br/>　　<br/>　　    王惊蛰这不算见死不救，话都已经提点两次了，但人家不接受，那真就得怪他们时运不济了。<br/>　　<br/>　　    黄九郎叹了口气，朝着黄詹很慎重的说道：“黄老，你有点心里准备吧！”<br/>　　<br/>　　    黄九郎说完就来到王惊蛰这边，伸手说道：“来一根烟……”<br/>　　<br/>　　    那个叫李菲儿的女人深深的望了眼那边正在和黄九郎神情平淡交谈的王惊蛰，略微寻思了下后，趴在栏杆上，朝着水里被喊懵了那人招了招手，等他游过来后，朝着下方说道：“秋雨一会下水之后，你的节奏放慢点，落在队伍后面自己小心下，然后让陪同的蛙人多留意你”<br/>　　<br/>　　    “呦，李菲儿你也信胡言乱语了啊？”<br/>　　<br/>　　    李菲儿皱眉说道：“我不信，但你最好还是心吧！”<br/>　　<br/>　　    几分钟后，水里的六个人在四名蛙人的带领下，开始朝着水下潜入，李菲儿，张生利和文峰还有剩余的几个人，围坐在电脑前面，看着水下的视频画面。<br/>　　<br/>　　    旁边，黄九郎和王惊蛰抽着烟，说道：“会出事啊？”<br/>　　<br/>　　    “嗯”王惊蛰舔了舔嘴唇，说道：“他面相不太对，印堂三道纹，应该是五行犯水了……不下水什么都好说，下水就出事，黄主任话我说了，他们不信那就不干我的事了”<br/>　　<br/>　　    黄九郎叹了口气，他自然是信了王惊蛰的，可对那帮常年搞学术研究的知识份子们来讲，这话要是说出来就等同于胡搅蛮缠了。<br/>　　<br/>　　    “让研究所和黄教授参与进来，一是上面也有这个意思，因为探索水下古城对我们国家的历史还有考古工作的发展都有着很深远的意义，也争取探索出，古人在几千年前是如何建造出如此一座庞大和壮观的城池，另外一层原因就是，他们毕竟有着丰富的理论和研究知识，可能在某些地方也会对我们起到帮助，哪怕仅仅就一点点，没准就会起到意想不到的效果，对不对？”<br/>　　<br/>　　    王惊蛰弹了弹烟灰，说道：“所以之前我并不太抗拒他们掺和，但现在么？得了，走一步看一步吧”<br/>　　<br/>　　    视频画面里，第一批下水的人在四个蛙人的陪同下，正在逐步下潜，速度不算太快，十几分钟之后到了水下五十多米的深度，这个时候就得渐渐放缓下潜了，然后逐步适应水压给人身体带来的压力。<br/>　　<br/>　　    无声的画面中，显示一切进展顺利。<br/>　　<br/>　　    张生利一派官僚风气的摇头说道：“胡乱闹，黄主任怎么会找了这么一个什么都不懂，还口出狂言的人来参与发掘工作，这简直是我对我们科学研究的一种开玩笑”<br/>　　<br/>　　    文峰笑道：“张所，你要跟他一般见识，那就没意思了”<br/>　　<br/>　　    半个小时后，古城的轮廓在水下强光手电的灯光中，已经隐约能够看得见了。
