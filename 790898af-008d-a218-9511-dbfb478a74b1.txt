　　脚下的大地在颤抖，在颤栗。
<br/>　　震耳欲聋的爆炸声，是这座军事监狱的主旋律。
<br/>　　一颗JDAM落在距离罗昊、麦克米兰等人不到一百米的地方，发生爆炸。那里是一座用木头搭建起来的机枪哨塔，在JDAM的直接攻击下，这座坚固的机枪哨塔瞬间被炸得四分五裂，被火焰所包裹的木头“噼啪噼啪”的燃烧着。
<br/>　　“Move！Move！Move！”
<br/>　　等到两架F/A-18E“超级大黄蜂”从军事监狱上空飞过之后，麦克米兰立刻从机枪堡垒中站起身来，一只手撑在沙袋上，翻身越过垒起来的沙袋，朝着军事监狱那扇已经被炸开的大门跑去。
<br/>　　人，在战斗机扔下的炸弹面前，渺小的如同蝼蚁一般不值一提，不管你是战斗精英还是手无寸铁的平民。
<br/>　　一轮空袭过后，这座军事监狱已经是一片狼藉，监狱中几栋位于地面上的建筑，全都遭到了不同程度的摧毁，尤其是那栋Parker平时居住，被JDAM和“宝石路”精确制导炸弹双重“关照”的楼房，楼房的一半已经被炸得坍塌掉，至于那个本来想跟Parker发生点什么的中东混血美女，此时应该是被彻底掩埋在了那堆坍塌下来的楼房废墟中。
<br/>　　军事监狱空地中，到处可以看到影子部队士兵的尸体，以及被炸弹炸出来的弹坑。有几个被压在废墟下的影子部队士兵，试图从废墟下爬出来，可是他们的下半个身体已经废墟完全压住，根本动弹不得，强行挣扎只是徒增痛苦而已。
<br/>　　麦克米兰握着HK416A5战术步枪，从那些压在废墟下还在挣扎的影子部队士兵身边经过时，毫不客气的给了他们一个点射，结束了他们的痛苦。麦克米兰可不想这些丧心病狂的家伙，明知自己无法活下来之后，拉响一颗手雷，拉着他们做垫背。
<br/>　　麦克米兰带着Bravo小队位于队伍最前方，李然和罗昊押着Parker在队伍中间，队伍末尾是李毅峰和慕容飞防范可能从背后追击而来的影子部队。
<br/>　　Alpha小队和Bravo小队向前推进的速度很快，因为他们每个人都知道，这只是一个开始，那两架“超级大黄蜂”战斗机的目的，是把这座军事监狱夷为平地，彻底抹除掉！接下来，两架“超级大黄蜂”会对地面下的目标，以及地面上的目标进行进一步的轰炸，因为根据情报，这座军事监狱在被影子部队控制之后，这里的地下牢房已经被打造成了影子部队用于存放军火的军火库！
<br/>　　影子部队有百分之四十的军火全都存放在这个军事监狱的地下牢房中，摧毁掉这里的地下军火库能对影子部队造成重创！
<br/>　　Alpha和Bravo两支小队刚逃离军事监狱，天空中那两架F/A-E“超级大黄蜂”就掉头飞了回来，对军事监狱的地下军火库实施轰炸。三枚使用了钻地弹头的JDAM精准的落在地下军火库所在的位置，引爆了储存在地下军火库中的军火。
<br/>　　这次引发的爆炸比起之前第一轮空袭的时候，还要更加剧烈，即便是逃离的军事监狱，还是能感觉到脚下地面在猛烈的晃动，三道火柱从被钻地炸弹轰炸的地下军火库冲出，将漆黑的夜空照亮如白昼。
<br/>　　Parker看着军事监狱的方向，脸色惨白，他知道影子部队要完蛋了。即便影子部队是西伦科特花费重金和大量资源打造出来的精锐作战部队，依靠这支部队足以在一些小国家发动并且完成一次政变，但是当它真正面对“世界五大流氓”的时候，它就变得那么的不堪一击。
<br/>　　联合行动司令部，詹姆斯看到Alpha和Bravo小队成功逃离军事监狱，紧绷的脸上总算是露出了一丝笑容。詹姆斯拿起耳麦，对麦克米兰说道：“Alpha，Bravo，现在立刻带着‘八爪鱼’撤离到我给你们的坐标位置，那里会有快速反应部队接应你们离开。记住下面的坐标，385621，5412893，8545824。”
<br/>　　“坐标，385621，5412893，8545824。”麦克米兰重复了一遍詹姆斯所说的坐标，随后说道：“Bravo1收到，我们会在最短时间内赶往坐标位置。”
<br/>　　“该死的混蛋，总算是抓到你了。”詹姆斯摘下耳麦，对身边的作战参谋问道：“兰利那边什么反应？”
<br/>　　“兰利已经找到了那个把联合行动情报出卖给‘暗月’的人，并且在夏威夷抓到了他，但是……”作战参谋犹豫着没有继续往下说。
<br/>　　“但是什么？”
<br/>　　詹姆斯那双如鹰隼一般的眼睛盯着自己的作战参谋。
<br/>　　“但是在押送他返回兰利的途中，遭到狙击手狙杀。”作战参谋回答道：“CIA一个行动小组全军覆没，车辆被炸毁，现场没有一个活口！”
<br/>　　“砰！”
<br/>　　詹姆斯的拳头狠狠地砸在桌子上，令得房间里其他正在忙着工作的人全都停下了手上的工作，凝神屏息，生怕再次触怒这个正在怒火中的将军。
<br/>　　“废物！”詹姆斯愤怒的说道：“兰利那群家伙就是废物，这次不是启用了秘密路线进行押送吗？为什么还会遭到附近，而且还全军覆没！？”
<br/>　　“经过分析，可能是对方提前知道了押送路线，提前进行了埋伏。”作战参谋擦了擦额头上的汗水，回答道：“除了这个解释外，没办法解释，为什么CIA的行动小组会在押送过程中遭到伏击，并且全军覆没。”
<br/>　　詹姆斯似乎想到了什么，突然伸手阻止了作战参谋后面所要说的话，拿起笔在纸上重新写了一组坐标，然后走到云熙面前，说道：“少校，听说你是个电脑高手。”
<br/>　　“将军看来似乎对我调查的很清楚。”云熙微笑着说道。
<br/>　　“有些事情不提及，只是不想提及，并不代表不了解。”詹姆斯将手中写了坐标的纸条递给云熙，说道：“麻烦你用加密的方式把这个坐标发送给Alpha1，并且告诉他们，放弃原来的撤离坐标，改道去这个坐标上的位置。记住，发送完以后就把纸条毁了，不要让其他人看到纸条上的坐标，‘血腥蔷薇’小姐！”
