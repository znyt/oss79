　　山林空地上，B组队员被押上“黑鹰”直升机后，“黑鹰”直升机立即起飞，带着B组的人离开。
<br/>　　CiA行动队和那些机动部队队员则是留了下来，很显然他们打算继续要搜索逃脱的A组和C组队员。
<br/>　　“撤！”
<br/>　　山林中，齐凡看到CiA行动队和机动部队队员牵着猎狗，在猎狗的带领下，朝着自己这边搜索过来，齐凡果断的下达了撤退的命令。
<br/>　　B组的人已经被抓，他们人员不容再有损失！
<br/>　　A组和C组的队员继续往山林深处撤退，这片山林面积很大，就靠CiA行动队和机动部队加起来这几十个人想要在这片茂密的山林中找到A组和C组，即便是有猎狗的帮忙，难度还是非常大的。
<br/>　　撤离的过程中，齐凡打开了龙牙军刀刀柄的后盖，从里面取出两颗剧毒的药丸，碾碎后涂在肉干上，丢在山林中用以对付那几条猎狗。
<br/>　　狗的鼻子太灵敏，只要被狗闻到气味就会带着追兵在屁股后面一直追，甩都甩不掉，想要暂时摆脱追兵，就只有先想办法把那几条猎狗给放倒。
<br/>　　Royce牵着一条猎狗走在最前面，猎狗嗅着气味一路紧追而来，绑在它身上的那根绳子被绷得很紧，拉着Royce往山林深处走。突然猎狗停下了脚步，站在原地鼻子翕动着嗅着空气中的气味。Royce看到猎狗的反应，立刻对身后的CiA行动队员打了个手势，周围的CiA行动队员瞬间进入战斗准备，Royce自己也是放开了抓在手里的狗绳，抓起挂在身上的M4A1战术步枪。
<br/>　　猎狗仔细地嗅着，然后循着气味往前走着，Royce则是带着CiA行动队员紧跟在猎狗的身后。
<br/>　　猎狗带着Royce和CiA行动队员走了差不多有三四十米，又一次停了下来，继续嗅着空气中的气味，然后加快脚步走到一株植被下找到了那块被齐凡涂了剧毒的肉干。
<br/>　　看到猎狗带着自己这么多人找到是一块肉干，Royce一阵失望，但是他又不能去责怪那只猎狗，毕竟它只是一只狗，在追踪目标和食物之间，它毫不犹豫的选择了食物。
<br/>　　猎狗把肉干吃到嘴里咀嚼着，可是它还没来得及把肉干咽下去，涂在肉干上的剧毒就发挥了作用。
<br/>　　“嗷呜……”
<br/>　　猎狗低呜一声，中毒倒地，四肢抽搐了几下，便不再动弹。剩下的那些猎狗看着自己的同伴突然中毒倒地死亡，竟是站在原地不愿意再继续往前搜索，不管那些牵着猎狗的机动部队队员如何催促，剩下的这几条猎狗都是不愿意再往前挪动一步。
<br/>　　死了！？
<br/>　　Royce眼神一凝，看着倒在地上的死狗，脸色变得有些难看，他没想到那些人在逃跑的过程中居然还会设置下这样的陷阱，这些陷阱显然是用来对付自己身边这些猎狗的。虽然自己这边猎狗不止一条，但是因为眼前这条猎狗被剧毒毒死，对其他剩余的几条猎狗造成了震慑，让剩下的猎狗都不愿意再继续往山林中搜索。
<br/>　　没有了猎狗的帮助，就算是把他们的人数再增加十倍，也很难找到那些人。
<br/>　　卫星搜索？
<br/>　　不可能！这些都是擅长特种作战的高手，跟踪和反跟踪的行家，伪装的专家，他们肯定有办法来骗过卫星。他们国家的特种作战起步虽晚，可是进步却很快，别的国家用几十年时间达到的水平，他们只花了其他国家一半的时间！
<br/>　　牵着猎狗的机动部队队员还在对猎狗进行着催促，催促它们上前继续搜索，可是那几条猎狗说什么也不愿意动弹，甚至直接趴在了地上。
<br/>　　“没用的。”Royce对那几个牵着猎狗的机动部队队员说道：“狗是一种很聪明的动物，它们现在意识到了死亡的威胁，而且同伴的尸体就在它们眼前，不管你们如何催促，它们也不会再继续进入山林搜索。我们先撤，他们的人在我们手中，他们就不会见死不救，肯定会有所动作。加勒比基地群的force-recon很快就要抵达，等到他们抵达后，我们再从长计议。”
<br/>　　那些被比洛多派来支援的机动部队队员在听到Royce的话后，忍不住松了一口气，在攻入布克尔特老巢的时候，他们还跟战术中队合作过，了解他们的战斗力，如果可以选择他们真的不愿意跟战术中队交手，因为他们很清楚，如果交手他们很有可能会死。
<br/>　　两名机动部队队员看了眼地上猎狗的尸体，眼中闪过犹豫之色，显然是有要把这条猎狗带回去改善一下伙食的打算。
<br/>　　“如果你们不怕被毒死，可以选择把这条狗带回去。”Royce对那两名还处于犹豫中的机动部队队员，说道：“这条狗身上的每一块肉、每一滴血都染上了剧毒，如果误食，你们下场就会跟这条猎狗一样。”
<br/>　　回想到刚才这条猎狗中毒死去时的惨状，这两名原本心中有些小打算的机动部队队员顿时一哆嗦，对这条死去的猎狗再也不敢有什么特别的想法。把这条猎狗弄回去，烹煮的就不是一锅美味，而是一锅毒药了。
<br/>　　……
<br/>　　A组和C组往山林深处跑了有几公里才停了下来，在地形并不平坦的山林中奔跑是很耗费体力的，尤其是对佟千雪这种本身状态就不佳的女人来说，到了后面，完全是靠慕容飞在架着她在跑。
<br/>　　孙涛一拳头打在树干上，看向佟千雪，顿时怒从心起，对着佟千雪大声指责道：“全都是你这个女人！”
<br/>　　齐凡上前拉住孙涛，呵斥道：“你在干什么！？担心追兵找不到我们所以要给他们提供一些线索？”
<br/>　　孙涛也意识到自己刚才情绪有些失控，压低声音，指着佟千雪狠声说道：“我承认我刚才有些激动，但是我说得有错吗！？如果不是她没事乱跑被盯上，我们会来到这个地方吗？会面临现在这种弹药短缺的境地吗？B组的人会被抓走吗？”
<br/>　　周围其他人全都是沉默了下来，因为他们没有办法去反驳孙涛的话。
