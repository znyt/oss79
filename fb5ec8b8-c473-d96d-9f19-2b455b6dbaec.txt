　　萧逸轻轻一挥剑，原本坚固无比的灵气牢笼瞬间溃散。
<br/>　　连同慕容墨那全力一掌，也在顷刻间消散。
<br/>　　萧逸持剑而上，直刺慕容墨而去。
<br/>　　慕容墨一惊，刚想躲避，却发现，自己竟被一股恐怖的力量禁锢住了，让他动弹不得分毫。
<br/>　　“怎么回事？我乃洞玄境武者，只有我禁锢人，岂会被人禁锢。”慕容墨顿时大惊。
<br/>　　然而此时，萧逸的身影一闪，已经来到他身前。
<br/>　　“哼。”慕容墨自信地想道，“区区一个后天境，就算老夫站着不动，你也休想伤我...”
<br/>　　他的念头才刚刚出现，萧逸已经一剑刺入了他的心窝。
<br/>　　感受到心窝上的疼痛，慕容墨瞪大了眼睛，“怎..怎么可能刺穿我洞玄境的强悍肉体...”
<br/>　　“这...这难道是紫色武魂...”
<br/>　　“你..你原本就有控火兽武魂...原来如此，你是万中无一的双生武魂，难怪你修炼得如此之快....”
<br/>　　慕容墨冰冷的眼睛死死地盯住萧逸。
<br/>　　他脸上的惊讶之色还未散去，准备反抗并不惜一切代价杀掉萧逸时....
<br/>　　刺入他心窝的冰鸾剑剑尖上，猛地爆发了一股恐怖的力量。
<br/>　　这股力量，异常冰冷，如同九幽之下的冰寒深渊，任何事物都会在其中冰封沦陷，永远无法逃出生天；
<br/>　　又如九霄之上的凛冽罡风，一切事物都会被摧毁成冰屑。
<br/>　　他的经脉、内脏以及血肉在快速冰封，一股死亡的气息笼罩了全身。
<br/>　　“不，不可能，你杀不了我。”慕容墨咬牙大喝，准备调动全身真气抵抗。
<br/>　　恰在此时，他体内猛然出现了一道冰蓝的身影。
<br/>　　身影如梦如幻，像极了一只冰蓝凤鸾。
<br/>　　“怎..怎么可能...这是什么东西...紫色武魂...魂中之魂...这...”
<br/>　　慕容墨的话语未能说完。
<br/>　　因为....
<br/>　　“啾....”冰蓝凤鸾啼叫了一声，声音异常冰冷。
<br/>　　他的血肉、经脉和内脏，在一瞬间彻底冰封，又在一瞬间全部化为了冰屑齑粉。
<br/>　　他已经死了。
<br/>　　身体外表看起来没有异状，但身体内，却只剩下一堆冰屑。
<br/>　　慕容墨已死。
<br/>　　“噗。”萧逸忽然猛地吐出一口鲜血，连忙散去了冰鸾剑。
<br/>　　他感到浑身冰冷，内视一番，发现自己的体内也没有好到哪里去....
<br/>　　八条主脉，千苍百孔，脉上结满了一层层冰痂。
<br/>　　身体内各处，也布上了一层寒霜。
<br/>　　黑墨曼陀罗的黑气莫名其妙地触发了冰鸾剑的神秘力量，这股力量之强，远非他能掌控。
<br/>　　虽然击杀了慕容墨，但自己体内在施展这股力量时，力量上带着的极致冰冷，连自己也一并受伤了。
<br/>　　若是慕容墨晚死几秒，自己散去冰鸾剑晚几秒，恐怕自己也会被彻底冻成一块冰锥。
<br/>　　“冰鸾剑阿冰鸾剑，我华夏传说中的神仙至宝，你到底强悍到什么地步....”
<br/>　　萧逸呢喃着，下一秒，晕倒在地，生死不知。
<br/>　　他的血液被冻住，连带着生机也被冻住了。萧逸也不知自己能否活下去，但他已经没有时间去思考了。
<br/>　　只要能杀了慕容墨，让大长老、二长老、三长老和依依他们多一线生机，付出再大的代价都是值得的。
<br/>　　.....
<br/>　　十数秒前，恰在冰鸾剑那道幽蓝身影出现之时，远在数万里之外，两道在空中惬意飞行的身影猛的一窒。
<br/>　　其中一道人影目光凝视着远方。
<br/>　　“好恐怖的气息，炎龙大陆上何时出现了此等厉害的妖兽。”
<br/>　　人影满脸凝重与惊骇。
<br/>　　旁边一人面露疑惑，道，“圣君，哪里有气息嘛，我怎么没感觉到？”
<br/>　　“你修为尚且，自然感觉不到。”
<br/>　　说罢，人影将身旁之人一把拉住，快速飞行。
<br/>　　数万里的距离，竟在几个呼吸间便完成了。
<br/>　　.....
<br/>　　另一边，大长老与二长老等人，正与慕容山等慕容家长老鏖战。
<br/>　　大长老的修为高于慕容山等人。
<br/>　　可惜，众人之前都中了慕容墨一掌，早已重伤。
<br/>　　此刻根本不是慕容山等人的对手，只是在苦苦支撑罢了。
<br/>　　“嘭、嘭、嘭...”
<br/>　　修为最弱的七八九长老同时被吐血击飞，跌落地面时已无战力，晕厥了过去。
<br/>　　“哼，死吧。”慕容家长老狰狞地笑着，准备结果三人的性命。
<br/>　　“三位长老。”大长老大惊，准备施以援手。
<br/>　　“萧离火，与我二人对战还敢分心，找死。”两个陌生中年男子都是先天九重，联手对付大长老。
<br/>　　大长老反应了过来，与两人对轰了一掌。
<br/>　　“噗。”大长老猛地吐出一口鲜血，虽仍有战力，但已是强弩之末。
<br/>　　“哈哈哈哈，萧家的人，今天注定死绝。”慕容山放声大笑。
<br/>　　“今日起，紫云城再无萧家。”
<br/>　　其余慕容家长老也是放声大笑。
<br/>　　“可恶，我跟你们拼了。”三长老萧重和二长老悲愤交加，拼死一战。
<br/>　　但他们本来就受了重伤，就算再愤怒也是无济于事。
<br/>　　慕容山一掌震伤二长老，而后再一掌打向三长老。
<br/>　　以他先天九重的修为，全力一掌，三长老必死无疑。
<br/>　　“三长老小心。”一旁的依依猛地跃了过来。
<br/>　　她没有战力，慕容家的人从头到尾都没有管过她。
<br/>　　“噗。”依依被慕容山一掌击中，直接吐血击飞，跌落地面后，已经满身是血。
<br/>　　三长老一把抱起依依，急道，“丫头，你犯什么傻，你没有修为，哪里受得了慕容山的一掌阿。”
<br/>　　依依大口大口地吐着鲜血，虚弱地呢喃道，“三长老死了，萧逸少爷会很伤心很伤心的...”
<br/>　　“你...”三长老不禁落下了一滴老泪，“好，很好，不愧是萧家的人，逸儿喜欢的女子。”
<br/>　　说罢，三长老悲怆而冷然地看着慕容山等人，“慕容家的杂碎，既然你们要灭我萧家，那便一起死吧。”
<br/>　　话语刚落，三长老身上气势大涨，但气息异常紊乱，如同暴走一般。
<br/>　　“不好。”慕容山本是洋洋得意之色，忽然脸色一变，“这家伙要发疯自爆了，退。”
<br/>　　“想退？没那么容易。”大长老也是一脸悲怆之色，浑身气势剧烈增长，但也同样气息紊乱。
<br/>　　“慕容家的杂碎，我们萧家的人尽是血性男儿，绝不容你们欺辱，要死便一起死吧。”
<br/>　　慕容山以及各个慕容家长老顿时一惊。
<br/>　　“该死，萧离火是半步洞玄，一旦自爆，我们也难以抵挡，快跑。”
<br/>　　慕容家的人万万没想到萧家之人竟因为一个依依的死而疯狂，纷纷欲要退离。
<br/>　　“同归于尽吧。”萧家众长老放声大笑。
<br/>　　下一秒，周围百米范围内的气息已经极其紊乱。
<br/>　　若萧家众长老齐齐自爆，其结果必然是同归于尽。萧家皆亡，慕容家的人不死也残。
<br/>　　倒在三长老身旁的依依还在大口吐着鲜血，呢喃道，“萧家就要没了，依依也快死了，少爷一定要活下去...”
<br/>　　但，正在此时，天空中忽然出现了一轮月光。
<br/>　　月光照耀之下，一切静止。
<br/>　　萧家即将自爆的长老，慕容家惊恐欲逃的小人，一瞬间定格。
<br/>　　周遭的空气、紊乱的气息、乃至天上的浮云，全部定格。
<br/>　　两道人影凭空出现，并来到依依身旁。
<br/>　　两道人影被一股光芒覆盖着，没人看得清他们的样貌。
<br/>　　唯独依依，虽已命在旦夕，但那双灵动的眸子，清楚地看到了这两人的样貌。
<br/>　　一个年轻的妇人，约莫三十出头，风姿卓卓，但气度逼人，让人不敢直视。
<br/>　　一个是比她大不了多少的少女，脸上带着俏皮的笑意，笑意盈盈地看着她。
<br/>　　年轻妇人轻柔地将手搭在依依身上，半晌，面露大喜。
<br/>　　“果然....果然....没想到来这一趟，竟有如此收获。”年轻妇人兴奋地说着，而后一挥手，一道白光打入依依体内。
<br/>　　依依顿觉原本重伤的身体舒服了一些，愣愣地说了一句，“谢谢姐姐。”
<br/>　　年轻妇人掩嘴一笑，柔声道，“小女娃，你这声姐姐可叫不得。倒是你，可愿意跟本圣君走？”
<br/>　　依依连忙摇头。
<br/>　　年轻妇人皱了皱眉头，但还是柔声道，“若我帮你，救你家中长辈呢？”
<br/>　　依依想了想，又看了眼萧逸离开的方向，眉头皱得极紧，似在做着纠结的考虑。
<br/>　　半晌，依依点了点头。
<br/>　　年轻妇人眉开眼笑，轻轻一挥手，萧家众人原本准备自爆的气息瞬间稳定下来。
<br/>　　而后，她看了眼定格在原地的慕容山等人，眼中一道精光闪过。
<br/>　　“咔、咔、咔....”
<br/>　　虚无的空气，此刻竟如同镜子开裂，出现了一条条裂痕。
<br/>　　裂痕之处，正是慕容山等人的位置。
<br/>　　慕容山以及慕容家长老忽然瞪大了双眼，而后悄无声息地倒地，身上没有一点生机。
<br/>　　“破碎空间？”被定格住的大长老愣愣地看着，心中惊骇到极点，这是洞玄竟、乃至破玄境，甚至是强大无比的地元、天元武者都绝对做不了的事。
<br/>　　恐怕，只有那等传说中可以遨游太虚的武道大能才可以做到。
<br/>　　嗖，一道白光闪过，两道人影以及依依凭空消失。天空中那轮月光，也在一瞬间消失。
<br/>　　一切回到了正常，风再次吹拂，空气再次流动，萧家众人愣愣地站在原地。
