　　黑齿常之的一席话说完，登时吓坏了跟在他身后的副将。
<br/>　　明知道这是敌人的圈套还要往里面钻，这，这不是疯了么？
<br/>　　不过黑齿常之前没有让他的副手等待太长时间，很快便继续说道：“你安排一些人，去咱们来时的路上多弄些芦苇，下午全军都给老子扎草人、还有草马，不要求多像，有那么个意思就行，明白么？”
<br/>　　“明白！末将马上去办。”本有些疑惑的副将就算是再笨，此时也想明白这一切应该只是黑齿常之给敌人设的一个圈套，立刻兴高采烈的去了。
<br/>　　而黑齿常之也在那副将走了之后，四下里又看了看，便又对身边的亲卫吩咐了几句，接着便将人都打发了出去。
<br/>　　……
<br/>　　玄菟城外，数万靺鞨大军枕戈待旦，气氛十分严肃。
<br/>　　中军大帐中，大祚荣高居首位，其他六部首领分列两旁，紧张的盯着大帐的门口，像是在等什么人一样。
<br/>　　终于，黑水部族长忍不住了，开口对大祚荣说道：“首领，到现在前面还没有消息，应该不会出什么岔子吧？”
<br/>　　“不会，如果唐军按照他们之前的行军计划没有变的话，我们的计划就不会失利。”大祚荣虽然也很紧张，但却没有表现出来，说话的声音带着沉稳，一副胸有成竹的样子。
<br/>　　“可是如果他们加快行军呢？”
<br/>　　“如果李绩派出来的那个黑齿常之不傻，他就不会干出这样的事情，如果他这样做了，我们正好是以逸待劳，到时候杀他的措手不及，所以他一定会在半路停下来。”大祚荣说道。
<br/>　　“首领，我们有十万大军，而唐军只有一万，我们何不趁机掩杀上去将他们彻底消灭？”白山部族长也有些坐不住了，打开话匣子问道。
<br/>　　“唐军武器犀利，不到万不得已，我们还是不要与他们直接交手，毕竟这十万勇士已经是我们全部的人手，而大唐国内却还有百万精锐，我们必须保存实力。”大祚荣耐心的解释道。
<br/>　　他现在需要的是一个团结的集体，而不是一团散沙，所以大祚荣并不像用雷霆手段让其余六大部族与他离心离德，是以耐心也是他的计划之一。
<br/>　　“可是首领，万一唐军发现了我们的计划怎么办？那么多人都被派到了后面去叠坝，大唐的探马很容易就会发现他们。”
<br/>　　“是啊首领，此事不可不防，那些唐人探子神出鬼没，而且个个身手不凡，我们勇士跟他们硬拼之下可是一点好处都没讨到。”
<br/>　　说到计划被发现，又有两族的族长发言，似乎很担心计划是否会失败。
<br/>　　不过对此大祚荣却有自己的打算，只见他微微一笑，满不在乎的说道：“我们十万大军的阵盘扎在这里，那些唐军的探子就算是能绕过去，等他们发现情况不对想要再回去时间也来不及，所以这一点诸位完全不必担心。”
<br/>　　“至于说拼不过大唐的探子，这一点无所谓，反正我们也不是打游击战，我们要作的便是一举将其击垮，仅此而已。”
<br/>　　在坐众人听大祚荣如此说，一个个也再没了想法，全都沉默着继续等待着已方探马的回报。
<br/>　　……
<br/>　　时间一点点过去，黑齿常之安排出去人的手很快就回来了，大量的芦苇被推到一起，而与此同时，唐军的大营也扎好了。
<br/>　　“将军，草人已经安排人在扎了，我们是不是该撤了？”副将安排好了一切，回到黑齿常之的身边。
<br/>　　“不能这么容易就撤出去，这周围一定会有靺鞨人的探子在观察我们，虽然他们看不出我们到底在忙些什么，但如果大军往外一撤必然会引起他们的警觉。”黑齿常之摇摇头，否定了副将的想法。
<br/>　　“那，那怎么办？我们总不能在这里等死啊。”副将一听黑齿常之的判断，顿时有些急了。
<br/>　　“不急，还不急，一切等天黑再说，那些靺鞨探子回去还需要时间，等他们确定了我们在此扎营再回去，然后等到他们开闸放水至少要午夜时分，所以我们还有时间。”黑齿常之慢悠悠的说着，似乎真的一点都不着急。
<br/>　　“可是，可是万一他们提前下手呢？”副将显然考虑的要多一些，必竟黑齿常之是百济人，谁也不能保证他会不会在这个时候背叛大唐。
<br/>　　“没有什么可是，他们一定不会在白天下手，如果是白天大水到了，我们完全有机会撤走，只有等到夜里我们都睡了，他们才会发起攻击。”黑齿常之摆摆手，再一次否定了副将。
<br/>　　“那，那我们接下来干什么？”副将见无法改变黑齿常之的主意，也没有什么好办法，于是便问起接下来的行动安排。
<br/>　　对于这一点，黑齿常之似乎也考虑到了，只见他淡淡一笑，慢条斯理的说道：“挖坑，坑不要太大，位置随便，只要有半身高就可以。”
<br/>　　“这，这又是为什么？”副将问道。
<br/>　　“你觉得大水过后那些靺鞨人会不会杀上来？营地被大水一淹那些坑里便会灌满水，那些靺鞨人杀上来之后，水坑也就会成为他们的死地。”
<br/>　　副将真的不知道应该怎么形容黑齿常之，带着命令离开之后，他只是觉得黑齿常之这人真的很厉害，好像什么事情都难不倒他，那些靺鞨人就好像是被他捏在手中随意玩弄的棋子一般。
<br/>　　……
<br/>　　唐军真的在河边驻扎下来了，而且看样子似乎并不打算继续前进。
<br/>　　河对岸的靺鞨探子远远望着忙碌的大唐军营，心中确定了这一点。
<br/>　　于是这些探子们利用芦苇的掩护一点一点爬向了远处，在三里之外，肉眼已经无法看到唐军大营的位置拉出了战马，一一翻身而上，向着玄菟城的方向赶去。
<br/>　　而他们并不知道，在他们身后不远处的唐军大营，吊斗里的侦查兵已经用望远镜把他们的一举一动都看在了眼中。
<br/>　　所以就在那些探子离开不长时间，唐军也开始集结了起了，利用已经暗下来的天色，一批批撤出了营地，消失在河岸的远处。
<br/>(本章完)
