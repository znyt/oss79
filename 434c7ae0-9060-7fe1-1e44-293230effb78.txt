　　“战战战！”
<br/>　　“杀杀杀！”
<br/>　　巫、妖、人三族汇聚而来，一场大战就此展开。
<br/>　　惊天的杀意轰动四方，冲霄的战意动荡乾坤。
<br/>　　让关注此战的众多大神门纷纷大惊失色，对于此战，他们也是期待已久了，此战过后，洪荒就要重新洗牌了。胜者为王，败者死亡。但究竟谁胜谁负，还要看这一战。
<br/>　　“期待已久的时刻终于来临了。”
<br/>　　“三族大战，再现远古时期龙凤麒麟三族神威，决定胜负的时刻来了，是如同远古时期一般，三族败退，黯然退场？还是崛起洪荒，主宰天地，就看这一战了。”
<br/>　　“胜王败寇，三族大战！”
<br/>　　“问苍茫大地，谁主沉浮？还看三族大战！”
<br/>　　“最好是三族同时落败，嘿嘿！”
<br/>　　“……”
<br/>　　三族强者也不废话，直接挥兵而上，开战洪荒。
<br/>　　三方气势凝聚，轰动九霄。刚刚还是晴朗天空，瞬间便变得乌云密布，乌云翻滚，雷闪电鸣，漫天雷雨疯狂降落而下，雷电肆意劈下，一时之间，千万里疆域，尽是雷闪电鸣，像是在诉说战场的激烈。
<br/>　　三族大战，完全分不出谁为敌人，只要碰上就打，碰上就杀。顿时，一阵血雨迅速掀起，无数残值断臂、五脏六腑，在四处溅射。
<br/>　　无数飞剑横空而出，各种禁制法术，如同光雨一般，到处狂轰乱炸，各种爆炸声响起，如同烟花一般绚烂，轰隆隆的巨响，像是在诉说量劫的残酷。
<br/>　　什么福德之人，什么神通修士，在这个时候，都变得不值钱了，死了不知多少，生命在这一刻，是如此的脆弱。
<br/>　　天地之间，无尽黑色劫煞之气在笼聚而来，煞气蔓延，杀气惊天，凶光闪烁。
<br/>　　什么神通秘法，什么神兵利器，在战场上一切都是虚的，唯有保命，才是最重要的。修士之间的战斗，一死便是一大片、一大片的，万物都是战场，一切可用之手段，不管是善是恶，是好是坏，是邪是正，都被用了出来，无所不用，无所不杀。
<br/>　　九天之上，血泪飘散，残值断臂漫天飞舞，发如血，性若魔，神如鬼，身似神。苍穹在哭泣，大地战颤抖，仿佛是在害怕。
<br/>　　“轰隆隆！”
<br/>　　一颗星辰被一只巨手捏爆，无尽星光散落下来，神光湮灭，爆炸声不断，玄光相触，仙光降落，各种玄法道术轰击在一切。
<br/>　　兵器的交接声，修士的嘶吼声，玄法道术的爆炸声，天上的雷霆声，大地的颤抖声，风云的呼啸声……各种声音夹杂在一起，场面混乱不堪，战场，是如此的残酷。
<br/>　　人族十大转轮皇者境强者对上妖族的准圣或巫族祖巫强者，无数法相境强者使出自己的天地法相，妖族幻化出本体，遮天蔽日，各种异象漫天飞舞，巫族也是出真巫之体，操控日月星辰，地水火风，不断战斗。
<br/>　　战场不断扩大，地上，天空，海里，地下，虚空中，天地间到处充斥着血的气息，弥漫着无尽杀机。
<br/>　　大海在咆哮，星空在颤栗，大地在颤抖，山河在破碎……
<br/>　　“桀桀桀，战吧，杀吧，尽情的屠杀吧，真是一场丰盛的晚宴。哈哈！无尽劫气，真是美味！”
<br/>　　太华山劫魔殿，万劫魔尊张口一吸，便将天地间无数劫煞之气吸引过来，如同一个倒斗一般，呼啸着被吞尽口中。
<br/>　　在他的身后，无数劫魔一个个从一道三丈大小的血池之中走了出来，汇聚起来，他们充斥着无尽暴掠气息，完全是像是杀戮的机器。
<br/>　　在虚空之中，一道黑色魂幡摇动，天地间无数陨落强者的灵魂被吸引过来，被魂幡吞噬。黑幡之上闪烁着惊人的魔气，带着摄人心魂的气息，更有一种嗜血的冲动。
<br/>　　万劫魔尊手中，更是提着一柄血红色的战刀，刀光冷厉，带着无尽魔气，血光闪烁，有一种吞噬之意，仿佛能够吞噬无尽生灵的鲜血。
<br/>　　这就是一柄嗜血的魔刀。
<br/>　　虚空之上，人族武祖玄天与妖族帝俊、东皇太一、巫族帝江，四人大战，风暴不断，虚空破碎，时光扭曲，雷霆若尘。
<br/>　　四人完全是在混战，有时候一人攻击三人，也有时候两两对战，更有时候几人攻击同一人，碰上谁，便战谁，完全没有规律可言。
<br/>　　“战！”
<br/>　　玄天头顶武塔，玄光垂落，万法不侵，武塔经过无数强者的祭炼，显得更加强大了。九层宝塔，里面有九方中千世界，镇压下来，就仿佛是九个世界镇压而来，让一片虚空颤栗动荡，空间被压碎。
<br/>　　“以我武塔，吞尽巫妖无数鲜血，想必能够再进一步，镇压诸天。杀！”
<br/>　　玄天手捏“武”，以一道演万法，玄光攻击，各种道术神通随手捏来，攻击不断，道则演化出种种攻击，血战苍穹。
<br/>　　“斩！”
<br/>　　帝俊手持金色王剑，目光冷峻，丝毫不为所动。他眼神淡漠，手中帝剑带着无双帝威，无尽神通法力涌动，惊天剑意通天彻地，煌煌之威，如同一轮大日横贯于虚空之中，释放出无尽神光，灭杀敌人。
<br/>　　一剑一剑，莫不带着巨大威压，横贯八方，引动天地规则，震动天穹。
<br/>　　祖巫帝江空间法则攻击，割裂苍穹，爆空拳意惊天动地，粉碎一切攻击，他攻防得体，无所畏惧。东皇太一手持混元凶剑，凶光四射，煞气缭绕，剑剑冷厉凶狠，带着无上杀机，斩杀两人。
<br/>　　嗡隆一声巨响，玄天头顶的宝塔在摇动，垂下苍茫之光，有混沌缭绕，看起来恐怖无比。
<br/>　　当武塔向前镇压之际，虚空一片抖动，仿佛承受不住那般恐怖威压，玄天全身威势爆发，惊动诸天，让洪荒无数大神通者纷纷瞩目，就连诸圣都看向这个方向，无不为之动容。
<br/>　　武塔砸击了过去，武祖玄天神勇盖世，武塔与帝剑相撞，同时自身的道法显现在武塔之上，镇压了下去。
<br/>　　“轰！”
<br/>　　妖帝帝俊冲天而起，披头散发，满身都是血迹，嘴里更是血水长流，还没等站稳，又是一个踉跄，差点栽了一个跟头。
<br/>　　“大哥，你没事吧！”
