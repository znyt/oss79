　　李承乾不得不承认，自己的确没有考虑过王玄策提出的问题，在其提醒之下不由了被吓出了一身的冷汗。
<br/>　　那可是一百多万人啊，一下子放到社会上根本无法被消化掉，就算是国家给他们一些补偿，可那也只能解一时的燃眉之急，时间长了终究还是会出现一些这样那样的问题。
<br/>　　先不说这些人能不能全都找到工作，就算是有些人找到工作了，可数十年的军营生活早就把他们变成了另外一种人，不管是脾气还是其他都与普通百姓大不一样。
<br/>　　这些人的性格多少有会有些扭曲，一言不合杀几个人简直太正常了。
<br/>　　如果真的发生了这样的事情怎么办？如果仅仅是一两起这样的事件还好说，如果多了呢？难道再把他们重新召回？就像后世生产的那些汽车一样？这显然是不可能的。
<br/>　　想到这些，李承乾有些颓然的坐回到了椅子上，看着王玄策等人久久无言，半晌方才说道：“刚刚是朕错了，你们说的对，朕有些急于求成了。”
<br/>　　“臣等不敢！”王玄策等人连忙躬身表示歉意，毕竟在古代的社会皇帝是不会错的，能让皇帝认错的就只有上天，谁让皇帝就是天子呢。
<br/>　　“不，错了就是错了，人非完人孰能无过，父皇曾经告诉过朕，要明事理辨是非，如果朕连自己的错误都看不到，又怎么能看到别人的错误？”李承乾叹了口气，苦笑着摇头说道。
<br/>　　这并不是李承乾在故意装大尾巴狼，事实上也只有到了他的这个位置上，才能真正体会到明事理辨是非是多么重要。
<br/>　　对于普通人来说，很可能分辨是非最多只能停留在防止上当受骗，不要走上歪路，不要打架斗殴的程度。
<br/>　　但是到了李承乾的这个位置，是与非对与错已经完全变了，不再像普通百姓眼中那么朴实，很多以前在他看来正确的事情，现在重新回头去看就会发现，当初的想法是那么的幼稚而无知。
<br/>　　举个例子来说，就好比大唐的军队现在已经可以横扫天下，这在以前的李承乾看来是一个很好的机会，完全可以继续打下去，最好能够统一全球，然后再统一宇宙，最后打破空间壁垒走向无限广阔的空间。
<br/>　　可是现如今李承乾的想法已经变了，在他现在看来，一个没有对手的国家是不应该存在的，因为当没有外敌的时候，人类就会开始内斗。如果真的统一全球，不，哪怕是统一了这片大陆，那么接下来大唐必然会发生内战，而内战一开必将生灵涂炭。
<br/>　　那么这样一来，横扫天下是正确的还是错误的？
<br/>　　老头子曾经说过，以史为镜，以人为镜的话，李承乾也只有坐到现在的位置上才能体会老头子的深意。
<br/>　　不得不说，千古一帝毕竟是千古一帝，虽然李承乾相信自己可以带领大唐走向再高的位置，但如果和老头子比起来，他还是差了不止一点半点。
<br/>　　因为他所作的一切全都是在借鉴，不管是民生方面还是军事方面，甚至对外的国策也是如此。而老头子则不然，他所做的一切都是在创造，是靠着自己得来的。
<br/>　　所以相比之下李承乾与老头子之间的高下几乎立即就可以判断出来。
<br/>　　马周等人并不知道李承乾在短短的一瞬间想了那么多的问题，在他们看来皇帝陛下只是有些消沉，并没有什么其它变化。
<br/>　　而李承乾也没有让他的这几个得力下属多等，消沉了片刻之后已经再次打起精神，抬头说道：“玄策，你们几个回头拿出一个方案来吧，军制改革必须要进行下去，虽然不能马上裁军，不过朕觉得可以放出一些风声，观察一下军中的反应，也可以先试着栽一部分人下来。”
<br/>　　王玄策见李承乾依旧不肯死心，心中暗自苦笑，毕竟刚刚才卷了皇帝的面子，而且李承乾也认识到了自己的错误，这一次如果再不顺着他点，估计这位新皇帝会再次暴走吧。
<br/>　　想到这里，王玄策犹豫了一下小心的说道：“陛下，臣觉得军制改革完全可以先放一放，暂时只要把消息通知下去就好，如果有人原意退役那就退，如果不愿退役就不退，一切凭自愿也就是了。”
<br/>　　“如果凭自愿朕怕下面的人好心办坏事啊，玄策！”李承乾轻轻叩击着桌面，模仿着老头子的动作，这完全是无意识的举动，连他自己都没有发现，经过短暂的停顿之后才继续说道：“下面总有一些人喜欢小题大做，如果朕下旨让士兵自愿退役，你能保证下面没有人为了溜须拍马不搞强制么？如果他们在搞强制这又与朕以前的想法有什么区别。”
<br/>　　“那陛下的意思是？”王玄策反问道。
<br/>　　他同样很清楚官场上的一些弊端，或许那些十六卫的将军们不会干出这样的事情，因为他们的地位已经足够高，不需要这样奉承李承乾。可是那些偏远地区则不一样，那些底层的官员为了表现自己很有可能会强制手下士兵退役。
<br/>　　“我的意思是把这件事情交到你们几个手里，你们与兵部联合，给你们十年时间把这件事情完成，总之，十年之后，朕要看到一个年轻化、现代化的大唐军队。”李承乾眼珠一转，很不负责任的一推六二五，利用皇帝的特权，把这个棘手的问题全部推给了王玄策等人。
<br/>　　“这……”王玄策、马周、杜构几人面面相觑，尴尬的发现自己似乎掉进了一个圈套里面，不管答不答应，似乎都是一件麻烦事。
<br/>　　不过，想想还有十年时间，三人却也没有再多说什么，同时低头答道：“是，臣等领旨。”
<br/>　　“嗯，这件事情就这样这下来了，你们几个回头好好商量一下。另外过段时间朕打算调各地驻军将领回长安，你们对此有什么建议？”把军制改革的事情安排好以后，李承乾话锋一转，扯到了那些在外驻扎了近十年的大唐将军身上。
<br/>　　这些人常年在外，按说李承乾还是能相信他们的，只不过却不相信他们的接班人，所以他打算趁着现在国家还算稳定，把这些人调回来，听听他们的意见。
