　　天空之上。
<br/>　　那些七大宗门的武者，已经是至少有着大半武者，已经是进入到了其中，而剩下的那些几人，则已经是联手起来，来阻拦这其他武者，进入其中。
<br/>　　“哼，归墟天这样的宝地，根本不应该跟这些废物一同分享，就凭他们还想要得到其中的宝藏？还是乖乖的跟在我们七大宗门的后面，捡我们挑剩下的东西吧。”
<br/>　　一尊身上布有星图的天枢殿弟子，冷笑的开口道。
<br/>　　“不错，让这些人进入归墟天，本身就是在浪费宝物，倒不如就在这里，将他们全部拦住，也让他们好好知道，这归墟天的宝物不是他们配得到的！”
<br/>　　又一尊七大宗门的弟子开口道。
<br/>　　就在这个时候，又是几尊武者，已经是忍耐不住了，纷纷出手，向着天空之上轰击过去。
<br/>　　这归墟天，便是一处拥有无数宝物的地方，如今他们都被拦在外面，又让谁可以忍耐得住，不由的纷纷出手，打算便是要强行冲入其中。
<br/>　　这局面，便是越发的混乱。
<br/>　　那些将归墟天撕裂出裂痕的强者，如今便是一个个的凌空负手而立，淡然的看着眼前这一切。
<br/>　　他们的约定，便是要打开这归墟天，至于其他的，任何争斗，便是也不在他们的约定之内。
<br/>　　归墟天已经开启，至于是足以从其中得到多少的宝藏，就要看武者自己的造化了，没有实力，就算是归墟天开启，也只能是跟在后面，捡一些别人不要的宝藏。
<br/>　　不管是到哪里，都是以实力为尊，没有实力，也将没有任何意义。
<br/>　　“动手！”
<br/>　　楚风眠清喝一声。
<br/>　　他身上的灵力，猛然暴起，远古战龙血脉的力量，被催动到了极致，猛然之间便是冲着天空之上，直接飞了过去。
<br/>　　他的身躯开始疯狂的变化着，在这一股股的灵力之下，顿时化为了远古战龙，冲着那归墟天飞过去，身旁的无数武者，在感受到这一股气息之下，都是纷纷退后。
<br/>　　在这个时刻，楚风眠便是没有打算在等待下去了，这归墟天中拥有无数宝藏，楚风眠可不想晚人一步。
<br/>　　他化身远古战龙，横冲直撞，直接便是打算带着寒月漓一起，冲入到了归墟天之中。
<br/>　　“什么？”
<br/>　　“拦住他！”
<br/>　　“此人就是之前跟玄天宗交手的那尊武者！不能让他进去！”
<br/>　　那些七大宗门的弟子，也一下子察觉到了楚风眠的存在，一尊尊催动灵力，便是要来阻拦楚风眠。
<br/>　　其中一尊身高三米的壮汉，更是大步上前，直接来到了楚风眠的面前。
<br/>　　这壮汉一身的肌肉，都是呈现出了一种金属般的光泽，显然是一尊炼体的武者，在七大宗门之中，唯有圣王宗，是炼体宗门。
<br/>　　圣王宗拥有九域之间最为完美的炼体手段，圣王宗的弟子，不修武道，不修兵器，所修炼的只有自己的身躯，那是极致的身体。
<br/>　　这一名圣王宗弟子，看到楚风眠过来，大步上前，直接来到了楚风眠的面前，猛然一拳便是冲着楚风眠轰击过来。
<br/>　　“碰！”
<br/>　　这一拳轰击而出，周围的空间都在破碎，力量惊骇至极，让生死境巅峰的强者来，都要被一拳打死。
<br/>　　但是楚风眠却是脸色不变，面对着这一拳轰击过来，他也将身躯之中战龙血脉的力量催动到极致，在他的身躯之上，都是凝聚出了一枚枚的龙鳞。
<br/>　　这龙鳞密布，组成了一道阵法，力量加持在楚风眠的身上，他的身躯猛然跟这一拳相撞。
<br/>　　“碰！”
<br/>　　这两股力量相撞，那圣王宗弟子的眼睛猛然瞪大，他感受到，这楚风眠的力量，甚至是毫不逊色与他，甚至是还要超越他几分。
<br/>　　他的那一拳力量，居然是在跟楚风眠碰撞之下，被直接大散。
<br/>　　同时楚风眠又是一道龙爪，突然轰击过去，硬生生的砸在这圣王宗弟子的身躯之上。
<br/>　　这圣王宗弟子，连连后退数百步，才稳定住了自己的身形，但是在他的身上，金属般的皮肤上，也是留下了一道白色的印记。
<br/>　　“好厉害的力量！没想到这偏远的北域，居然是出现了如此人物！”
<br/>　　这圣王宗弟子被打飞后，刚刚反应过来，想要再度出手。
<br/>　　但是楚风眠所化身的远古战龙，却是丝毫没有理会他，而是直接向着那归墟天的裂缝飞了过去。
<br/>　　“不好，万世居然没有拦住他！”
<br/>　　“我们先拦住此人再说！”
<br/>　　看到楚风眠居然是以力相搏，居然便是将这圣王宗弟子都给击退，其他的几名七大宗门弟子，一个个的脸色都是露出了几分震惊。
<br/>　　纷纷身形一动，四面八方之间，已经是想要楚风眠围困过来。
<br/>　　“想拦我？”
<br/>　　楚风眠冷哼一声，他那巨大无比的战龙身躯，在空中猛然舞动起来。
<br/>　　“战龙舞苍穹！”
<br/>　　轰隆！
<br/>　　这一股股的灵力，将那些七大宗门的弟子，直接都给震开，没有任何一人，可以阻拦楚风眠，一个个都吃了不小的亏。
<br/>　　“走！”
<br/>　　抓住这个机会，楚风眠也没有兴趣在纠缠下去，这些七大宗门弟子，楚风眠想要击败他们也需要浪费不少力气。
<br/>　　现在当务之急还是进入到归墟天中，至于这些人，楚风眠可没有时间来跟他们浪费。
<br/>　　楚风眠的身躯直接变化回来本来的样子，抓起寒月漓，化为了无数幻影，直接便是一跃进入到了归墟天之中去。
<br/>　　“我们也上！”
<br/>　　看到楚风眠成功进入归墟天，那些剩下的武者也都按耐不住，纷纷出手，不一会，又是更多的武者，都是进入到了归墟天之中去。
<br/>　　而那些七大宗门的弟子，也逐渐的抵挡不住围攻，直接便是纷纷退入到了里面。
<br/>　　“这里的对面，就是归墟天！”
<br/>　　楚风眠跟寒月漓的身躯，在进入到了这裂缝之后。
<br/>　　楚风眠便是感觉到，这周围的裂缝，已经是极为不稳定。
<br/>　　在临近到归墟天的一刻，他们二人的身躯，被两股不同的力量席卷，直接便是被带走。
<br/>　　这两股力量，便是带着楚风眠跟寒月漓，落入到了归墟天之中去。
