        饭后离席，傅昼景第一次带童筱颖去自己小时候在这傅家大庄园里的住处。
<br/>
<br/>        虽然他太久没在傅家大庄园里住过了，但是冷太太还是会安排女佣，隔三差五地来他房间打扫卫生。
<br/>
<br/>        傅昼景的住处和她之前住的那个偏院一样，是个三合院，但比起偏院更华丽，这里前有院子和正大门，后是庄园主路，两边柳树成荫。
<br/>
<br/>        “中间的主屋是大哥的，我住的地方在右边。”傅昼景牵着童筱颖的手，一边抬手指了指，一边微笑着说道。
<br/>
<br/>        童筱颖忍俊不禁起来：“小卫，你是不知道。我每次来傅家大庄园，就有种自己好像被穿越了的错觉！”
<br/>
<br/>        外边的世界，已经是高楼大厦，而进了这傅家大庄园，就恍若时光倒流，回到了古代。
<br/>
<br/>        “本来这儿不叫‘傅家庄园’，是封地的‘王府’，也是这临海城里唯一的一座古宅。是爷爷的父亲，花了一番心血，改了名字，才把这王府保下来。”傅昼景介绍道。
<br/>
<br/>        童筱颖不禁好奇地问道：“既然是上了百年的王府，那么应该有古董吧？岂不是会被小偷惦记着？”
<br/>
<br/>        “曾祖父把所有价值连城的古董全都上缴了，所以，庄园里的摆饰品，都是现代仿品。而且，这古屋维修什么的。也早就不是原来的那个样子了。只是比原来，更奢华。”傅昼景苦笑道。
<br/>
<br/>        虽然，傅家大庄园里二十四小时都有从部队里退伍回来的保安巡逻，但是至今从未遭遇过小偷事件。
<br/>
<br/>        大概，外边的小偷都知道，这傅家大庄园里是真的没啥值钱的东西可偷。
<br/>
<br/>        “想不想知道，这庄园里最值钱的是什么？”傅昼景牵着童筱颖的手，卖起了关子。
<br/>
<br/>        童筱颖好奇地眨巴着眼睛，问道：“是什么？”
<br/>
<br/>        傅昼景莞尔一笑，拉着童筱颖的手离开了院子，去了后花园。
<br/>
<br/>        要说最值钱的东西，只怕就是这座府邸以及后花园里这棵百年金丝楠树了。
<br/>
<br/>        小偷应该是无法把这么大一个府邸和那棵老树给偷走吧！
<br/>
<br/>        童筱颖被傅昼景带到了一棵参天大树下，之前她就有留意过这棵树，只是不知道这棵树叫什么名字。
<br/>
<br/>        这棵树的树干呈灰色，估摸着要五六个人手牵手才能把这树干给抱住。
<br/>
<br/>        “这棵树，是金丝楠树，听爷爷说，它有一百多岁了！”傅昼景微笑着介绍道。
<br/>
<br/>        童筱颖长这么大，真的还是第一次接触金丝楠树，她抬起手来，忍不住抚摸着树干，感慨道：“当一棵树真好，可以长命百岁，还可以永远停驻在自己生长的地方。”
<br/>
<br/>        “等我们去澳洲了，我们也在前院里种棵树，让那棵树，见证我和你，还有将来我们的孩子，在一起的幸福时光。”傅昼景憧憬地说。
<br/>
<br/>        童筱颖满心欢喜地点头赞同，高兴地扑入傅昼景的怀中，一手搂着他的腰畔，一手在半空中比划道：“那我们种一棵金合欢树，等树长大了，我们也老了，然后坐在大树底下乘凉，看着孙儿承欢膝下。”
<br/>
<br/>        他两在树下谈笑风生，却不知树上还坐着一个人。
<br/>
<br/>        饭后，爷爷傅炎廷回屋前，跟他的老友打电话絮叨说想要一串金丝楠木手串，这话被傅夜沉无意间听到后，就上了心。
<br/>
<br/>        所以，他才来这割树枝，但刚爬上树没多久，拿着军用口袋刀正在割这金丝楠树的树枝时，傅昼景和童筱颖便过来了。
<br/>
<br/>        听着他俩的对话，傅夜沉心里既羡慕又难受。
<br/>
<br/>        童筱颖和傅昼景一直在树下的石桌前坐着聊天，傅夜沉就一直坐在树上待着，默默无闻地听着他们两的聊天内容。
<br/>
<br/>        他俩聊了许久都不见有要离开的意思，傅夜沉又不好意思跳下去打扰他俩，于是，手里拿着军用口袋刀，静默地在这金丝楠树的树干上，一笔一划，认认真真地刻着“夜沉爱筱颖”这五个字。
<br/>
<br/>        反正这五个字，是被他刻在这树干上端的枝繁叶茂之下，从此以后，估计除了他，就再无第二人知晓这个秘密了吧！
<br/>
<br/>        傅夜沉苦笑，背靠着树干，仰望着被树枝遮去了一大半的夜空。
<br/>
<br/>        树下，他俩携手闲聊，时间不知不觉地过去了，一轮圆月挂上枝头。
<br/>
<br/>        童筱颖也看得出，傅昼景对这里似乎一点都不留恋。
<br/>
<br/>        他没有什么东西可带走，也没有什么东西能让他带走。
<br/>
<br/>        就在童筱颖和傅昼景坐在金丝楠树下的石桌前有说有笑的时候，管家突然疾步走寻了过来，欠身通报道：“二少爷，外边有人找您，希望您出去跟她见个面。”
